# OFCA / licensed telecommunications works — coordination map

**Halt:** Do not certify undertaker approval or clearance without written confirmation from the licensed undertaker. Verify current OFCA codes and undertaker procedures at project inception.

## Authority split

| Body | Role | Not the same as |
|------|------|-----------------|
| OFCA | Regulates telecom licensing and undertaker standards | EMSD (lifts/electrical), CLP/HKE (power) |
| Licensed undertakers | Own/maintain fixed telecom plant; approve protection/diversion | BD building approval |
| BD | GBP referral for coordinated utility drawings | Undertaker field approval |

## Decision tree — protection vs diversion

```
Excavation / hoarding near recorded telecom plant?
├─ No conflict → Document search result; proceed with standard PCI note
├─ Plant exposed but intact → Protection scheme to undertaker; trial hole if uncertain
└─ Plant must move → Diversion application; programme float; as-built records
```

## Typical workflow (lane 1)

1. **Record search** — undertaker plans, trial holes, GPR if budget allows.
2. **Protection notice** — before piling, hoarding footings, or bulk excavation.
3. **Diversion agreement** — if reroute required; sequence with `hk-construction-programme` lane 1–4.
4. **Site supervision** — no excavation within protection zone without undertaker witness where required.
5. **As-built** — update utility record drawings for handover and CC external works.

## Document register

| Document | Owner | When |
|----------|-------|------|
| Utility search report | PM / MEP | Before hoarding design freeze |
| Protection proposal | Telecom consultant / contractor | Before lane 1 start |
| Undertaker approval letter | Undertaker | Before excavation in zone |
| Diversion completion certificate | Undertaker | Before backfill / road reopening |
| Record drawing update | AP / MEP | BA14 / CC package |
