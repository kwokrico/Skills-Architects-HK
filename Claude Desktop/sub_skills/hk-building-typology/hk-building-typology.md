# HK Building Typology

Deep-dive on Hong Kong's characteristic building types: public housing, MTR TOD, composite buildings, village houses, and commercial typologies.

---

## 1. Public Housing — Housing Authority (HA)

### 1.1 Harmony Block Types

| Type | Configuration | Units/Floor | Typical Height |
|---|---|---|---|
| Harmony 1 | Cruciform; 4 wings | 8 | 35–40 storeys |
| Harmony 2 | Modified cruciform | 6–8 | 35–40 storeys |
| Non-standard (NS) | Site-specific; irregular | Varies | Up to 40 storeys |

### 1.2 HA Unit Size Standards

| Flat Type | Persons | Min Internal Floor Area |
|---|---|---|
| 1-person/2-person (1P/2P) | 1–2 | 14 m² (1P); 21 m² (2P) |
| 2-person/3-person (2P/3R) | 2–3 | 28 m² |
| 3-person/4-person (3P/4R) | 3–4 | 35 m² |
| 4-person/5-person (4P/5R) | 4–5 | 43 m² |

> HA allocates by household size; design must accommodate full range of flat types per brief.

### 1.3 HA Design Requirements

- Cross-ventilation: all units dual-aspect or single-aspect with ventilated kitchen/bathroom
- Communal facilities: estate management office, refuse collection point, recreational area
- Accessibility: DMBA 2008 throughout; lift to all floors
- Structural: RC shear wall system; typical grid 3.3–3.6 m

## 2. Public Housing — Housing Department (HD)

- HD manages rental public housing estates (distinct from HA Home Ownership Scheme)
- Design standards similar to HA; HD issues its own schedule of accommodation
- Typical typologies: slab blocks (older estates), point towers (newer)

---

## 3. MTR Transit-Oriented Development (TOD)

### 3.1 Rail + Property Model

| Aspect | Description |
|---|---|
| Development rights | MTR Corp acquires land at pre-rail value; develops above/adjacent to stations |
| Typical structure | Retail podium (L1–L5) + residential/office towers; station concourse at B1/B2 |
| Pedestrian network | Covered walkway linking station exits to towers and surrounding streets |
| Key examples | Tung Chung, Tseung Kwan O, Tin Shui Wai, LOHAS Park |

### 3.2 TOD Design Drivers

- Station box structure governs tower grid above (typically 8.1–9.0 m)
- Transfer slab at podium/tower interface: coordinate with MTR structural requirements
- Noise: rail vibration isolation required for residential floors near tracks
- Retail podium: anchor tenant strategy; atrium daylight; FEHD licensing for F&B

---

## 4. Composite Buildings

Residential over commercial podium — the dominant HK urban typology.

### 4.1 Key Design Issues

| Issue | Guidance |
|---|---|
| Structural transfer | Transfer slab at L3–L5; fix level at concept; coordinate grid, risers, unit module |
| Separate lobbies | Residential lobby must be separate from commercial entrance (BO / management) |
| Acoustic separation | STC 60+ floor assembly between commercial and residential; floating slab for kitchen/bar |
| Servicing | Commercial loading dock separate from residential drop-off |
| Fire compartmentation | Separate fire compartments for residential and commercial portions |
| Plot ratio | Domestic and non-domestic PR calculated separately per OZP |

## 5. NT Exempted Houses (Village Houses)

| Parameter | Limit |
|---|---|
| Max storeys | 3 |
| Max floor area per storey | 65.03 m² |
| Max height | 8.23 m (27 ft) |
| Max site coverage | 66.7% |
| Eligible applicants | Indigenous villagers with Small House Policy entitlement |
| Lift | Not required |

> Strictly enforce all four limits simultaneously — BD rejects if any one is exceeded.

## 6. Grade A Office Tower

| Metric | Value |
|---|---|
| Typical floor plate | 1,500–2,500 m² NFA |
| Floor-to-floor height | 4.0–4.2 m |
| Core-to-facade depth | ≤ 15 m |
| Raised floor void | 150–200 mm |
| Ceiling void | 600–750 mm |
| OTTV target | ≤ 20 W/m² (BEAM Plus) |
| Lift interval target | ≤ 30 sec (CIBSE Guide D) |
| Key submarkets | Central, Admiralty, Wan Chai, Kowloon East (KE) |

## 7. Heritage Adaptive Reuse

| Aspect | Guidance |
|---|---|
| AAB grading | Grade 1 (highest) / Grade 2 / Grade 3; check before any intervention |
| AMO consultation | Required for declared monuments and proposed Grade 1 buildings |
| Revitalisation scheme | Govt scheme for historic buildings; social enterprise operators |
| Key principle | Retain character-defining elements; reversible interventions preferred |
| GFA exemption | Existing GFA of heritage building may be retained even if exceeding current PR |
| Examples | PMQ (Police Married Quarters), Tai Kwun (Central Police Station), Blue House (Wan Chai) |
