# HK Concept Design

Concept design in HK is shaped by OZP constraints, BD statutory submissions, and high-density typology conventions. Appoint BEAM Pro and RSE at concept stage.

---

## 1. BD Submission Stages

| Stage | BD Milestone | Key Deliverables |
|---|---|---|
| Feasibility | Pre-application consultation (optional) | Site analysis, OZP check, BHR, PR |
| Concept | Preliminary Approval (PA) application | Massing, GFA schedule, MOE strategy |
| Scheme | Building Plan (BP) submission | General Building Plans, drainage, structural |
| Detail | Consent to Commence Works (CC) | Working drawings, structural calculations |
| Construction | Supervision plan, site inspections | AP/RSE/RC site duties |
| Completion | Occupation Permit (OP) application | As-built drawings, FSD/BD inspections |

---

## 2. OZP Massing Constraints (Concept Checklist)

- [ ] Zone confirmed: R(A/B/C), C(1/2), OU, G/IC
- [ ] Plot ratio (domestic + non-domestic) checked against OZP annotation
- [ ] Building Height Restriction (BHR) in mPD or storeys confirmed
- [ ] Site coverage limit confirmed (typically 33% for residential towers)
- [ ] s.16 planning permission required? (Column 2 uses or PR/BHR exceedance)
- [ ] Aviation height restriction checked (CAAS) for Kowloon/Lantau sites

---

## 3. Urban Design Drivers (HKPSG Ch. 11)

| Driver | Guidance |
|---|---|
| Ridgeline protection | No visual intrusion on natural ridgelines from key vantage points |
| Harbour view corridors | Maintain corridors to Victoria Harbour per HKPSG Fig. 11.1 |
| Podium height | ≤ 15 m to maintain street-level human scale |
| Building separation | Min 15 m between residential towers (HKPSG recommendation) |
| Air ventilation | AVA required for major urban sites; design for S/SE summer breeze |
| Active frontage | Ground floor commercial/community use along primary streets |

---

## 4. HK Typology Vocabulary

| Element | Description |
|---|---|
| Podium-tower | Retail/carpark podium (L1–L5) + residential/office tower above |
| Composite building | Residential over commercial; separate lobbies; transfer slab |
| Transfer slab | Structural transfer at podium/tower interface (L3–L5); fix at concept |
| Scissor stair | Two interlocking staircases in one shaft; standard HK high-rise residential |
| Sky garden | GFA-exempt communal amenity floor; BEAM Plus SA credit |

---

## 5. BEAM Plus at Concept

| Action | Timing |
|---|---|
| Appoint BEAM Pro | At concept stage (before BP submission) |
| Target credit level | Confirm Bronze/Silver/Gold/Platinum at concept |
| SA credits | Site orientation, AVA, greenery ratio (PNAP APP-40) |
| EU credits | Fix OTTV strategy: orientation, shading, glazing spec |
| IEQ credits | Natural ventilation feasibility; dual-aspect unit layout |

---

## 6. MTR TOD Concept Integration

| Aspect | Approach |
|---|---|
| Station box | Tower grid must clear station box structure; coordinate with MTR Corp |
| Covered walkway | Design from MTR exit to building lobby at concept; level changes via lift/ramp |
| Retail podium | Anchor tenant strategy; atrium daylight; FEHD F&B licensing |
| Noise/vibration | Floating slab for residential above MTR tunnels; coordinate with MTR Corp |
