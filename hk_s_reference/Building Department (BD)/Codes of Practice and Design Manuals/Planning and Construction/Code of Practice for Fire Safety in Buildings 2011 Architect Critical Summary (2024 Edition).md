# Code of Practice for Fire Safety in Buildings 2011
**Architect critical summary for schematic design**  
2024 Edition (incorporates amendments through September 2024) | BD Code of Practice

> Scope note: This Code is the **unified MOE + FRC + means of access rulebook** replacing the former MOE 1996, FRC 1996 and MOA 2004 Codes. It is the Deemed-to-Comply path for Building (Construction) Regulation s.35 and Building (Planning) Regulations 41, 41A–41D. Read with the FSD *Code of Practice for Minimum Fire Service Installations and Equipment* for active systems (sprinklers, hose reels, alarms). Licensed premises also follow Annex A licensing guides.

---

## Regulatory Overview

This Code applies to **all buildings** except single-family domestic buildings of not more than three main domestic storeys with highest floor not more than **13 m** above ground (Part B exemption). It sets Use Classifications 1–8, occupant capacity, exit geometry, fire compartment / FRR limits, fireman’s lifts, firefighting & rescue stairways, and EVA — i.e. the passive fire-safety envelope that locks massing, core layout, floor plates, and site access at schematic design.

Compliance is by Deemed-to-Comply provisions (Parts B–E), Alternative Solution via fire engineering (Part G), or a combination. Change of Use Classification or of Bounding Conditions may require BA submission under Cap. 123 s.25 / Part G.

## Critical main topics and subtopics

### 1. Use Classification — lock this first (Table A1)

Every FRR, exit width, travel-distance rule, and access provision is keyed to Use Classification. Assign early; composite = most stringent mix per Part C / D.

| Class | Label | Typical inclusions (schematic-critical) |
|---|---|---|
| **1** Residential | 1a House (≤3 storeys, single family); **1b Flats** (incl. service apt.); 1c Tenement |
| **2** Transient sleeping | Hotels, guesthouses, dormitories, hostels, barracks, motels |
| **3** Institutional | 3a Health/child care / elderly homes; 3b Detention / correctional |
| **4** Commercial | **4a** Offices; **4b** Retail, F&B, banking halls, markets, karaoke, clubs |
| **5** Assembly | **5a** PPE (Cap. 172 — cinemas, theatres, stadia…); **5b** Schools/libraries; **5c** Transport terminals; **5d** Halls, churches, gyms, columbaria |
| **6** Industrial | 6a Workplaces / commercial kitchens & labs; 6b Warehouses / logistics; 6c Hazardous DG storage / manufacturing |
| **7** Carparks | Parking structures / garages |
| **8** Plant & the like | M&E plant, IT, TBS rooms (often treated as low occupancy) |

**SD takeaway:** Flat lobby / office podium / retail / carpark stacking = different compartments, exits, FRR and often separate stair / fireman’s-lift strategies. Do not borrow “office” numbers for F&B or PPE.

### 2. Occupant capacity — design load (Table B1)

Use **usable floor area** (excl. staircases, public circulation, lift landings, lavatories, WCs, flat kitchens, plant for lifts/AC) unless Notes say otherwise. Actual counting accepted by BA.

| Use | Occupancy factor (m²/person) or basis |
|---|---|
| Flats 1b — ≥5 flats per staircase per floor (corridor/balcony access) | **4.5** |
| Flats 1b — other | **9** |
| Tenement 1c | **3** |
| Hotel/guesthouse sleeping | No. of **bedspaces** |
| Dormitories | **3** |
| Offices / staff rooms / board rooms | **9** / **9** / **10** |
| Retail G–2/F incl. arcade | **3**; 3/F & above **4.5** |
| Markets, showrooms, jewellery/pawn/money changers | **2** |
| Café / restaurant / bar / lounge | **1** |
| Banking / betting / public counters (public areas) | **0.5** |
| Cinema / theatre seating | No. of **seats**; foyer **0.5** |
| Exhibition / museum / art gallery | **2** |
| Library | **2**; reading/study **1**; classroom/lecture **2** or seats |
| Factory / workshop / commercial kitchen | **4.5** |
| Warehouse / DG / carpark / plant (≥100 m² net for Class 8) | **30** |
| Class 8 room ≤100 m² net | Occupant capacity **zero** |

**SD takeaway:** Retail lower floors and F&B at 1 m²/person drive exit banks and stair cores — size cores from the densest floor, not the average.

### 3. Exit count & clear widths (Table B2) — hard geometry

| Occupants in room / compartment / storey | Min no. exits | Min total width exit doors / routes (mm) | Min each door / route (mm) |
|---|---|---|---|
| 4–30 | 1 | 750 / 1050 | — |
| 31–200 | 2 | 1750 / 2100 | 850 / 1050 |
| 201–300 | 2 | 2500 / 2500 | 1050 / 1050 |
| 301–500 | 2 | 3000 / 3000 | 1050 / 1050 |
| 501–750 | 3 | 4500 / 4500 | 1200 / 1200 |
| 751–1000 | 4 | 6000 / 6000 | 1200 / 1200 |
| Then +1 exit & +1500 mm total per +250 persons | … | … | 1350 → 1500 |

- Uneven exit group: discard any door/route width **>50% above the narrowest** when summing totals.
- PPE Class 5a: use Section 3 / Table B5 when ≥12 m above G/F — not Table B2 alone.
- >3000 persons on a floor: expect fire engineering, not Table B2 extrapolation.

### 4. Single-staircase buildings (B6) — rarely usable at SD

Only if **all** apply:

| Criterion | Limit |
|---|---|
| Height | ≤ **6 storeys** and uppermost floor ≤ **17 m** above discharge ground |
| Use above G/F | Domestic or offices only (G/F may be shop/carpark with proper enclosure) |
| Highest floor ≤13 m | Usable floor / storey above G/F ≤ **250 m²** |
| Highest floor >13 m | Usable floor / storey ≤ **150 m²** + protected lobby to stair + stair to **roof** + roof refuge ≥ **0.5 m²/person** for all above-G/F occupants |

Otherwise **≥2 exit routes / storey** (B8).

---

### 5. Travel distance & stair separation (B10–B11) — core drivers

Measure along centreline of actual exit route (not diagonal). “Alternative direction” requires ≥**30°** angle between lines to two exits (B11.6–B11.7).

#### 5.1 Dead-end limits (B11.2)

| Use | Dead-end rule |
|---|---|
| **1 & 2** | ≤**24 m** within flat/guestroom to flat door; then ≤**15 m** from flat door to staircase or point of choice |
| **3** | ≤**12 m** to protected exit / point of choice (**24 m** on compliant balcony approach) |
| **Other** | ≤**18 m** to protected exit / point of choice (**24 m** on balcony approach) |

#### 5.2 Maximum travel where ≥2 protected exits available (B11.3)

| Use | Max travel (internal) | Along balcony approach |
|---|---|---|
| **1 & 2** | **24 m** from flat/guestroom exit door to nearest required staircase | **45 m** |
| **3** | **30 m** to nearest protected exit | **45 m** |
| **Other** | **36 m** to nearest protected exit | **45 m** |

#### 5.3 Stair location rules (layout killers)

| Rule | Number |
|---|---|
| Min separation between required staircases (door-to-door / landing) | **≥ 6 m** (straight-line along wall or route centreline, lesser) |
| Max separation between staircases / discharge points | **≤ 48 m** along exit-route centreline |
| Inter-stair access (if ≥2 stairs) | At least every **5 storeys**, and on roof / top accessible floor — **or** at refuge floors + roof |
| Do not escape through one stair enclosure to reach another | B10.3 |

#### 5.4 Protected lobby to staircase (B10.4) — when required

Provide protected lobby to each required staircase unless:

- single-stair building with highest floor ≤**13 m**; or
- ≥2 staircases and highest floor ≤**20 m**; or
- pressurization to FSD CoP; or
- ventilated staircase (B10.5: ≥50% perimeter open, ≥50% of that on long face, ≥25% other faces).

Basement stairs need protected lobby unless pressurized or independent upward discharge (B17.5).

### 6. Exit-route construction basics (B5, B14–B17)

| Item | Rule |
|---|---|
| Headroom | ≥ **2000 mm** clear |
| Lighting | ≥ **30 lux** at floor + FSD emergency lighting |
| Ultimate place of safety | Street or open area ≥**1.5 m** wide (or ≥ total discharging exit width), connecting to street for dispersal |
| Private lane discharge | Only if paved, permanently unobstructed, BA accepts integrity |
| Discharge return walls | ≥ **450 mm** return/projection where discharge adjoins other exits/accommodation |
| Stair flights | Straight; **2–16** risers/flight; no winders |
| Going / rise (general) | Going ≥**225 mm**; rise ≤**175 mm** |
| Going / rise Class 5a | Going ≥**280 mm**; rise ≤**150 mm** |
| Going / rise Class 5b | Going ≥**250 mm**; rise 75–150 mm |
| Max stair clear width (single flight) | **1800 mm** — wider → split by central handrail into 1050–1800 mm sections |
| Handrails | Both sides; height **850–1100 mm**; projection ≤**90 mm** each |
| Ramp in exit route | Max gradient **1:12** |
| Doors >30 occupants | Open in direction of escape (or bi-swing with vision panel) |
| Min exit door leaf | ≥**750 mm** clear (≥**600 mm** per leaf if double) |
| Lift lobby | Unobstructed access to exit route at all times, or intercom + ≥2 h emergency power |

#### Basements (B17)

- Normally ≥**2** exits (exception: floor ≤3 m below discharge ground, ≤150 m², lavatory/plant only).
- Upper-floor escape stairs must **not** continue directly into basement.
- ≥1 exit per basement floor should serve **that floor only** and discharge independently upward (except lavatory-only).

### 7. Staircase discharge value (B12 / Tables B3–B4)

Sum of discharge values of all required stairs serving floors above G/F ≥ total occupant capacity of those floors (same for basement stairs vs basement capacity).

- Use **Table B3** if building (or shared stair) is **not** fully sprinklered; **Table B4** if whole building sprinklered to FSD CoP.
- Sprinklered discharge ≈ **2×** non-sprinklered for 1-storey served, then same incremental add per extra storey.
- Upward escape: × **0.8**. Scissor stair with no intermediate landing between floors: × **0.7**.

**Snapshot — persons discharged (selected):**

| Stairs serve | Width 1050–<1200 (non-sp / sp) | Width 1200–<1350 (non-sp / sp) |
|---|---|---|
| 1 storey | 210 / **420** | 240 / **480** |
| 10 storeys | 498 / **708** | 582 / **822** |
| Each extra storey add | +32 / +32 | +38 / +38 |

**SD takeaway:** Without whole-building sprinklers, tall residential cores need more / wider stairs much sooner.

### 8. Refuge floors (B18 + C17) — tall-building massing

| Trigger | Provision |
|---|---|
| Building **>25 storeys** above lowest ground storey | Refuge floors required (plant-only storeys may be excluded from count) |
| Spacing — Use **6** Industrial | ≤ **20 storeys** from another refuge / street / UPS |
| Spacing — other Uses | ≤ **25 storeys** |
| Domestic / composite **26–40 storeys** | Main roof as refuge (B18.3–B18.4) may replace intermediate refuge |

**Intermediate refuge must (inter alia):**

- No occupied accommodation / accessible plant at same level (except FS tanks / FS plant).
- Net refuge area ≥ **50%** of refuge-floor GFA; headroom ≥ **2300 mm**.
- Min dimension of refuge area ≥ **1.5×** widest passing stair width.
- Open-sided above parapet on ≥ **2 opposite sides** for cross-ventilation; 6 m separation rules vs opposing façades / boundaries (C17.2).
- Required stairs **discontinued** — divert across refuge before continuing down.
- Served by fireman’s lift only (doors locked until fireman’s switch) — ordinary lifts not onto refuge (unless sky-garden package under C17 commentary).
- FRR of walls/floors separating refuge: **-/120/120** walls, **120/120/120** floors (C17.1).

### 9. Disability temporary refuge (B29–B30)

Applies unless DMBA / site exemptions (small site ≤500 m² floors ≤200 m² or ≤2 units; ventilated balcony-approach without fireman’s lift; refuge floor; G/F direct to UPS, etc.).

| Item | Rule |
|---|---|
| Provision | ≥1 temporary refuge **1.5 × 1.5 m** in protected exit or fireman’s-lift lobby per fire compartment / floor (or 2 × 0.75×1.5 m mutually visible) |
| Coverage | No part of floor > **60 m** from a temporary refuge |
| Door clear width to space | ≥ **850 mm** (or Table B2 if greater) |
| CCTV + intercom | Emergency power ≥ **1 hour** to management office / FS control panel |

### 10. Fire compartment limits & FRR (Table C1) — plate sizing

| Use | Max compartment area / volume | Min FRR (min) |
|---|---|---|
| 1 Residential | Not limited (each flat still separated) | **60** |
| 2 Hotel | Not limited (each guestroom separated) | **60** |
| 3 Institutional | ≤ **2,500 m²** | **60** |
| 4a Business | ≤ **10,500 m²** | **60** |
| 4b Mercantile | ≤2,500 m² → **60**; >2,500 to ≤10,500 m² → **120** | as shown |
| 5a / 5d Assembly | ≤2,500 → **60**; >2,500–10,500 → **120** | |
| 5b Educational | same banding | 60 / 120 |
| 5c Transport | ≤ **10,500 m²** | **120** |
| 6a Industrial workplaces | ≤ **10,500 m²** | **120** |
| 6b Warehouse | ≤ **28,000 m³** and ≤10,500 m² | **120** |
| 6c Hazardous DG | ≤ **7,000 m³** | **120** |
| 7 Carpark | ≤ **10,500 m²** | **60** |

Different Use Classifications / different occupancies → separate by **longer** of the two FRRs (C7). Exceptions: small ancillary offices in industrial; retail shops not needing shop-to-shop separation in arcade if under common management/alarm (C7.3–C7.4).

Oversized compartments → case-by-case enhanced MOE / MOA / FSI / FRC (Note 1).

### 11. Flat / corridor / special hazard details (C6, C11, C13)

| Topic | SD constraint |
|---|---|
| Flat/guestroom doors to common corridor | FRR of corridor barriers + **smoke seals** (C6) |
| Spandrel (unsprinklered; not single family) | **900 mm** vertical or **500 mm** horizontal projection with FRR of intervening storey (C11.1) — **waived if sprinklered** |
| Area of special hazard | Separated **-/120/120**; if opens to protected exit → **-/240/240** |
| Restaurant kitchen | Enclosure **-/60/60**; protected lobby to building exit route and (if kitchen >**45 m²**) to dining exit route |
| Open kitchen in Class 1 (single exit) | Smoke detectors (in unit + common), sprinklers over notional kitchen, **600 mm** full-height **-/30/30** wall beside exit door, etc. (C13.4) — budget FSI & layout early |

### 12. Opening protection, atriums, basements (C8–C10, C14)

| Topic | Rule |
|---|---|
| Openings in compartment walls | Fire doors/shutters at wall FRR; if openings >**25%** of wall length → insulation criterion applies (can drop insulation to 30 min if sprinklers both sides) |
| Floor voids in sprinklered buildings | **450 mm** non-combustible downstand **-/30/-** around opening, or approved smoke curtain |
| Curtain-wall perimeter void | Seal with FRR ≥ floors + D-stability / leakage limits (C10.2) |
| Atrium (sprinklered) | Separated by FRR barriers; volume ≤**28,000 m³**; linked height ≤**15 m**; ≤**3** interconnected floors; sprinklers at atrium base |
| Basement vs G/F | Compartment **-/240/240**; basement elements of construction **240/240/240**; internal basement walls **-/120/120** |
| Basement smoke outlets (no dynamic extract) | ≤**30 m** apart; aggregate ≥**0.5%** floor (warehouse **2.5%**); least dim ≥**1 m** |

### 13. Means of access — fireman’s lift & FRS (Table D1 + Part D)

Threshold sketch (confirm exact cell of Table D1 for your Use / height / cubical extent):

| Building type (simplified) | Fireman’s lift | Firefighting & Rescue Stairway |
|---|---|---|
| Domestic / composite domestic >**30 m** | Yes — within **60 m** of any part of floor | No (typical) |
| Non-domestic (not 5a/hotel/hospital/6) >**30 m**, or >15 m & >7000 m³ (etc.) | Yes — ≤60 m coverage | No (typical) |
| **5a / hotel / hospital** >2 storeys | Yes — ≤60 m | No (typical) |
| **Use 6** industrial when height/volume triggers (often >30 m or >7000 m³) | Yes | **Yes** — ≤60 m |
| Deep / large basements (>2 storeys & >7000 m³, or depth >9 m & >7000 m³) | Yes | Yes |

**Fireman’s lift schematic checklist**

| Item | Number |
|---|---|
| Ground access to lift / FRS | Perimeter street or open area ≥**1.5 m** min dimension; setback path ≤**18 m** horizontal |
| Access passage | Width ≥**1500 mm**; headroom ≥**2000 mm**; FRR walls + protected lobby if communicating |
| Floor coverage | No point >**60 m** from lobby door along routes (**45 m** straight-line allowed for unknown fit-out at plan stage) |
| Lift car | Min dim **1.1 m**; floor ≥**1.35 m²**; load ≥**680 kg**; door ≥**800 × 2000 mm** |
| Lobby (non-FRS fireman’s lift) | ≥**2.25 m²**, min dim **1.5 m**; FRR ≤120 min; direct to protected exit |
| Pattern | Alternate floors OK for single lift; refuge floors must be served |

**FRS lobby (D16)** — when FRS required: lobby **5–10 m²**, min dim 1.5 m; external wall + natural ventilation ≥**25%** lobby area (or smoke outlets / pressurization alternatives); stair ↔ lift communicate every floor; stair width ≥**1050 mm**.

### 14. Emergency Vehicular Access — EVA (D22–D23 / B(P)R 41D)

Design vehicle: **30,000 kg** gross; **26 m** turning circle; **12 m** length. Dead-end EVA needs turning space. Gradient ≤**1:10** (≤**1:6** only where not serving major façade). Clear headroom under structures ≥**4.5 m**.

| Site / use | EVA geometry (schematic) |
|---|---|
| **Virgin site** (and later redevelopments of same) | Carriageway ≥**7.3 m** (or hard-paved non-carriageway ≥**6 m**); serve ≥**¼** perimeter as “major façade”; façade served only if EVA ≤**10 m** from façade (waived above 60 m façade height) |
| Virgin + **Use 6** | **Two opposite** facades each ≥¼ perimeter; 2-way carriageway ≥**13.5 m** undivided (or 7.3 m each side if divided); preferably from different streets |
| Virgin + **Use 5a** | Thoroughfare widths per **B20** (e.g. >500–2000 persons → one ≥**12 m**; >2000–3000 → 12 m + 9 m carriageway / 6 m footway, etc.) |
| **Redevelopment** (most urban sites) | Use 6: major façade, ≥**7.3 m**; other Uses: major façade, ≥**6 m**; 5a still follows B20 |

Crash gates / demarcation per Diagrams D7–D8 when EVA is not day-to-day vehicular access. Exemptions / enhanced FSI packages: D25–D26.

### 15. PPE / cinema extras (B19–B27, C18) — if Use 5a

- Site should abut **≥2 thoroughfares**; frontages ≥½ perimeter enabling exits to those streets (relaxed ≤500 persons / non-industrial with one street EVA + FSD OK — B20.9).
- Storey ≥**12 m** above G/F → Table **B5** exit widths (stricter than B2).
- Small cinema ≤50 persons (aggregate ≤150 in building) may share non-domestic exits if not in basement and foyer/projection rules met (B19.2).
- Auditorium as own fire compartment; dedicated smoke extraction typically required when sharing building (C18).

---

### 16. Fire engineering & Bounding Conditions (Part G / A5)

Use Alternative Solution when size, use, complexity or location make DTC impractical (e.g. mega atriums, oversized compartments, complex MOE).

- Pre-submission enquiry → Fire Safety Assessment Report (FSAR).
- **Bounding Conditions** (occupancy, compartment size, FSI, MOE geometry, management) must appear on GBP and be maintained; change invalidates the performance design and may require BA re-approval.

Part F (management / maintenance / training / fire action plans) is operational — flag for OP / IO manuals, not massing.

### 17. Schematic design checklist (day-one)

1. Assign **Use Classification(s)** per floor / fire compartment; flag composite interfaces.
2. Run **Table B1** occupants on worst floors → **Table B2** exit widths → place **≥2** stairs (**6–48 m** apart).
3. Check **dead-end / travel** envelopes before committing corridor layout.
4. Decide **sprinkler / pressurization** strategy early (unlocks B10 lobby omission, C11 spandrel waiver, Table B4 discharge).
5. If >**25 storeys**, reserve **refuge** (or roof-refuge strategy for 26–40 storey domestic/composite).
6. Apply **Table C1** FRR / compartment caps; separate uses / special hazards.
7. Size **fireman’s lift / FRS** from Table D1; keep ≤**60 m** coverage; protect G/F FS access.
8. Trace **EVA** path to ≥¼ façade (stricter for industrial / PPE / virgin sites).
9. Provide **temporary refuge spaces** in cores for DMBA-applicable floors.
10. Cross-check FSD FSI CoP and licensing guides (Annex A) for the intended tenancies.
