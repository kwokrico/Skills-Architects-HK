# Code of Practice for Fire Safety in Buildings 2011
**Architect critical summary for schematic design**  
2024 Edition (amendments through September 2024 via PNAP APP-153) | Buildings Department

> Scope note: Unified **MOE + FRC + means of access** Code replacing MOE 1996, FRC 1996 and MOA 2004. Deemed-to-Comply path for **Building (Construction) Regulation s.35** and **Building (Planning) Regulations 41, 41A–41D**. Always read with the FSD *Code of Practice for Minimum Fire Service Installations and Equipment* (active systems). Licensed premises also follow Annex A licensing guides (hotel, karaoke, RCHE, education, etc.).

---

## Regulatory Overview

This Code applies to **all buildings** except single-family domestic buildings of not more than three main domestic storeys with highest floor not more than **13 m** above ground (Part B exemption B3.1). It sets Use Classifications 1–8, occupant capacity, exit geometry, fire compartment / FRR limits, fireman’s lifts, firefighting & rescue stairways (FRS), and EVA — the passive fire-safety envelope that locks massing, core layout, floor plates, and site access at schematic design.

Compliance is by Deemed-to-Comply (Parts B–E), Alternative Solution via fire engineering (Part G), or a combination. Change of Use Classification or of Bounding Conditions may require BA submission under Cap. 123 s.25 / Part G. Arson/terrorism scenarios are outside DTC (A6).

## Critical main topics and subtopics

### 0. Definitions that lock schematic decisions (Part A §3)

| Term | Meaning — why it matters at SD |
|---|---|
| **Use Classification** | Table A1 categories 1–8 (and sub-classes). Every travel, FRR, and access rule keys off this — assign before cores. |
| **Usable floor area** | Floor area **excluding** staircases, public circulation, lift landings, lavatories, WCs, **kitchens in flats**, and plant for lifts/AC. Occupant capacity uses this (unless Notes say net / water surface / bedspaces). |
| **Occupant capacity** | Design population of room / storey / fire compartment from Table B1 (or actual count). Maisonette entrance storey includes all maisonette floors. |
| **Travel distance** | Horizontal centreline of exit route from furthest point to stair door centre / first tread / UPS discharge. **Not** a straight diagonal. |
| **Deadend travel distance** | Length before an alternative exit / point of choice exists. |
| **Protected exit** | Required staircase, ramp or passageway enclosed by fire barriers leading to ultimate place of safety. |
| **Protected lobby** | Lobby of fire-and-smoke-resisting construction (Part C / C16.5). |
| **Required staircase** | Escape stair **or** access stair in an FRS. |
| **Ultimate place of safety (UPS)** | Street or open area outside the building connecting to a street; width ≥ **1.5 m** or ≥ total discharging exit width, whichever greater. |
| **Fire compartment** | Space fully enclosed by fire barriers so fire does not spread in/out. |
| **Compartment area / volume** | Floor area / volume within external surfaces of the compartment — atria and double-height space inflate volume fast. |
| **FRR (X/Y/Z)** | Stability / Integrity / Insulation minutes (ISO 834 / BS 476 Pts 20–24). |
| **Element of construction** | Floor, beam, column, hanger; loadbearing wall; **required staircase** incl. landings/supports. Raised floor ≤ **600 mm** is **not** an element of construction — compartment walls start from structural floor. |
| **Special hazard** | Higher ignition risk: transformer, central AC plant, lift machine, main switch, generator, boiler, DG store, fuel tank, etc. |
| **Refuge floor** | Protected temporary assembly floor before UPS. |
| **Firefighting and rescue stairway (FRS)** | Stairway containing an access staircase **and** a fireman’s lift. |
| **EVA** | Access for FSD vehicles to the building for firefighting / emergency. |
| **Virgin site** | Site with no temporary or permanent building previously (incl. NTEH-exempt). |
| **Redevelopment site** | Not a virgin site — most urban plots. |
| **Bounding Conditions** | Fire-engineering package that must be maintained; change invalidates the Alternative Solution. |
| **Balcony approach** | External approach to a common staircase serving ≥2 occupancies. |

**SD takeaway:** Wrong Use Class, wrong “basement” call, or measuring travel as a diagonal are the most expensive early mistakes.

### 1. Compliance framework & when DTC fails (A1–A5)

| Path | When |
|---|---|
| **Deemed-to-Comply** (Parts B–E) | Standard buildings — reliable BA compliance |
| **Alternative Solution** (Part G) | Genuine difficulty due to size, use, complexity or location |
| **Combination** | Allowed |

BA assesses Alternative Solutions against risk of fire occurring, severity, structural resistance / fire & smoke spread, and danger to people in/around the building (A3 commentary).

**Change triggers (A5):** material change of Use Classification → Cap. 123 s.25; change to Bounding Conditions → review / possible re-submission.

### 4. Exit count & clear widths (Table B2)

|---|---|---|---|
| 4–30 | 1 | 750 / 1050 | — |
| 31–200 | 2 | 1750 / 2100 | 850 / 1050 |
| 201–300 | 2 | 2500 / 2500 | 1050 / 1050 |
| 301–500 | 2 | 3000 / 3000 | 1050 / 1050 |
| 501–750 | 3 | 4500 / 4500 | 1200 / 1200 |
| 751–1000 | 4 | 6000 / 6000 | 1200 / 1200 |
| 1001–1250 | 5 | 7500 / 7500 | 1350 / 1350 |
| 1251–1500 | 6 | 9000 / 9000 | 1350 / 1350 |
| 1501–1750 | 7 | 10500 / 10500 | 1500 / 1500 |
| 1751–2000 | 8 | 12000 / 12000 | 1500 / 1500 |
| 2001–2250 | 9 | 13500 / 13500 | 1500 / 1500 |
| 2251–2500 | 10 | 15000 / 15000 | 1500 / 1500 |
| 2501–2750 | 11 | 16500 / 16500 | 1500 / 1500 |
| 2751–3000 | 12 | 18000 / 18000 | 1500 / 1500 |

**Critical measurement / sizing rules:**
- Door width = least clear width between vertical frame members.
- Route width = between finished walls / inner balustrade; only handrails may project (≤**90 mm** each).
- Uneven exit group: discard any door/route width **>50% above the narrowest** when summing totals (B7.3 / B8.1).
- ≤200 persons may share a single corridor/balcony approach **if** escape is possible in more than one direction (B7.2).

### 5. Single-staircase buildings (B6) — rarely usable

Only if **all** apply:

| Criterion | Limit |
|---|---|
| Height | ≤ **6 storeys** and uppermost floor ≤ **17 m** above discharge ground |
| Use above G/F | Domestic or offices only (G/F may be shop/carpark with Part C enclosure) |
| Highest floor ≤13 m | Usable floor / storey above G/F ≤ **250 m²** |
| Highest floor >13 m | Usable floor / storey ≤ **150 m²** + protected lobby + stair continued to **roof** + roof refuge ≥ **0.5 m²/person** for all above-G/F occupants + roof per C12 |

Otherwise every storey needs **≥2** exit routes (B8), or more per Table B2.

### 6. Ground-storey discharge & stair access (B9–B10)

#### 6.1 Ground storey (B9)

| Rule | Detail |
|---|---|
| Stair enclosure continued to discharge | Separate passage/corridor from remainder; openings via **protected lobby** |
| Exception (≥2 stairs) | Cloakroom, lavatory, WC, caretaker office/counter, fire control room may open off passageway |
| Exception (≥3 stairs) | 1 in 3 stairs may discharge through fire-rated doors to entrance hall (hall portion still meets B5) |
| Shared ground exit with stair | Width ≥ ½ ground-storey exit width **+** upper-stair width **+** basement-stair width |

#### 6.2 Access to stairs (B10)

| Rule | Number / detail |
|---|---|
| Approach from different directions | Deadends only as B11.2 allows |
| Min stair-to-stair separation | **≥ 6 m** (door centre / landing perimeter; lesser of wall path or route centreline) |
| Must not pass through one stair enclosure/lobby/landing to reach another | B10.3 |
| Door swing into corridor | Must not reduce required corridor width |
| Protected lobby to each stair | Required unless: single-stair ≤13 m highest floor; **or** ≥2 stairs & highest floor ≤**20 m**; **or** pressurization (FSD CoP); **or** ventilated staircase (B10.5) |
| Ventilated staircase (B10.5) | ≥**50%** perimeter open to external air; ≥**50%** of opening on long face; ≥**25%** on other faces; openings from top of balustrade to soffit of flight above; evenly distributed |
| Balcony approach (B10.6) | Part C protection; landing not part of balcony; door swing must not reduce width; windows 180° fix-open or fanlights ≥2000 mm if two-direction escape |

### 7. Travel distance (B11) — core drivers

Measure along exit-route centreline. “Alternative direction” / point of choice requires ≥**30°** angle between lines to two exits (B11.6–B11.7).

#### 7.1 Dead-end limits (B11.2)

| Use | Dead-end rule |
|---|---|
| **1 & 2** | ≤**24 m** within flat/guestroom to exit door; then ≤**15 m** from that door to staircase or point of choice |
| **3** | ≤**12 m** to protected exit / point of choice (**24 m** on compliant balcony approach) |
| **Other (4–8)** | ≤**18 m** to protected exit / point of choice (**24 m** on balcony approach) |

#### 7.2 Maximum travel where ≥2 protected exits available (B11.3)

| Use | Max travel (internal) | Along balcony approach |
|---|---|---|
| **1 & 2** | **24 m** from flat/guestroom exit door to nearest required staircase | **45 m** |
| **3** | **30 m** to nearest protected exit | **45 m** |
| **Other** | **36 m** to nearest protected exit | **45 m** |

#### 7.3 Stair separation & inter-stair access

| Rule | Number |
|---|---|
| Max separation between staircases / discharge points | **≤ 48 m** along exit-route centreline |
| Min separation | **≥ 6 m** (B10.2) |
| Inter-stair access (≥2 stairs) | At least every **5 storeys** + roof/top accessible floor; **or** at refuge floors + roof |
| Security locking stairs | Must auto-release on fire alarm **or** power failure (B8.2) |

#### 7.4 Inner rooms & multi-level flats (B11.4–B11.5)

| Topic | Rule |
|---|---|
| Flat ≥3 levels | ≥**2** levels each need access to a protected exit |
| Inner room (exit only through another room) | Occupant capacity ≤**30**; pass through ≤1 other room; travel ≤ deadend limits; outer room not special hazard & same occupant; vision panel (or audible alarm) — **except** toilet; does **not** apply to flat rooms escaping through living/dining/kitchen |

### 8. Staircase discharge value (B12 / Tables B3–B4)

Sum of discharge values of all required stairs serving floors **above** G/F ≥ total occupant capacity of those floors. Same logic for basement stairs vs basement capacity.

- **Table B3** = non-sprinkler-protected building (or shared stair serving mixed sprinklered/unsprinklered parts).
- **Table B4** = **whole** building sprinklered to FSD CoP.
- Upward escape: × **0.8**. Scissor stair without intermediate landing between floors: × **0.7**.
- Width >1800 mm: linear projection from table.

#### Table B3 — non-sprinkler (persons)

| Storeys served | 1050–<1200 | 1200–<1350 | 1350–<1500 | 1500–<1600 | 1600–<1700 | 1700–1800 |
|---|---|---|---|---|---|---|
| 1 | 210 | 240 | 270 | 300 | 320 | 340 |
| 5 | 338 | 392 | 450 | 504 | 548 | 588 |
| 10 | 498 | 582 | 675 | 759 | 833 | 898 |
| Each extra | +32 | +38 | +45 | +51 | +57 | +62 |

#### Table B4 — sprinklered (persons)

| Storeys served | 1050–<1200 | 1200–<1350 | 1350–<1500 | 1500–<1600 | 1600–<1700 | 1700–1800 |
|---|---|---|---|---|---|---|
| 1 | 420 | 480 | 540 | 600 | 640 | 680 |
| 5 | 548 | 632 | 720 | 804 | 868 | 928 |
| 10 | 708 | 822 | 945 | 1059 | 1153 | 1238 |
| Each extra | +32 | +38 | +45 | +51 | +57 | +62 |

**SD takeaway:** Without whole-building sprinklers, tall residential cores need more / wider stairs much sooner. Partial sprinklers on a shared stair → use B3.

### 9. Exit-route construction, doors, stairs, ramps, lifts, basements (B5, B13–B17)

#### 9.1 General exit route (B5)

| Item | Rule |
|---|---|
| Lead to UPS | Direct; doors/gates readily openable from inside in exit direction without key |
| Discharge to upper open area (podium) | Defined by permanent features; continue as protected exit to UPS; floor FRR per C12.4 |
| Special hazard opening onto exit | Protected lobby; normal-hazard route must **not** pass through special hazard |
| Lighting | ≥ **30 lux** at floor + FSD emergency lighting |
| Private lane | Only if paved, permanently unobstructed, BA accepts integrity as UPS |
| Discharge return | ≥ **450 mm** return/projection where discharge adjoins other exits/accommodation |
| Drop at discharge | Door set back ≥ exit width; single step clearly marked |
| Headroom | ≥ **2000 mm** (sprinkler heads: ≤90 mm from side wall; ≤105 mm headroom reduction) |
| Signs | Directional / exit signs per FSD CoP |
| No roof access | Notice on every floor: “This staircase has no access to the roof / 此樓梯不能通往天台” (≥50 mm letters) |

#### 9.2 Doors (B13)

| Situation | Rule |
|---|---|
| Across exit / into exit from room/storey >30 persons | Open in exit direction, or bi-swing with transparent upper panel |
| Security lock | Openable from inside without key; electrical lock auto-releases on FS alarm / power failure + local override |
| Door onto stair landing | Swing must not reduce landing effective radius below stair width |
| Min door (room/storey >3 persons) | ≥**750 mm** clear; double leaf each ≥**600 mm** + rebated-stile closer sequence |
| Door to protected lobby / stair | Transparent upper view panel of required FRR |
| Notifiable workplace (Cap. 59V) | Exit doors open outward; workplace exits self-closing |
| Stair / lobby doors | Self-closing without hold-open check; “keep closed” notices both sides |
| Other self-closing doors | May use hold-open if auto-releases on alarm / power failure + smoke detectors both sides (FSD) |

#### 9.3 Required staircase geometry (B14)

| Item | Rule |
|---|---|
| Plan | Straight flights; **no winders** |
| Risers per flight | **2–16** |
| Going / rise (general) | Going ≥**225 mm**; rise ≤**175 mm** |
| Class **5a** | Going ≥**280 mm**; rise ≤**150 mm** |
| Class **5b** | Going ≥**250 mm**; rise **75–150 mm** |
| Landings | Top & bottom of each flight; width & length ≥ stair width |
| Clear width / headroom | ≥ B8.1 width; headroom ≥**2000 mm** |
| Max single-flight width | **1800 mm** — wider → central handrail split into 1050–1800 mm sections |
| Handrails | Both sides; height **850–1100 mm**; projection ≤**90 mm** each; continuous per flight (5a: carry round landings) |

#### 9.4 Ramps, lift lobbies, basements (B15–B17)

| Topic | Rule |
|---|---|
| Ramp in exit | Max gradient **1:12** |
| Lift lobby | Unobstructed access to exit at all times, **or** intercom + ≥**2 h** emergency power to management office |
| Basement exits | ≥**2** unless: floor ≤**3 m** below discharge ground **and** ≤**150 m²** **and** lavatory/plant only |
| Upper stairs into basement | Must **not** continue directly |
| Independent basement exit | ≥1 exit per basement floor serves that floor only & discharges independently upward (except lavatory-only) |
| Basement stair lobby | Protected lobby unless pressurized or independent B17.4 stair |

### 10. Refuge floors (B18 + C17) — tall-building massing

| Trigger | Provision |
|---|---|
| Building **>25 storeys** above lowest ground storey | Refuge required (plant-only storeys may be excluded from count) |
| Spacing — Use **6** | ≤ **20 storeys** from another refuge / street / UPS |
| Spacing — other Uses | ≤ **25 storeys** |
| Domestic / composite **26–40 storeys** | Main roof as refuge (B18.3–B18.4) may replace intermediate refuge |

#### Intermediate refuge must include:

| Item | Rule |
|---|---|
| Occupancy at refuge level | No occupied accommodation / accessible plant (except FS tanks & FS plant) |
| Net refuge area | ≥ **50%** of refuge-floor GFA; headroom ≥ **2300 mm** |
| Min dimension | ≥ **1.5×** widest passing stair width |
| Open sides | Open above parapet on ≥ **2 opposite sides**; C17.2 → open side ≥**6 m** from opposing street/boundary/low-FRR wall/other building (unless same plane) |
| Stair continuity | Stairs **discontinued** — divert across refuge before continuing |
| Lifts | Fireman’s lift only; doors locked until fireman’s switch (ordinary lifts: sky-garden package under C17 commentary only) |
| Lighting | ≥30 lux + emergency lighting |
| FRR (C17.1) | Walls **-/120/120**; floors **120/120/120**; shafts/ducts must not open directly onto refuge |
| Signs | Stair no., REFUGÉ FLOOR / 庇護層 notices, directional exits — bilingual, illuminated, ≥50 mm letters |

#### Main roof as refuge (B18.3):

- Flat surface; FRR ≥ storey below; net refuge ≥ **50%** of typical floor GFA below; stairs continue to roof unobstructed; min dimension ≥1.5× widest stair; lighting as above.
- Private roof remainder: solid fence ≥**1.5 m**; stairhood within 1.8 m of refuge → FRR **-/60/60**; plant adjoining refuge → **-/120/120** with unprotected openings ≥1.8 m away.

### 11. Disability temporary refuge (B29–B30)

**Applies** unless: DMBA-exempt building/part; floor on site ≤500 m² with floor ≤200 m² or ≤2 units; ventilated balcony-approach without fireman’s lift; refuge floor; G/F direct to UPS.

| Item | Rule |
|---|---|
| Size | ≥1 space **1.5 × 1.5 m** in protected exit or fireman’s-lift lobby per fire compartment / floor (or 2 × **0.75 × 1.5 m** mutually visible) |
| Coverage | No part of floor > **60 m** from a temporary refuge |
| Must not reduce | Exit width, stair landing effective width/radius, or fireman’s-lift lobby min area |
| Door to space | Clear width ≥ **850 mm** (or Table B2 if greater); handle 950–1050 mm AFFL |
| CCTV + intercom | Emergency power ≥ **1 hour** to management office / FS control panel; intercom 900–1200 mm AFFL |
| Demarcation | Contrasting floor colour / strips + bilingual notices with accessibility symbol |

Every element of construction and every fire barrier of each compartment ≥ Table C1 FRR. Oversized → case-by-case enhanced MOE/MOA/FSI/FRC.

| Use | Max compartment area / volume | Min FRR (min) |
|---|---|---|
| **1** Residential | Not limited (**each flat** still separated) | **60** |
| **2** Hotel | Not limited (**each guestroom** separated) | **60** |
| **3** Institutional | ≤ **2,500 m²** | **60** |
| **4a** Business | ≤ **10,500 m²** | **60** |
| **4b** Mercantile | ≤2,500 m² | **60** |
| **4b** Mercantile | >2,500 to ≤10,500 m² | **120** |
| **5a / 5d** Assembly | ≤2,500 / >2,500–10,500 | **60 / 120** |
| **5b** Educational | same banding | **60 / 120** |
| **5c** Transport | ≤ **10,500 m²** | **120** |
| **6a** Industrial workplaces | ≤ **10,500 m²** | **120** |
| **6b** Warehouse | ≤ **28,000 m³** and ≤10,500 m² | **120** |
| **6c** Hazardous DG | ≤ **7,000 m³** | **120** |
| **7** Carpark | ≤ **10,500 m²** | **60** |

### 13. Separation between buildings, uses, flats (C5–C7)

#### 13.1 Buildings on same site (<1.8 m apart = adjoining) (C5.2)

| Façade angle | Rule |
|---|---|
| **>135°** | Within **900 mm**: imperforate external walls at storey FRR; unprotected openings OK if ≥900 mm from other building’s unprotected opening |
| **≤135°** | Within **1.8 m**: imperforate FRR walls; openings ≥900 mm from other building with fixed-light FRR; unprotected openings only if ≥**1.8 m** apart |
| Roof within 1.8 m | Imperforate at storey FRR; openings ≥900 mm with fixed-light FRR |

#### 13.2 Buildings on adjoining sites (C5.3)

Within **900 mm** of common boundary: external walls/roofs at storey FRR, imperforate; openings ≥**450 mm** from boundary with fixed-light FRR.

#### 13.3 Flats / guestrooms on common corridor (C6)

Corridor fire barriers per Table C1; flat/guestroom doors FRR ≥ corridor; **smoke seals** on top & sides of every such door.

#### 13.4 Different Use Classifications / occupancies (C7)

Separate by **longer** of the two Table C1 FRRs. Exceptions: small ancillary offices/caretaker/storage in industrial; pantries in offices; snack bars in 5a; Class 4a offices and 4b retail shops among themselves (if Table C1 areas met). Shopping arcade shops need **not** be separated from mall if common management + common alarm. Deadend balcony approach → barriers/doors/fixed lights at occupancy FRR (Class 1 windows on deadend balcony: fixed **-/30/30**).

### 14. Openings, shafts, stairs, lifts, atriums, façades, roofs (C8–C12)

| Topic | Rule |
|---|---|
| Openings in compartment walls | Fire doors/shutters at wall FRR; if openings >**25%** of wall length → insulation criterion applies (may reduce insulation to **30 min** if sprinklers both sides) |
| Services through barriers | Fire dampers / fire stops maintaining barrier FRR |
| Vertical shafts | Fire-sealed penetrations; doors/hatches self-closing at shaft FRR |
| **Liftwells (C9.1)** | Barriers **120/120/120**; landing doors integrity ≥ wall or **-/120/-**; basement lift doors → smoke-seal lobby |
| Protected exits / lobbies (C9.3) | FRR ≥ Use Classification of connecting compartment; doors integrity+insulation ≥ barriers; lobby doors per C16.5; **no non-emergency services** unless enclosed at wall FRR |
| Stair elements inside FRR enclosure | Need not have FRR but must be **non-combustible** |
| Stair external wall within **6 m** of street opposite / boundary / other low-FRR wall / other building | That wall → full stair FRR, imperforate (fixed light ≤25% of storey wall area OK; G/F/roof discharge door OK) |
| Coplanar low-FRR external wall | Return fire-resisting walls ≥**450 mm** at junction (not beyond façade at G/F discharge) |
| Floor voids in sprinklered buildings | **450 mm** non-combustible downstand **-/30/-** (below false ceiling if present), or approved smoke curtain (BS EN 12101-1 @ 600°C / 30 min + BS EN 1634-3) |
| Curtain-wall perimeter void | Seal FRR ≥ floors + D-stability ≥ FRR + leakage ≤25 m³/h/m² @ 25 Pa |
| **Atrium** (sprinklered) | Separated by FRR barriers/glazing/shutters/curtains; volume ≤**28,000 m³**; height base→lowest ceiling structure ≤**15 m**; ≤**3** interconnected floors; sprinklers at atrium base |
| **Spandrel (C11)** | Unsprinklered (not single-family): **900 mm** vertical **or** **500 mm** horizontal projection at intervening-storey FRR — **waived if sprinklered** |
| Kitchen/bath openings in Class 1 spandrels | ≤**110 mm** dia for plumbing/drainage |
| Roofs | Non-combustible; single-stair roof (highest floor >13 m) & refuge roofs → FRR ≥ storey below; podium temporary UPS floor → FRR ≥ storey below |

|---|---|
| Special hazard enclosure | **-/120/120**; if opens to protected exit → **-/240/240** |

| Class 1 kitchen beside single exit | Walls/door **-/30/30** |

| Basement vs G/F | Compartment **-/240/240** (incl. protected exits); basement elements **240/240/240**; internal basement walls **-/120/120**; uniting basement with upper storeys forces upper storeys to basement FRR |

| Refuge intermediate floor (C17) | See §10; sky garden extras: non-combustible fixed furniture; no naked flame; artificial plants non-combustible; optional ordinary lifts via **-/120/120** lobby with auto-lock on alarm |

### 19. Firefighting & rescue stairway (D14–D21)

Required when Table D1 says so — typical industrial / deep basement cases.

| Item | Rule |
|---|---|
| G/F access | Same street-access / ≤18 m setback / 1500×2000 passage as fireman’s lift; lift **and** stair from **same** street |

| Ancillary uses | Carpark / loading areas also served by FRS |

### 20. Emergency Vehicular Access — EVA (D22–D26)

**Design vehicle:** 30,000 kg gross; **26 m** turning circle; **12 m** length. Dead-end EVA needs turning space. Gradient ≤**1:10** (≤**1:6** only where **not** serving major façade). Change-of-gradient acute angle ≤**6°** or insert ≥10 m intermediate plane (≥12 m on bend). Clear headroom under structures ≥**4.5 m**.

| Site / use | EVA geometry |
|---|---|
| **Virgin site** (and later redevelopments of that virgin site) | Carriageway ≥**7.3 m** (or hard-paved non-carriageway ≥**6 m**); serve ≥**¼** of total perimeter as “major façade”; façade “served” only if EVA ≤**10 m** from façade (distance waiver above **60 m** façade height); EVA portion serving façade uncovered |
| Virgin + **Use 6** | **Two opposite** façades each ≥¼ perimeter; preferably from different streets; 2-way ≥**13.5 m** undivided (or 7.3 m each side if divided); if EVA outside site → reserve ≥**6 m** strip along boundary as additional EVA |
| Virgin + **Use 5a** | Thoroughfare widths per **B20** (see §21) |
| Mixed Uses | Most stringent Use applies |
| **Redevelopment** | Use 6: major façade ≥**7.3 m**; other Uses: major façade ≥**6 m**; 5a still follows B20; still meet gradient / headroom / vehicle / crash-gate rules |
| Crash gate | Required when EVA is not day-to-day vehicular access (Diagrams D7–D8) |
| Signs | Layout sign at entrance; indicators ≤100 m; No Parking ≤50 m on carriageway EVAs |

#### Exemption / enhancement (D25–D26)

| Path | Detail |
|---|---|
| Exemption grounds | Low fire risk use **or** topographical impracticability |
| Process | Apply under B(P)R 41D(3) with **FSAR** (Part G); BA + FSD on merits |
| Typical low-risk examples | Microwave station, unmanned transposer, beach house, cable pylon |
| Typical topography | Stepped street; road not to EVA standard outside owner control |
| Compensating enhancements (examples) | Fast-response sprinklers to domestic common areas (+ whole commercial in composites); stair pressurization/venting; direct line to FSCC; enlarged tanks/inflow — **not** a free pass to omit EVA |

#### 21.4 Class 5a stair / barriers / lighting

- Flights 3–16 steps; ≤2 successive flights without turn; turn required if any flight >12 steps; going ≥280 / rise ≤150.
- No chains/padlocks on exits; slip-resistant surfaces; guard rail ≥1100 mm where fall >1000 mm; public areas ≥30 lux + emergency lighting.
- Permanent stage: exit each side (one direct to thoroughfare); stage basement exit to thoroughfare.

### 24. Schematic design checklist (day-one)

1. Assign **Use Classification(s)** per floor / fire compartment; flag composite and licensing uses (5a, hotel, RCHE, karaoke).
2. Run **Table B1** on worst floors → **Table B2** (or **B5**) exit widths → place **≥2** stairs (**6–48 m** apart).
3. Check **dead-end / travel** envelopes and **30°** exit angle before locking corridors.
4. Decide **sprinkler / pressurization** early (lobby omission ≤20 m / ventilated stair; C11 spandrel waiver; Table B4 discharge).
5. If >**25 storeys**, reserve **refuge** (or roof-refuge for 26–40 storey domestic/composite); FRR **120** package.
6. Apply **Table C1** FRR / compartment caps; separate uses, special hazards, open kitchens.
7. Size **fireman’s lift / FRS** from Table D1; ≤**60 m** coverage; protect G/F FS access (≤18 m setback).
8. Trace **EVA** (virgin vs redevelopment; Use 6 / 5a extras); budget crash gates / signs.
9. Provide **temporary refuge spaces** in cores for DMBA-applicable floors.
10. For cinemas/PPE: seatways, foyer area, smoke extract, street widths, basement ban.
11. Cross-check FSD FSI CoP and Annex A licensing guides for intended tenancies.
12. If DTC fails → Part G pre-enquiry + FSAR + Bounding Conditions on GBP.
