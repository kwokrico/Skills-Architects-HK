# Design Manual: Barrier Free Access 2008
**Architect critical summary for schematic design**  
2025 Edition (June 2025 revision; amendments via PNAP APP-41 through May 2024 / June 2025) | Buildings Department

> Scope note: Chapters 3–4 Obligatory Design Requirements align with the **Third Schedule of the Building (Planning) Regulations**. Alternative designs are acceptable only if they meet the stated Performance Objectives. Best Practice / Recommended items are **not** statutory minima — use them for tender quality and elderly-friendly schemes. Read with PNAP APP-41 / APP-152 for BD plan-marking practice.

---

## Regulatory Overview

This Manual applies to **new buildings and A&A works**. Extent follows **Table 1**: all parts of non-domestic buildings; common areas of domestic buildings over 4 storeys (or G/F entrance + access only if ≤4 storeys); composite = non-domestic parts + domestic common areas as above. **Table 2** then adds assistive provisions (tactile guide path, Braille floor plan, accessible counter, visual fire alarm, assistive listening, etc.) by use type.

**Exempt from Obligatory Requirements:** single-family buildings ≤**13 m** above ground; temporary buildings / contractor’s sheds; and high-risk / non-public areas listed in §2.2.2 (commercial kitchens, plant/switch rooms, hazardous stores, water body of swimming pools, security platforms, mezzanine storage, paths serving only exempted areas).

## Critical main topics and subtopics

### 1. Lock the BFA extent first (Tables 1 & 2) — SD killer checklist

| Building / use | What you must provide (SD takeaway) |
|---|---|
| **Domestic >4 storeys** | Full BFA to **all common areas** (lobby, lift, corridor, stair, common toilet if any) |
| **Domestic ≤4 storeys** | Accessible **main entrance + G/F common area + access route** only |
| **Non-domestic / composite non-domestic** | Entire building (minus §2.2.2 exemptions) |
| **Shopping / department store** | + Braille floor plan, tactile guide path, accessible counter, visual fire alarm |
| **Hotel / bank / worship / restaurant / market / club / carpark** | + Braille floor plan (varies), counter and/or visual fire alarm per Table 2 |
| **Cinema / theatre / stadium / museum / theme park** | Full Table 2 package incl. **visual display board + assistive listening** |
| **School / sports / swimming complex / elderly home** | Braille plan + tactile path + counter + visual alarm |
| **Hospital / clinic / transport terminal** | Full Table 2 including assistive listening |
| **Office common areas / factory** | Visual fire alarm only (among Table 2 extras) |
| **Pure domestic use (inside flats)** | Table 2 assistive extras **not** applicable |

**SD takeaway:** Assign use against Table 2 before fixing lobby sequencing, counter locations, and whether a continuous tactile guide path is required from lot boundary.

### 2. Continuous accessible route (Division 4) — the primary spine

| Item | Obligatory rule |
|---|---|
| Start / end | From a **prominent lot-boundary point** connected to a public street/pedestrian way → **commonly used public entrance** → **accessible lift** |
| Min clear width | **≥ 1050 mm** |
| Barriers | No steps, projecting kerbs, steep ramps, or wheelchair-blocking doors |
| Surface | Firm |
| Main entrance not accessible | Allowed only for difficult terrain / steep driveway **if** PWD can arrive by vehicle **and** directional signs to the accessible entrance are conspicuously posted |
| Tactile guide path (Table 2 uses) | Boundary access → main entrance → lift zone, nearest accessible toilet, info/service counter, Braille floor plan, and staircase (Div. 14) |

**Best practice (not obligatory):** prefer **≥ 1500 mm** clear; on sites ≥1000 m² (hospital, school, shopping, transport, etc.) recommend corridors **≥ 1500 mm**.

**SD takeaway:** Principal entrance should be the accessible entrance. Do not invent a remote “disabled entrance” as the SD default.

### 3. Level changes — ramps, dropped kerbs, stairs, handrails

#### 3.1 Ramps (Division 5)

| Item | Obligatory |
|---|---|
| Clear width | **≥ 1050 mm** |
| Top/bottom landing | **≥ 1500 × 1500 mm** clear (door swings must not eat the landing) |
| Max gradient | **1:12** generally |
| Minor-rise exceptions (single use only — **not combinable**) | 1:10 → max length 1500 / rise 150 mm; **1:8** → max length 600 / rise 75 mm |
| If ≥ **1:20** | Intermediate landing ≥**1200 mm** long every **10 m** horizontal run; handrails both sides; tactile warning at head, foot, landings |
| Edge protection | Kerb ≥**100 mm** **or** rail **200 mm** above ramp on both sides |
| Projection hazard | Fixtures ≤**90 mm** from wall below **2000 mm**, else extend to floor / tactile guide |
| Traffic end | Rise >200 mm toward vehicles → barrier across full width ≥**1500 mm** from foot of ramp |
| Short/lift ramps | Landing/handrail/tactile rules do not apply where length **< 300 mm** or ramp access to lift |

**Preferred (recommended):** 1:20–1:15; width ≥1200 (pass) / ≥1500 (two chairs); avoid curved ramps and crossfalls.

#### 3.2 Dropped kerbs (Division 6)

| Item | Rule |
|---|---|
| Where | Pedestrian crossings; each end of footpath of private street / access road; kerb separating footpath/ramp from vehicles |
| Size | ≥ **1200 × 1200 mm**; ≥ **800 mm** clear behind on footway |
| Gradient | ≤ **1:10** |
| Level to road | ≤ **15 mm** |
| Tactile | Warning strip **300 mm** from vehicles + **600 mm** nominal at the ramp |

#### 3.3 Steps & staircases (Division 7) — required stairs + main common circulation stairs only

| Item | Rule |
|---|---|
| Tread / riser (internal) | Tread ≥**225 mm**; riser ≤**175 mm**; nosing projection ≤**15 mm**; **no projecting nosing** |
| Flight | ≤ **16** risers without landing |
| Handrails | Both sides (Div. 8) |
| Nosing | Non-slip, contrasting colour; tread/wall colour contrast |
| External stairs | Riser ≤**160 mm**; tread ≥**280 mm** |
| Tactile warning | Top, bottom, and landings — **300 mm** where landing enclosed; **600 mm** where leading to open space / building exit (+ Braille/tactile “opening” sign on adjacent wall) |
| Projection | Same 90 mm / 2000 mm rule as corridors |

#### 3.4 Handrails (Division 8)

| Item | Rule |
|---|---|
| Height | Top of rail **850–950 mm** above nosing/floor/landing |
| Clearances | **30–50 mm** clear of wall; **70 mm** clear above brackets |
| Profile | Tubular **Ø 32–50 mm**, or equivalent grip |
| Extensions | ≥ **300 mm** horizontal beyond first/last nosing (or ramp ends); closed return; ≥**100 mm** allowed if door opening conflicts |
| Loading | ≥ **1.3 kN** vertical or horizontal; no rotation in fittings |
| Braille on stair handrails | Directional arrow + floor number at designated location on every floor |

### 4. Corridors, lobbies, paths & doors (Divisions 9–10)

#### 4.1 Circulation (Division 9)

| Item | Obligatory |
|---|---|
| Clear width | ≥ **1050 mm** |
| Dead-end turning | ≥ **1500 × 1500 mm** within **3500 mm** of every dead end |
| Corridor lobby length | ≥ **1200 mm** excl. door swings |
| Building entrance | Level area ≥ **1200 mm** beyond door swings **and** ≥ **1500 mm** wide on **both** sides (not for stair-only lobbies) |
| Channel covers | Flush; holes ≤ **20 mm** |
| Gratings | Slot ≤ **13 mm**; slots **not parallel** to travel |
| Controlled passages / turnstiles | At least one path ≥ **800 mm** with accessibility symbol (or adjacent alternative) |
| Low headroom ≤2000 mm | Warning guardrail / barrier with leading edge ≤ **680 mm** AFL |

#### 4.2 Doors on accessible routes (Division 10)

| Item | Obligatory |
|---|---|
| Clear opening | ≥ **800 mm** (incl. one leaf of a pair) |
| Latch-side clear (leading face) | ≥ **330 mm** unobstructed beside handle |
| Corner rule | If door <330 mm from room corner → hinge on the corner side |
| Handle height | Top of grip **950–1050 mm** AFL |
| Threshold | ≤ **20 mm**, bevelled |
| Opening force | Exterior / FRR doors ≤ **30 N**; interior ≤ **22 N**; interior closer ≥ **3 s** from 70° to 75 mm from closed |
| Frameless glass | Prominent marking with portion between **900–1500 mm** AFL (also on adjacent glass panels) |
| Auto main entrance door — mandatory for | Sports stadium, town hall, civic centre, theatre, museum, public library, shopping complex, sports complex, public swimming pool complex, **office**, **hotel**, **hospital** |

**Recommended:** clear width ≥850 mm; latch approach ≥600 mm; fire doors to accessible areas — hold-open (except required stair/lobby FR doors).

### 5. Sanitary & bathing (Divisions 11–12, 17) — early floor-plate locks

#### 5.1 Count & type (Division 11)

| Trigger | Provision |
|---|---|
| Toilet on a floor | ≥ **1 accessible unisex** toilet (entered without crossing opposite-sex area) |
| Accessible W.C. cubicles on that floor/part | ≤20 cubicles → **1**; >20 → **2** (by sex count if inside sex toilets) |
| Not applicable | Domestic buildings / domestic parts of composite; floors with **no** toilet |
| Location | Direct from public corridor; if inside multi-cubicle toilet → **1500 × 1500 mm** clear immediately in front (or straight-on approach) |

#### 5.2 Accessible W.C. geometry (hard numbers)

| Element | Dimension |
|---|---|
| Cubicle / unisex room | ≥ **1500 × 1750 mm** |
| Manoeuvring space | ≥ **1500 × 1500 mm** @ **350 mm** AFL |
| W.C. seat height | **380–450 mm** to top of seat; back support; **not** spring-actuated |
| Door clear (Fig. 24 / guest rooms) | Typically **≥ 800 mm** clear (recommended ≥850) |
| Flush control | Wide side of cubicle, **600–1050 mm** AFL; ≤ **22 N** |
| Basin | Rim ≤ **750 mm**; knee clearance to apron bottom ≥ **550 mm**; lever/auto tap ≤ **22 N** |
| Wall grab rails | ≥2; Ø **32–40 mm**; ≥ **30 mm** clear of wall; length ≥ **600 mm**; 150 kg |
| Door grab rails | Both faces of door |
| Folding grab rail | Wide side beside W.C., **725–750 mm** AFL when down |
| Emergency call (Div. 17) | Weatherproof push **600–650 mm** AFL below vertical grab rail beside W.C.; “Emergency Call” EN/中/Braille; alarm outside or at caretaker |
| Urinal (if >1 provided) | ≥1 ambulant: clear **800 × 1500 mm** in front; rim ≤ **400 mm**; vertical grabs Ø32–40 × 600 mm at **1200 mm** AFL both sides |

#### 5.3 Accessible guest bathrooms / showers (Division 12 — hotels etc.)

| Element | Obligatory |
|---|---|
| Clear in front of bathtub | ≥ **1500 × 800 mm** |
| Bath seat | ≥ **250 mm** wide; tub height ≤ **380 mm** |
| Bath grab rails | Horiz./≤20° rail ≥900 mm along tub at **150–300 mm** above rim; vertical ≥600 mm at plug end |
| Taps | Lever ≥75 mm; at plug end; ≤ **450 mm** above rim |
| Hand shower | Hose ≥ **1500 mm** + wall bracket; vertical bar if used: ≥500 mm, lower end ≥450 AFL, 150 kg |
| Shower compartment | Internal ≥ **1500 × 900 mm**; clear in front ≥ **1500 × 800 mm** |
| Shower L-grabs | ≥ **750 × 900 mm**; **700–800 mm** AFL; 150 kg |
| Roll-in threshold | ≤ **13 mm**, bevelled |
| Shower seat | ≥ **550 × 400 mm**; top **430–480 mm** AFL |

### 6. Occupancy-specific counts that change massing

#### 6.1 Auditoria (Division 1) — theatre, cinema, concert / sports hall, lecture / conference hall

| Seats at spectator level | Min wheelchair spaces |
|---|---|
| ≤ **800** | **4** (in pairs; not separated from other seats) |
| >800 | +**2** per additional **400** seats (or part) |
| Each space | ≥ **800 × 1300 mm** (800 mm faces stage); aisle from accessible entry ≥ **1500 mm** |
| Also | Stage, backstage, changing, toilets, showers accessible; if levels connect, provide ramp/lift equivalent; Braille fire-exit maps at all auditorium entrances; tactile at top/bottom of stage stairs |

#### 6.2 Hotels / hostels / guesthouses (Division 2)

| Rooms | Accessible guest rooms (full Div. 11/12 facilities) |
|---|---|
| Baseline | **Min 2** |
| Scale | **+2 per 100 guest rooms** (or part) → e.g. 150 rooms = **4** |

#### 6.3 Carparks (Division 3)

| Total car spaces in lot | Accessible spaces |
|---|---|
| 1–50 | **1** |
| 51–150 | **2** |
| 151–250 | **3** |
| 251–350 | **4** |
| 351–450 | **5** |
| >450 | **6** |

| Geometry | Rule |
|---|---|
| Single accessible bay | ≥ **3500 mm** wide |
| Paired bays + shared loading strip | Each bay ≥ **2500 mm** + shared yellow-hatched strip ≥ **1200 mm** |
| Location | Near lobby with accessible lift / entrance |
| Marking | International accessibility symbol + bay number on floor |

### 7. Vertical circulation (Divisions 19–20)

#### 7.1 Lifts (Division 19) — obligatory if any passenger lift exists

| Rule | Dimension / requirement |
|---|---|
| Coverage | **Every floor** reachable by ≥1 lift fully complying with Div. 19, with **direct access to main lift lobby** |
| Other passenger lifts | Must still meet door + control button rules (¶79–80) |
| Min car (any accessible lift) | **1200 × 1100 mm**; door clear ≥ **850 mm**; handrails 850–950 mm AFL, within 150 mm of rear/side corners, 30–50 mm wall gap |
| If **>3 lifts** in building | ≥1 car **1500 × 1400 mm** (either orientation), door ≥850 mm |
| Doors | Horizontal sliding, power, auto; audible close signal; obstacle re-open detector at **500–600 mm** AFL |
| Buttons (car & hall) | **900–1200 mm** AFL; ≥ **20 mm** size; Braille/tactile (numerals ≥15 mm, raised ≥1 mm); main entrance floor = **star**; alarm = **bell** shape |
| Lobby call indication | Illuminated + audible (1× UP / 2× DOWN) before arrival |
| Jamb designations | Arabic numerals ≥**60 mm**, raised 1 mm, both jambs @ **1200 mm** AFL |
| In-car | Visual position/direction indicators (chars ≥50 mm); audio floor announce in **English, Cantonese, Putonghua** |
| Lobby symbol | Identify compliant lifts with accessibility symbol (or at least one lobby symbol if all lifts comply) |

**SD takeaway:** Escalators / conveyors are **not** barrier-free routes (Div. 20). Always keep an accessible lift (or ramp) as the independent vertical path; provide tactile warning strips at escalator/conveyor ends.

### 8. Wayfinding, counters, light, hearing & fire alarms

#### 8.1 Signs (Division 13)

- Accessibility symbol (white wheelchair on blue) at accessible entrances/exits, PWD parking, accessible toilets, usable vertical circulation, cloakrooms, counters/telephones.
- Directional arrows with the symbol to those facilities.
- Min sign height: doors **60 mm**; corridors **110 mm**; external **200 mm**.
- Braille/tactile signs on adjacent wall/door of accessible facilities at **900–1500 mm** AFL.
- Braille/tactile fire-exit map where a public floor plan is provided — map at **800–1200 mm** AFL.
- Hearing-loss symbol where assistive listening is provided.

#### 8.2 Info / service counters (Division 15 — Table 2 uses)

| Item | Rule |
|---|---|
| Height | ≥1 counter portion ≤ **750 mm** AFL and ≥ **750 mm** wide |
| Knee space | Depth **400–600 mm**; height ≥ **680 mm** AFL |
| Noise / screened counters | ≥1 counter with assistive listening |

#### 8.3 Illumination (Division 16)

| Location | Min lux @ floor (passage centreline) |
|---|---|
| G/F entrance lobby & lift | **120** |
| Upper lift lobbies, corridors, accessible paths, stairs | **85** |
| Accessibility signs (Div. 13) | **120** on sign face |

#### 8.4 Assistive listening (Division 18 — Table 2)

Required in **≥1 public counter** (if any) **and** in the **hall / auditorium** for applicable uses (induction loop / infrared / equivalent). Recommended coverage ≥70% of served floor area; receivers ≥1 per 50 persons (min 2).

#### 8.5 Visual fire alarm (Chapter 5.2 — Table 2)

Flashing **red** visual alarm labelled “Fire Alarm / 火警” as part of the fire alarm system at prominent, occupant-accessible locations. **Not required** in exit staircases / adjoining smoke lobbies, or in back-of-house / staff-only / domestic unit interiors listed in §5.2.3(b).

#### 8.6 Switches & outlets (Chapter 5.1 — Best Practice)

Recommended reachable band for wheelchair users: controls **450–1200 mm** AFL; sockets ≥ **450 mm** AFL; ≥30% luminous contrast.

#### 8.7 Vertical lifting platforms (Chapter 5.5 — Recommended alternative)

Only where passenger lift / ramp is impractical: platform ≥ **1100 × 1500 mm**; door clear ≥900 mm; max travel **7000 mm**; load ≤500 kg; barriers ≥1100 mm; auto-homing on power failure. Prefer designing out the need at SD.

### 9. Schematic design hold-points (use before freezing cores)

1. **Table 1 extent + Table 2 assistive package** by use.
2. **One continuous accessible route:** street → entrance → lift → every floor → accessible unisex toilet → principal public destinations.
3. **Level strategy:** prefer level thresholds (≤20 mm doors / ≤13 mm roll-in showers); if ramping, reserve landing cubes **1500 × 1500** and stay ≤ **1:12**.
4. **Lift bank:** min car **1200 × 1100 / 850 door**; if >3 lifts lock one **1500 × 1400**.
5. **Toilet count:** ≥1 unisex accessible per toilet floor; room envelope ≥ **1500 × 1750** with **1500** turn.
6. **Typology counts:** auditorium wheelchair spaces; hotel accessible rooms; carpark accessible bays (**3500 mm** or paired 2500+1200).
7. **Auto door** required for listed public/commercial types at a commonly used main entrance.
8. **Tactile guide path** only where Table 2 says Y — but then it must be continuous from lot boundary, not a lobby token.

### 10. Chapter 6 elderly-friendly guidelines (not Obligatory Schedule)

Chapter 6 is **guidance** for aging-in-place (door handles/faucets, lower eye viewers, adaptive layouts, lower windows, elderly toilets, community-farming knee space, etc.). Adopt when the brief is elderly, residential, or welfare — do **not** treat as B(P)R Third Schedule minima unless client/design team elevates them to project standards.
