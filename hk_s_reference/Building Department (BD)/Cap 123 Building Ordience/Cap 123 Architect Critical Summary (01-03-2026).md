# Cap 123 — Buildings Ordinance
**Architect critical summary for schematic design**  
Consolidated version: 1 March 2026 | Cap. 123

> Scope note: Cap 123 is the **primary statute** — who must be appointed, when BA approval/consent is mandatory, what kills plan approval, OP/TOP gates, change-of-use notice, scheduled-area triggers, street/EVA/projection control, and built-in exemptions. It is **not** the SC/PR/light/air rulebook (Cap **123F**), not the plan-package rulebook (Cap **123A**), and not materials/structure performance (Cap **123Q**). At SD: settle AP/RSE/RGE + contractor class → map Scheduled Areas → freeze use class for FSI certificate → design so s.16 refusal grounds cannot bite → path to OP under s.21.

---

## Regulatory Overview

Cap 123 controls the **planning, design and construction of buildings and associated works** in Hong Kong (and in the New Territories only as applied by Cap **121**). It applies to virtually every structure the Ordinance defines as a *building* (domestic, non-domestic, composite, PPE venues, industrial, underground space, oil storage ≥110 000 L, etc.) and to *building works* / *street works* — except Government / HA / Garrison land and the limited self-contained exemptions in s.41.

It is the **workflow statute**: appoint registered professionals and contractors → submit regulated plans → obtain BA **approval** then **consent** (or use Minor Works / designated exempted works) → supervise to a **supervision plan** → obtain **OP/TOP** before occupation. Detailed dimensional / intensity / construction rules live in the Cap 123 subsidiary legislation (A–Q) made under s.38 and in BA Codes accepted via PNAP.

## Critical main topics and subtopics

### 1. What counts as “building” / “building works” (s.2) — SD scope traps

| Term | Schematics implication |
|---|---|
| **building** | Extremely wide: whole or part of domestic / public / PPE buildings; factories, warehouses, offices, shops; arches, bridges, chimneys, docks, garages, hangars; walls, stairs, towers; sea-walls, jetties; **caverns / underground space** + access tunnels/shafts; oil storage installations; and anything BA Gazettes as a building |
| **building works** | Construction, **site formation**, **GI in scheduled areas**, foundations, repairs, demolition, A&A, **drainage works**, and every building operation |
| **street works** | Construct / form / reconstruct / alter / repair a **private street** or **access road** (surfaced, channelled, drained, lighted) |
| **site formation works** | Excavations on sloping land, filling, landslip preventive / remedial works, **groundwater drainage works** |
| **new building** | Any newly erected building **or** existing building where ≥½ of volume is rebuilt **or** ≥½ of main-wall superficial area must be reconstructed — OP regime (s.21) applies |
| **domestic / non-domestic / composite** | Habitation includes hotel, guest-house, boarding-house, hostel, dormitory. Composite = partly domestic + partly non-domestic — drives Cap 123F PR maths and FSI “purpose” on plans |
| **habitation** | Use as hotel / guest-house / boarding / hostel / dormitory is **habitation** for Ordinance purposes |
| **oil storage installation** | Single tank or group with any tank ≥ **110 000 L**, above ground, for petroleum products → special Cap 123 regime + oil storage regs |
| **signboard** | Structure solely for advertisement / announcement / visual image — street-projection & UBW track under ss.31 / 39C / Sch. 8 |
| **EVA** | Vehicular access for FSD in fire / other emergency — keep clear for life of building (s.29A) |
| **design assumption / performance review** | Assumptions in design calcs; PR = AP report proving works were inspected/monitored and geo assumptions remain valid — can gate OP (s.21(6)(f)) |

**NT filter:** Cap 123 applies to the New Territories **only as provided by Cap 121** — confirm NTEH / Cap 121 exemptions before treating a village house as fully under Cap 123.

### 2. Team you must lock before schematic start (ss.3–4, 9, 2A)

| Role | When required | SD action |
|---|---|---|
| **Authorized Person (AP)** — architect / engineer / surveyor on BA register | Co-ordinator for all building works / street works **except** MW under simplified requirements | Appoint AP as scheme lead; AP owns co-ordination, contravention notices, supervision-plan duties |
| **Registered Structural Engineer (RSE)** | When Ordinance / regs require structural elements | Budget RSE for anything beyond Cap 123A small-works structural exemptions |
| **Registered Geotechnical Engineer (RGE)** | When Ordinance / regs require geotechnical elements | Trigger early if slopes, retention, deep dig, fill, scheduled areas, groundwater drainage |
| **Registered General Building Contractor (RGBC)** | All building / street works that are **not** specialized works and **not** minor works | Programme assumes registered contractor; specialist works need right RSC category |
| **Registered Specialist Contractor (RSC)** | **Specialized works** designated by BA under **s.2A** (Gazette — not sub-leg) | Check current Gazette categories (e.g. demolition, foundation, site formation, ground investigation, ventilation) before choosing constructability |
| **Registered Minor Works Contractor / prescribed professionals** | MW commenced under simplified requirements | Cap 123N / Cap 123A MW track — no s.14 approval/consent |

**Standing AP duties (s.4(3)):** supervise per supervision plan → notify BA of any regulation contravention that would result from approved-plan works → comply with Ordinance generally.  
**Replacement rule:** if AP/RSE/RGE unwilling or unable to act, client must appoint another; temporary illness / absence from HK → nominate a peer for that period only.

### 3. Approval + consent — the irreversible SD gate (ss.14–16, 14A, 14AA)

#### 3.1 Default rule (s.14)
**No person** shall commence or carry out any building works or street works without BA:
1. **written approval** of documents submitted per regulations; **and**
2. **written consent** to commence the works shown on the approved plan.

Approval / consent does **not** confer land title, waive lease terms, or legalise any Ordinance / other-enactment breach (s.14(2)).

| Path | Rule |
|---|---|
| Full BD track | Approval **then** separate consent to commence |
| Minor Works (simplified) | **s.14AA** — s.14(1) does **not** apply |
| Deemed approval / consent (s.15) | If BA does not refuse in writing (with grounds) within the **period prescribed in regs** (see Cap 123A), approval/consent is **deemed given** |
| Supervision plan (ss.14A, 16(3)(bc)) | BA is **not deemed** to consent if AP has not lodged a supervision plan (unless BA does not require one). Prep must follow **current technical memorandum** under s.39A |

#### 3.2 Consent clock / freeze (ss.16(3), 20)
- Consent refused if not all prescribed plans approved, fees unpaid, s.17 conditions outstanding, demo protection inadequate, **or no supervision plan lodged**
- Consent refused if **> 2 years** since approval of any prescribed plan
- If works **not commenced within 3 months** of consent, or **suspended for 3 months**, consent is **deemed revoked** — renewal can bring new conditions (s.20)

#### 3.3 Plan-approval kill switches — design so these never appear (s.16(1))

Refuse-to-approve grounds an architect must design against at schematic:

| Ground | Practical SD reading |
|---|---|
| **(b) FSI certificate missing / failed** | DFS must certify either *no FSI needed* for stated purpose, **or** plans show **minimum FSI** per DFS CoP for that purpose. **Freeze intended use early** — purpose is written into the certificate |
| **(d) Ordinance or Cap 131 conflict** | BO regs + **approved / partly approved / draft OZP** under Cap 131 |
| **(da) CDA + MLP conflict** | Inside a Cap 131 **comprehensive development area**, works must not contravene TPB-approved **master layout plan** |
| **(g) Neighbourhood character** | Height, design, type or intended use **differing** from immediate neighbourhood or previously on same site — BA discretion; use for early advice on odd massing / use insertions |
| **(h) Street opening / access danger** | Drop-offs, ramps, run-ins dangerous or prejudicial to street traffic safety / convenience |
| **(l)–(m) Collapse risk to neighbours / street / land** | Demo, site formation, piling, excavation, foundations that may collapse adjoining building, street or land — must prove avoidable |
| **(n) Cap 123F reg. 49 use** | Proposed use would breach Cap 123F reg. 49 (inappropriate use of domestic / non-domestic space) |
| **(o) Lands resumption notice** | Site under Cap 124 resumption notice or lease resumption proviso |
| **(p) Inadequate street connexion** | Site that ought to have streets with adequate connexion to a public street — and does not |
| **(q) Scheduled Area No. 1 landslip debris** | In Mid-levels (Area 1), if BA says building must resist landslip debris and plans are inadequate |
| **(1A) Hand-dug caisson ban** | **Must refuse** unless: depth ≤ **3 m** and inscribed circle diameter ≥ **1.5 m**; **or** only practical method; **or** no other safe engineering alternative |

#### 3.4 Consent-stage geotech / neighbour protection (s.16(4)–(5))
BA may refuse consent if works would endanger adjoining buildings / streets / land, or (for demo) until adequate precautions are proven.

### 4. Conditions BA can bolt onto your scheme (ss.17, 17A) — bake into programme

When the application matches Column A triggers, BA may impose Column B conditions / refuse until complied with. SD must anticipate:

| Trigger (Col. A) | What BA can demand (Col. B) — SD impact |
|---|---|
| A&A that creates a **new building** | Extra building works so the whole complies with current structural stability + public health standards |
| New building served by **non-compliant existing access road** | Street works plans + construction to bring access road to regs |
| Street works on weak ground / heavy traffic | Carriageway built to BA-prescribed (not just default reg) standard |
| Any **access / opening to/from street** | Siting & construction to BA’s traffic/pedestrian safety standards |
| Building on **new private street** or **new / altered access road** | Street works approved **before** building consent |
| Structural materials; SF / excavation / piling / foundations; **GI in scheduled areas** | Max loads/stresses; material tests; instrumentation; workmanship; qualified supervision; **Area 1 sequence of works**; **performance review** (scheduled areas **or** sites needing geo verification / groundwater regime impact / unconventional design) |
| SF / piling / excavation / foundations | Conditions to prevent collapse of adjoining buildings / streets / land |

**Sewage tunnels (s.17A):** if works are incompatible with Cap 438 sewage tunnel (or Gazetted proposed tunnel) works, BA may refuse / withdraw approval or consent, or condition piling / excavation / foundations.

**Urgent works (s.19):** emergency shore / underpin / demo / make-safe (or street works) may start without consent if specified-form notice is in before authorize **or within 48 hours** of start — BA can order stop once emergency ends. Urgent safety deviations from supervision plan allowed with after-the-fact AP notice + revised plan.

### 5. Scheduled Areas (Sch. 5 + ss.16–17, 28A) — check **before** massing / basement

If any part of the works sits in a Scheduled Area, that part is treated as in the Scheduled Area (s.2).

| Area No. | What it is | SD / programme trigger |
|---|---|---|
| **1** | **Mid-levels** (plan GCB1, Land Registry) | Landslip-debris resistance (s.16(1)(q)); BA may dictate **sequence of works** (s.17 item 6); **groundwater drainage works** can be required **beyond site boundary** (s.28A) + lasting maintenance duty; Cap 123A typically forces SF + foundation together |
| **2** | NW New Territories (plan GS-SP/714-1) | Heightened geo / GI control; performance-review path |
| **3** | **Railway protection** corridors (MTR + former KCR / WR / ERE plan sets — Land Registry; updated through L.N. 221/2021) | Underground drainage exemption carve-outs (s.41(3C)(f)); specialist GI / foundation / monitoring; early MTRCL / BA consultation |
| **4** | **Ma On Shan** (plan GS-SP/717-3) | Same family of geo / SF / foundation conditions |
| **5** | **Sewage tunnel protection** areas (plan set KCE/S/G/… & 90806/STPA/…) | Align with s.17A / Cap 438 — piling & basement envelopes sensitive |

**Performance review (s.17 item 6(g) + s.21(6)(f)):** if required, OP can be refused when BA is not satisfied the PR justifies monitoring and geo design assumptions. OP applications lodged within 60 days of PR receipt are timed from **60 days after PR receipt** for the deemed-grant clock (s.21(8)).

### 6. Occupation before OP = offence territory (s.21)

| Rule | Detail |
|---|---|
| Occupation ban | No **new building** may be occupied except by ≤ **2 caretakers**, unless OP issued, or unexpired / unrevoked **TOP** covers the occupied part |
| TOP | May cover whole or completed part; BA may limit duration and set conditions; revoke on breach; OP of whole building revokes remaining TOPs |
| After TOP expiry / revoke | s.21(1) reapplies after **7 days** if no OP |
| Deemed OP/TOP | If BA has not refused in writing within **14 days** of specified-form application → permit deemed granted (subject to PR timing rule above) |

**OP refuse grounds (s.21(6)) — design completion criteria:**
- Any building works contravened Ordinance
- Required street works for private street / access road incomplete
- Liftway present but lift not installed **and** liftway not protected to BA satisfaction
- Where DFS certified FSI needed under s.16(1)(b)(ii): no DFS certificate that installed FSI matches plans and is in efficient working order
- Required water supply connexions not proven
- Required **performance review** fails BA’s adequacy / geo-assumption test

### 7. Change of use — material change needs 1-month notice (s.25)

| Step | Rule |
|---|---|
| Notice | Person intending a **material change** in use of a building must give BA **1 month’s** notice in specified form |
| BA powers | Within 1 month of notice: prohibit intended use; or order discontinuance of present use within 1 month; may permit building works to make building suitable |
| When is change “material”? | (a) erecting a building for that use would have breached the Ordinance; **or** (b) BA could have refused plans under **s.16(1)(g)** (neighbourhood height/design/type/use) |

**SD implication for A&A / conversion:** hotelisation, domestic↔non-domestic flips, industrial-to-office, etc. are often material — treat as BO + OZP + lease + Cap 123F use-fitness problem from day one, not a fit-out afterthought.

### 8. Streets, openings, EVA, projections — envelope & public-realm hold points

#### 8.1 Private streets / access roads (s.29)
Frontagers must surface, channel, sewer, drain, light and maintain to BA satisfaction / regs. Inadequate roads → BA orders works or carries them out and recovers cost (apportioned by frontage or lot area). Programme new development with street-works path when s.17 items 2/5 apply.

#### 8.2 EVA maintenance (s.29A)
EVA owner must keep EVA in good order. Dilapidation, defect, alteration or addition that defeats fire-emergency purpose → BA order to reinstate. **Do not sacrifice EVA clear width / turning / loading at SD for landscape or parking gain.**

#### 8.3 Openings to / from streets (s.30)
No construction / formation / alteration of any means of access or opening to/from a street without BA consent (standalone or as part of building/street works consent). BA refuses if dangerous or prejudicial to traffic / pedestrian safety or convenience; may condition siting and construction.

#### 8.4 Projections on / over streets (s.31)
No building / structure in, over, under or upon any street (leased or not) unless:
- it complies with Cap **123F** Part II criteria; **or**
- it is a **signboard** whose location/dims do **not** create structure danger, traffic hazard, fire hazard, traffic-signal obstruction, MOE / firefighting-access obstruction, light/air reduction below regs, or public danger; **or**
- BA grants **s.42** exemption

BA can order alteration / removal of projections in the public interest (3 months or shorter). **Canopies, bridges, architectural projections, signboards** are SD space claims — check Cap 123F Part II + PNAP early.

### 9. When you can *skip* s.14 approval / consent (s.41) — use with care

| Exemption | Exempt from | Hard carve-outs (still controlled) |
|---|---|---|
| **Gov buildings**; HA land (subject to Cap 283); Garrison land; certain unleased / vested land; Gov streets | Ordinance generally | Projections over streets + hoardings still apply; s.27C pipe-in-slope duties may still bite private buildings whose pipes lie in exempt land |
| Existing lawfully built buildings / private streets / access roads | No forced upgrade merely by Ordinance existence | New A&A / new building rebuild thresholds still trigger current standards via ss.14 / 17 |
| **Non-structural works in a building** (s.41(3)) | ss.4, 9, 9AA, 14(1), 21 | **Still not exempt:** drainage works; **GI in scheduled areas**; **site formation**; **minor works**; anything involving **structure**. Must still comply with regs (s.41(3D)) |
| **Designated exempted works** prescribed in regs (s.41(3B)) | ss.4, 9, 9AA, 14(1), 21 | Must still comply with regs |
| **Drainage works** non-structural (s.41(3C)) | ss.4, 9, **14(1)** only (not s.21) | Not exempt if involving structure; trade effluent / chemical / petroleum / acid drains; altering public-sewer manhole connexion; septic tank / cesspool work; **underground drainage in Scheduled Area No. 3** |
| Temporary building used as PPE | Ordinance provisions | Controlled by Places of Public Entertainment regime |

**s.42 modifications:** BA may permit Ordinance modifications on application + fee when “special circumstances” so require — **never to prejudice** structural stability or public health standards. **s.42 cannot waive s.14** (approval/consent). Past exemptions create **no precedent**.

### 10. MBIS / MWIS awareness for existing-stock schemes (Part 2A, ss.30B–30C)

Not a new-build massing rule, but critical for A&A / adaptive reuse feasibility:

| Regime | Trigger | Schematics alert |
|---|---|---|
| **MBIS (s.30B)** | Buildings aged **≥ 30 years** (age from OP date, or other BA evidence) | BA may order prescribed inspection / repair of **common parts**, **external walls**, prescribed **projections**, **signboards** |
| **MWIS (s.30C)** | Window inspection / repair obligations (age thresholds in regs) | Façade / window replacement strategies must assume RI / QP path and possible BA notices |

Aged envelopes, illegal signboards, and cantilevered projections are programme and cost risks — survey early.

### 11. Subsidiary legislation map — where Cap 123 sends you next

Cap 123 s.38 enables the design detail. At schematic design, open the matching Cap:

| Cap | What the architect actually designs to |
|---|---|
| **123A** | Plan contents, scales, signing (AP/RSE/RGE), consent/OP forms, supervision certificates |
| **123C** | Demolition standards / precautions |
| **123F** | Site class, SC/PR, open space, light/air, lanes, stairs / firefighting access / EVA envelopes, BFA (reg. 72), projections |
| **123J / 123O** | Refuse / material recovery; energy (OTTV etc.) where applicable |
| **123N** | Minor Works classification & simplified procedures |
| **123P** | Building (Inspectors) — MBIS professional side |
| **123Q** | Materials, loads, structure, geo/foundations/SF, enclosure, FRC, barriers, lifts — performance + APP-53 CoPs |
| **123I** | Measurement of UFA / GFA interfaces used by Cap 123A plan notes |

**Technical memorandum (s.39A):** sets when a supervision plan is required, supervision classes, manpower / qualifications, method statements, deviation / emergency amendment procedures — current TM binds the lodged plan (s.14A(3)).

### 12. Pre-schematic checklist (Cap 123 only)

1. Confirm Ordinance applies (not Cap 121 NTEH-exempt / not s.41 fully exempt land class).
2. Classify **use** (domestic / ND / composite / PPE / oil storage) — locks FSI certificate purpose and Cap 123F tables.
3. Overlay **Scheduled Areas 1–5** on site + basement / GI extent; if Area 1 or 5, assume landslip / groundwater / tunnel conditions in massing.
4. Appoint **AP**; trigger **RSE / RGE / RSC** from geo depth, spans, specialist works Gazette.
5. Choose track: **full s.14**, **Minor Works**, or **designated exempted / non-structural** — and verify carve-outs.
6. Design out **s.16** refusals: Cap 131 / MLP compliance, street access safety, neighbour collapse risk, FSI concept, no hand-dug caissons unless exception proven.
7. Protect **EVA**, legal **street openings**, and Cap **123F**-compliant **projections**.
8. Plan OP path: street works done, water + DFS certificates, lifts/liftway, performance review if geo unconventional / scheduled.
9. For conversions: run **s.25 material-change** + s.16(1)(g) neighbourhood test before planning reuse.
10. Dig into Cap **123A / 123F / 123Q** for numbers; Cap 123 only sets the gates those regulations feed.
