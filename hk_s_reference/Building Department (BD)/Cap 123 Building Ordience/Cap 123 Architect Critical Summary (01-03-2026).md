# Cap 123 — Buildings Ordinance
**Architect critical summary for schematic design**  
Consolidated version: 1 March 2026 | Cap. 123

> Scope note: Cap 123 is the **primary statute** — who must be appointed, when BA approval/consent is mandatory, what kills plan approval, OP/TOP gates, change-of-use notice, scheduled-area triggers, street/EVA/projection control, UBW / stop-work / slope / drainage enforcement, MBIS/MWIS, and built-in exemptions. It is **not** the SC/PR/light/air rulebook (Cap **123F**), not the plan-package rulebook (Cap **123A**), and not materials/structure performance (Cap **123Q**). At SD: settle AP/RSE/RGE + contractor class → map Scheduled Areas → freeze use class for FSI certificate → design so s.16 refusal grounds cannot bite → path to OP under s.21 → know which enforcement regimes can hit an existing-stock scheme.

---

## Regulatory Overview

Cap 123 controls the **planning, design and construction of buildings and associated works** in Hong Kong (and in the New Territories only as applied by Cap **121**). It applies to virtually every structure the Ordinance defines as a *building* (domestic, non-domestic, composite, PPE venues, industrial, underground space, oil storage ≥110 000 L, etc.) and to *building works* / *street works* — except Government / HA / Garrison land and the limited self-contained exemptions in s.41.

It is the **workflow and enforcement statute**: appoint registered professionals and contractors → submit regulated plans → obtain BA **approval** then **consent** (or use Minor Works / designated exempted works) → supervise to a **supervision plan** → obtain **OP/TOP** before occupation. It also arms BA to stop dangerous works, demolish UBW, order slope / drainage / EVA repairs, and force MBIS/MWIS cycles. Detailed dimensional / intensity / construction rules live in Cap 123 subsidiary legislation (A–Q) made under s.38 and in BA Codes accepted via PNAP.

## Critical main topics and subtopics

### 1. What counts as “building” / “building works” (s.2) — SD scope traps

| Term | Schematics implication |
|---|---|
| **building** | Extremely wide: whole or part of domestic / public / PPE buildings; factories, warehouses, offices, shops; arches, bridges, chimneys, docks, garages, hangars; walls, stairs, towers; sea-walls, jetties; **caverns / underground space** + access tunnels/shafts; oil storage installations; and anything BA Gazettes as a building |
| **building works** | Construction, **site formation**, **GI in scheduled areas**, foundations, repairs, demolition, A&A, **drainage works**, and every building operation |
| **plan** | Includes drawings, details, diagrams, **calculations**, structural and geotechnical details/calcs — submission duty is not “drawings only” |
| **street works** | Construct / form / reconstruct / alter / repair a **private street** or **access road** (surfaced, channelled, drained, lighted) |
| **access road** | Road on leased / licensed / ROW land providing access **only** to buildings used/intended mainly for habitation — and which is **not** a street |
| **private street** | Street on leased / licensed / ROW land |
| **street** | Includes square, court, alley, highway, lane, road, road-bridge, footpath, passage (whether thoroughfare or not) |
| **site formation works** | Excavations on sloping land, filling, landslip preventive / remedial works, **groundwater drainage works** |
| **new building** | Any newly erected building **or** existing building where ≥½ of volume is rebuilt **or** ≥½ of main-wall superficial area must be reconstructed — OP regime (s.21) and often “current standards” upgrade via s.17 item 1 |
| **domestic / non-domestic / composite** | Habitation includes hotel, guest-house, boarding-house, hostel, dormitory. Composite = partly domestic + partly non-domestic — drives Cap 123F PR maths and FSI “purpose” on plans |
| **habitation** | Use as hotel / guest-house / boarding / hostel / dormitory is **habitation** for Ordinance purposes |
| **oil storage installation** | Single tank or group with any tank ≥ **110 000 L**, above ground, for petroleum products → special Cap 123 regime + oil storage regs |
| **signboard** | Structure solely for advertisement / announcement / visual image — street-projection & UBW track under ss.31 / 39C / Sch. 8 |
| **EVA** | Vehicular access for FSD in fire / other emergency — keep clear for life of building (s.29A) |
| **design assumption / performance review** | Assumptions in design calcs; PR = AP report proving works were inspected/monitored and geo assumptions remain valid — can gate OP (s.21(6)(f)) |
| **contraventions** | Breach of BA order / notice / condition; material divergence from approved plans; for simplified MW — material divergence from submitted plans **or** failure to submit required certificates |
| **owner** | Includes Government lessee/licensee, mortgagee in possession, rent recipient, and often an agent — BA orders are not limited to the Land Registry “registered owner” name |
| **drain vs sewer** | Drain = one building (+ appurtenances); sewer = more than one building — combined drainage under s.28(2) is a multi-owner BA lever |
| **groundwater drainage works** | Subsurface water drainage — **distinct** from ordinary drainage works; heavy control in Scheduled Area 1 (s.28A) |
| **minor works** | Only items designated in regs (Cap 123N); simplified requirements are prescribed, not informal |
| **s.2(1B)** | Works without s.14 approval/consent are treated as MW under simplified requirements **if** the required prescribed professional/contractor is appointed (or prescribed contractor carries them out) |

**NT filter:** Cap 123 applies to the New Territories **only as provided by Cap 121** — confirm NTEH / Cap 121 exemptions before treating a village house as fully under Cap 123.

### 2. Team you must lock before schematic start (ss.3–4, 4A–4B, 9, 9AA, 2A)

| Role | When required | SD action |
|---|---|---|
| **Authorized Person (AP)** — architect / engineer / surveyor on BA register | Co-ordinator for all building works / street works **except** MW under simplified requirements | Appoint AP as scheme lead; AP owns co-ordination, contravention notices, supervision-plan duties |
| **Registered Structural Engineer (RSE)** | When Ordinance / regs require structural elements | Budget RSE for anything beyond Cap 123A small-works structural exemptions |
| **Registered Geotechnical Engineer (RGE)** | When Ordinance / regs require geotechnical elements | Trigger early if slopes, retention, deep dig, fill, scheduled areas, groundwater drainage |
| **Registered General Building Contractor (RGBC)** | All building / street works that are **not** specialized works and **not** minor works | Programme assumes registered contractor |
| **Registered Specialist Contractor (RSC)** | **Specialized works** designated by BA under **s.2A** (Gazette — not sub-leg) | Check current Gazette categories (demolition, foundation, site formation, ground investigation, ventilation, etc.) before locking constructability |
| **Registered Minor Works Contractor / prescribed professionals** | MW commenced under simplified requirements | Cap 123N track — no s.14 approval/consent; still need correct appointment |

**Standing AP / RSE / RGE duties (s.4(3)):**
1. Supervise per **supervision plan**
2. Notify BA of any regulation contravention that would result from carrying out approved-plan works
3. Comply with the Ordinance generally

**Replacement rule (s.4(2)):** if AP/RSE/RGE unwilling or unable to act → client must appoint another; temporary illness / absence from HK → original may nominate a peer for that period only.

**Contractor duties (s.9):** RGBC / RSC must continuously supervise under their supervision plan, notify BA of approved-plan regulatory contraventions, and comply generally. Specialized works need the RSC registered in the **matching category**.

**Minor Works professionals (ss.4A–4B, 9AA):**
- Knowingly failing to appoint required prescribed professionals / contractors = offence
- Prescribed professional on simplified MW must also ensure: FSI still meets s.16(1)(b)(ii) Code minimums; works do not breach any enactment, Cap 131 approved/partly approved/draft plan, or CDA master layout plan

### 3. Approval + consent — the irreversible SD gate (ss.14–16, 14A, 14AA, 19–20)

#### 3.1 Default rule (s.14)
**No person** shall commence or carry out any building works or street works without BA:
1. **written approval** of documents submitted per regulations; **and**
2. **written consent** to commence the works shown on the approved plan.

Approval / consent does **not** confer land title, waive lease terms, or legalise any Ordinance / other-enactment breach (s.14(2)).

| Path | Rule |
|---|---|
| Full BD track | Approval **then** separate consent to commence |
| Minor Works (simplified) | **s.14AA** — s.14(1) does **not** apply |
| Deemed approval / consent (s.15) | If BA does not refuse in writing (with grounds) within the **period prescribed in regs** (Cap 123A), approval/consent is **deemed given**. Refusal grounds are **not exhaustive** and do not imply approval of any other part of the plans |
| Supervision plan (ss.14A, 16(3)(bc)) | BA is **not deemed** to consent if AP has not lodged a supervision plan (unless BA does not require one). Prep must follow **current technical memorandum** under s.39A; preparer owns content |

#### 3.2 Consent clock / freeze (ss.16(3), 20)
- Consent refused if not all prescribed plans approved, fees unpaid, s.17 conditions outstanding, demo protection inadequate, **or no supervision plan lodged**
- Consent refused if **> 2 years** since approval of any prescribed plan
- If works **not commenced within 3 months** of consent, or **suspended for 3 months**, consent is **deemed revoked** — renewal (specified form) can bring new delay-related conditions (s.20)

#### 3.3 Plan-approval kill switches — design so these never appear (s.16(1))

| Ground | Practical SD reading |
|---|---|
| **(a)** Plans not prescribed / not as BA required | Wrong sheet set or missing Cap 123A contents |
| **(b) FSI certificate missing / failed** | DFS must certify either *no FSI needed* for stated purpose, **or** plans show **minimum FSI** per DFS CoP for that purpose. **Freeze intended use early** — purpose is written into the certificate |
| **(c)** Wrong / incomplete specified-form application | Admin kill — still stalls programme |
| **(d) Ordinance or Cap 131 conflict** | BO regs + **approved / partly approved / draft OZP** under Cap 131 |
| **(da) CDA + MLP conflict** | Inside a Cap 131 **comprehensive development area**, works must not contravene TPB-approved **master layout plan** |
| **(e)–(f)** Missing prescribed docs / unpaid fees | Cap 123A pack + fee schedule |
| **(g) Neighbourhood character** | Height, design, type or intended use **differing** from immediate neighbourhood or previously on same site — BA discretion; get early advice on odd massing / use insertions |
| **(h) Street opening / access danger** | Drop-offs, ramps, run-ins (not being street / access road) dangerous or prejudicial to street traffic safety / convenience |
| **(i)–(j)** BA needs more particulars / further submissions unsatisfactory | Incomplete geo / structural / planning narrative |
| **(k)** Plans materially differ from Cap 7 Landlord & Tenant exemption-approved plans | Rare legacy trigger on older stock |
| **(l)–(m) Collapse risk to neighbours / street / land** | Demo, site formation, piling, excavation, foundations that may collapse adjoining building, street or land — must prove avoidable |
| **(n) Cap 123F reg. 49 use** | Proposed use would breach Cap 123F reg. 49 |
| **(o) Lands resumption notice** | Site under Cap 124 resumption notice or lease resumption proviso |
| **(p) Inadequate street connexion** | Site that ought to have streets with adequate connexion to a public street — and does not |
| **(q) Scheduled Area No. 1 landslip debris** | In Mid-levels (Area 1), if BA says building must resist landslip debris and plans are inadequate |
| **(1A) Hand-dug caisson ban** | BA **shall refuse** unless: depth ≤ **3 m** and inscribed circle diameter ≥ **1.5 m**; **or** only practical method; **or** no other safe engineering alternative |

#### 3.4 Street-works plan refusal (s.16(2))
Same family of kills: non-prescribed plans, bad application, statutory breach, **dangerous access-road junction**, unpaid fees, insufficient / unsatisfactory particulars.

#### 3.5 Consent-stage geotech / neighbour / demo protection (s.16(3)–(5))
BA may refuse consent if:
- works would endanger adjoining buildings / streets / land; **or**
- for demolition — until adequate precautions by AP / RSE / RGE / RGBC / RSC are proven

#### 3.6 Urgent works without consent (s.19)
Emergency shore / underpin / demo / make-safe (building or land), or urgent street works, may start without consent if specified-form notice is in **before authorization or within 48 hours of start** (whichever earlier). BA can order stop once emergency ends. Urgent safety deviations from supervision plan allowed — AP must promptly notify BA with revised plan and further amendments.

### 4. Conditions BA can bolt onto your scheme (ss.17, 17A) — bake into programme

| Item | Trigger (Col. A) | What BA can demand (Col. B) — SD impact |
|---|---|---|
| **1** | A&A that creates a **new building** | Extra building-works plans so the **whole** complies with current structural stability + public health standards; withhold approval until those plans are also approved; require the additional works |
| **2** | New building served by **non-compliant existing access road** | Street works plans + construction to bring access road to regs |
| **3** | Street works on weak ground / heavy expected traffic | Carriageway built to **BA-prescribed** (not just default reg) standard |
| **4** | Any **access / opening to/from street** | Siting & construction to BA’s traffic / pedestrian safety standards |
| **5** | Building on **new private street** or **new / altered access road** | Street works plans approved **before** building consent; then build to approved plan |
| **6** | Structural use of materials; SF / excavation / piling / foundations / other structural works; **GI in scheduled areas** | Max loads/stresses; material tests & use; **instrumentation** to check design assumptions & monitor effects; workmanship; qualified supervision; **Area 1 sequence of works**; **performance review** for scheduled-area works **or** sites needing geo verification before OP / groundwater-regime impact / unconventional not-fully-proven design |
| **7** | SF / piling / excavation / foundations | Conditions to prevent collapse of adjoining buildings / streets / land |

**Sewage tunnels (s.17A):** if works are incompatible with Cap 438 sewage tunnel (or Gazetted proposed tunnel) works, BA may refuse / withdraw approval or consent, or condition piling / excavation / foundations.

### 5. Scheduled Areas (Sch. 5 + ss.16–17, 27B, 28A) — check **before** massing / basement

If any part of the works sits in a Scheduled Area, that part is treated as in the Scheduled Area (s.2).

| Area No. | What it is | SD / programme trigger |
|---|---|---|
| **1** | **Mid-levels** (plan GCB1, Land Registry) | Landslip-debris resistance (s.16(1)(q)); BA may dictate **sequence of works** (s.17 item 6); **groundwater drainage works** can be required **beyond site boundary** (s.28A) + lasting maintenance duty; Cap 123A typically forces SF + foundation together |
| **2** | NW New Territories (plan GS-SP/714-1) | Heightened geo / GI control; performance-review path; **s.27B** — if well abstraction makes a building dangerous, BA can order permanent stop + well closing |
| **3** | **Railway protection** corridors (MTR + former KCR / WR / ERE plan sets — Land Registry; updated through L.N. 221/2021) | Underground drainage exemption carve-out (s.41(3C)(f)); specialist GI / foundation / monitoring; early MTRCL / BA consultation |
| **4** | **Ma On Shan** (plan GS-SP/717-3) | Same family of geo / SF / foundation conditions |
| **5** | **Sewage tunnel protection** areas (plan set KCE/S/G/… & 90806/STPA/…) | Align with s.17A / Cap 438 — piling & basement envelopes sensitive |

**Performance review (s.17 item 6(g) + s.21(6)(f)):** if required, OP can be refused when BA is not satisfied the PR justifies monitoring and geo design assumptions. OP applications lodged within 60 days of PR receipt are timed from **60 days after PR receipt** for the deemed-grant clock (s.21(8)).

**Area 1 groundwater drainage (ss.28A–28D) — basement / Mid-levels special:**
- BA may require plans showing groundwater drainage **even beyond lot boundary**
- Approval includes a **maintenance obligation** for a BA-specified period
- If neighbouring owners object / will not agree → applicant (or successor) applies for BA authorization; 30-day objection period + Gazette / newspaper notices
- BA may authorize (with conditions / alterations) entry onto identified land
- Magistrate warrant can overcome obstruction (Sch. 6 form)
- Authorized person must make good damage; compensation disputes → Lands Tribunal (normally within 3 years)

### 6. Occupation before OP = offence territory (s.21)

| Rule | Detail |
|---|---|
| Occupation ban | No **new building** may be occupied except by ≤ **2 caretakers**, unless OP issued, or unexpired / unrevoked **TOP** covers the occupied part |
| TOP | May cover whole or completed part; BA may limit duration and set conditions; revoke on breach; OP of whole building revokes remaining TOPs |
| After TOP expiry / revoke | s.21(1) reapplies after **7 days** if no OP |
| Deemed OP/TOP | If BA has not refused in writing within **14 days** of specified-form application → permit deemed granted (subject to PR timing rule above) |

**OP refuse grounds (s.21(6)) — design completion criteria:**
- Any building works contravened Ordinance
- Required street works for private street / access road incomplete
- Liftway present but lift not installed **and** liftway not protected to BA satisfaction
- Where DFS certified FSI needed under s.16(1)(b)(ii): no DFS certificate that installed FSI matches plans and is in efficient working order
- Required water supply connexions not proven
- Required **performance review** fails BA’s adequacy / geo-assumption test

### 7. Change of use — material change needs 1-month notice (s.25)

| Step | Rule |
|---|---|
| Notice | Person intending a **material change** in use of a building must give BA **1 month’s** notice in specified form |
| BA powers | Within 1 month of notice: prohibit intended use; or order discontinuance of present use within 1 month; may permit building works to make building suitable |
| When is change “material”? | (a) erecting a building for that use would have breached the Ordinance; **or** (b) BA could have refused plans under **s.16(1)(g)** (neighbourhood height/design/type/use) |

**SD implication for A&A / conversion:** hotelisation, domestic↔non-domestic flips, industrial-to-office, etc. are often material — treat as BO + OZP + lease + Cap 123F use-fitness problem from day one, not a fit-out afterthought.

### 8. Shoring rights when your works need them (ss.18–18A)

Where demolition precautions or s.17 conditions require shoring:
- Proponent may erect / maintain / remove shoring **on the project building**
- Shoring in a **street** needs DB / authorized BD officer permission + conditions
- Does **not** authorize shoring on third-party land or third-party buildings without separate rights
- Must promptly make good damage and compensate loss
- Authorized entry for survey / install / maintain / inspect can be backed by magistrate warrant (bilingual posting)
- Compensation disputes → Lands Tribunal (normally within 3 years)

**SD takeaway:** adjoining-owner shoring / Party Wall-type negotiations are a programme risk on tight urban sites — start early; Ordinance alone does not give you a free right to shore the neighbour.

### 9. Streets, openings, EVA, projections — envelope & public-realm hold points

#### 9.1 Private streets / access roads (s.29)
Frontagers must surface, channel, sewer, drain, light and maintain to BA satisfaction / regs. Inadequate roads → BA orders works or carries them out and recovers cost (apportioned by frontage or by lot area for access roads). Dangerous roads → emergency works. Programme new development with street-works path when s.17 items 2/5 apply.

#### 9.2 EVA maintenance (s.29A)
EVA owner must keep EVA in good order. Dilapidation, defect, alteration or addition that defeats fire-emergency purpose → BA order to reinstate; BA may execute / register order / recover cost. **Do not sacrifice EVA clear width / turning / loading at SD for landscape or parking gain.**

#### 9.3 Openings to / from streets (s.30)
No construction / formation / alteration of any means of access or opening to/from a street without BA consent (standalone or as part of building/street works consent). BA refuses if dangerous or prejudicial to traffic / pedestrian safety or convenience; may condition siting and construction. These works are treated as **street works** for enforcement sections.

#### 9.4 Projections on / over streets (s.31)
No building / structure in, over, under or upon any street (leased or not) unless:
- it complies with Cap **123F** Part II criteria; **or**
- it is a **signboard** whose location/dims do **not** create: structure danger; traffic hazard; fire hazard; obstruction of traffic signs / signals / markings / monitoring equipment; obstruction or reduction of MOE / firefighting-access width or height; light/air reduction below regs; or public danger; **or**
- BA grants **s.42** exemption

BA can order alteration / removal of projections over street or unleased GLA (generally within 3 months or shorter); may execute and recover cost. **Canopies, bridges, architectural projections, signboards** are SD space claims — check Cap 123F Part II + PNAP early.

### 10. Drainage, slopes, pipes-in-slopes — site & envelope risk (ss.27A–28)

#### 10.1 Dangerous hillsides / retaining structures (s.27A)
BA may order owner (or Government-lease maintenance obligor) to appoint AP/RSE/RGE, investigate, propose remedies, and carry out approved works where natural / formed / man-made land or earth-retaining structures are dangerous or liable to become so (collapse risk → injury / property damage). Order registrable; BA may execute on default / refuse proposals; emergency action available. **Sloping sites and podium-edge retention are Cap 123 live liabilities for the life of the building — not only construction-phase.**

#### 10.2 Pipes / drains / sewers in slopes (s.27C)
Where leakage / defect / inadequacy could cause landslip or retaining collapse, BA may require AP investigation, report, remedial proposals and approved works. Before ordering, BA considers risk info including pipe age and investigation / maintenance history. Private owners remain exposed even if the pipe runs under HA / Garrison / unleased / Gov street land (s.41(1A) carve-out).

#### 10.3 Drainage of buildings (s.28)
- Owner must carry out all drainage works to BA satisfaction and regs
- BA may require a **combined drainage plan** for a group of buildings (more advantageous together than separately)
- Inadequate / defective / insanitary drains or sewers → order works, AP investigation, remedial proposals; BA may approve / amend / refuse; then order approved works
- Default / refusal → BA executes + recovers; order registrable; emergency intervention available

### 11. UBW, stop-work, demolition, closure — enforcement that shapes A&A / existing-stock SD (ss.22–24C, 26–27)

#### 11.1 BA entry & investigation (s.22)
BA may enter to assess danger, inspect drainage / groundwater drainage, verify compliance, and perform authorized works. Outside emergency: need owner/occupier consent **or** magistrate warrant. Warrant grounds include reasonable suspicion of material plan deviation, structural / public-health / fire non-compliance, unlawful change of use, danger, defective drainage, or uncomplied orders. Contractors must provide access; BA may make openings, take samples, order AP tests.

#### 11.2 Stop-work (s.23)
BA may order works to cease where they: contravene Cap 123; cause / likely cause collapse or dangerous conditions; present dangerous site conditions; materially deviate from supervision plan / TM in a way that could lead to danger; or breach / cannot achieve an approval or consent condition. Works must stop as quickly and safely as possible.

#### 11.3 UBW demolition / alteration orders

| Section | When used | SD relevance |
|---|---|---|
| **s.24** | Unauthorized building / street works (other than simplified MW) | Classic UBW order — demolish / alter; serve owner, frontagers, signboard parties, or connected premises owner; registrable; BA may execute + recover (including from later owners if registered) |
| **s.24AA** | Simplified MW that breach Cap 123, Cap 131 plans, or CDA MLP | Same family for MW track |
| **s.24A** | Dangerous works | Order specified remedial method + programme; BA may execute without further notice |
| **s.24B** | **Priority demolition** via District Court | Imminent danger; UBW for sale/letting/disposal; seriously detrimental common-part UBW; or public nuisance. Posted notice freezes other works, may close premises |
| **s.24C** | UBW **notice** (warning / registration track) | Identifies location, contravention, BA powers, deadline before Land Registry registration if not rectified |

#### 11.4 Dangerous / defective buildings & closure (ss.26–27)
- **s.26 Dangerous buildings:** order demolition, make-safe, specified works, shoring, public fencing/hoarding, closure + deadlines; registrable; BA may execute / emergency works
- **s.26A Defective buildings:** order specified works, AP investigation + remedial proposals; approve/amend/refuse; require approved works; execute on default
- **s.27 Closure Order (District Court):** where building is dangerous or needs closure for safe statutory works — entry prohibited except BA permission; police may remove persons; BA may seal entrances

**SD implication:** adaptive reuse / A&A must survey for UBW, illegal projections, defective envelopes and slope pipes — these are not “tenant fit-out” issues; they can freeze consent, force priority demo, or trigger closure.

### 12. When you can *skip* s.14 approval / consent (ss.41–42) — use with care

| Exemption | Exempt from | Hard carve-outs (still controlled) |
|---|---|---|
| **Gov buildings**; HA land (subject to Cap 283); Garrison land; certain unleased / vested land; Gov streets | Ordinance generally | Projections over streets + hoardings still apply; **s.27C** pipe-in-slope duties may still bite private buildings whose pipes lie in exempt land (s.41(1A)) |
| Existing lawfully built buildings / private streets / access roads (s.41(2)) | No forced upgrade merely by Ordinance existence | New A&A / new-building rebuild thresholds still trigger current standards via ss.14 / 17 |
| **Non-structural works in a building** (s.41(3)) | ss.4, 9, 9AA, 14(1), 21 | **Still not exempt:** drainage works; **GI in scheduled areas**; **site formation**; **minor works**; anything involving **structure**. Must still comply with regs (s.41(3D)) |
| **Designated exempted works** prescribed in regs (s.41(3B)) | ss.4, 9, 9AA, 14(1), 21 | Must still comply with regs |
| **Drainage works** non-structural (s.41(3C)) | ss.4, 9, **14(1)** only (not s.21) | Not exempt if involving: structure; trade effluent / chemical / petroleum / acid / grease / oil drains; altering public-sewer manhole connexion; septic tank / cesspool alteration or additional connexion; **underground drainage in Scheduled Area No. 3** |
| Temporary building used as PPE (s.41(4)) | Ordinance provisions | Controlled by Places of Public Entertainment regime |

**s.42 modifications:** BA may permit Ordinance modifications on application + fee when “special circumstances” so require — **never to prejudice** structural stability or public health standards. **s.42 cannot waive s.14** (approval/consent). Past exemptions create **no precedent**. Breach of conditions invalidates the exemption (no defence thereafter).

### 13. MBIS / MWIS — existing-stock schemes (Part 2A, ss.30A–30F)

**s.30A:** Part 2A does **not** apply to domestic buildings not exceeding **3 storeys**.

| Regime | Age trigger | Scope | Re-notice lock-out |
|---|---|---|---|
| **MBIS (s.30B)** | Building ≥ **30 years** (from OP date, or other BA evidence) | Prescribed inspection / repair of **common parts**, non-common **external walls**, prescribed owner-connected **projections**, **signboards**. If dangerous → repair to render safe | Fresh notice for same part not within **10 years** after compliance |
| **MWIS (s.30C)** | Building ≥ **10 years** | Window inspection / repair; exclusive-use window notice served only on that owner. Dangerous windows → repair to safety | Fresh notice for same window not within **5 years** after compliance |

**Who does what:**
- **MBIS (s.30D):** appoint **RI** for inspection and RI for repair supervision (may be same person). Inspection personal unless regs exempt. Supervising RI ensures compliant materials / correct installation and building rendered safe. RI must notify BA of emergencies and report detected UBW at common parts, non-common external wall / roof / podium, adjoining yard / slope, or fronting street. **RI cannot also be the repair contractor** for the same part. Inspecting RI cannot nominate temporary substitute; supervising RI may temporarily nominate during illness / absence.
- **MWIS (s.30E):** appoint **qualified person** (AP / RSE / RI / RGBC / RMWC registered for applicable window works) for inspection and supervision. Natural-person QP (or prescribed representative of corporate QP) must personally inspect. RGBC / RMWC QP **may also perform the repair**. No temporary nomination if QP unavailable — must replace.
- **s.30F:** contractor doing prescribed repair must ensure the part is rendered safe.

**SD alerts:** aged envelopes, illegal signboards, cantilevered projections, and window replacement strategies are programme / cost risks — survey early; coordinate RI / QP path with façade and A&A packages.

### 14. UBW validation path for prescribed works (s.39C + Sch. 8)

BA **must not** serve a s.24 order or s.24C notice **solely** because prescribed building / works were completed in contravention of s.14(1) where:
- completed before **31 December 2010** and the inspection / document / rectification path below is satisfied; **or**
- for **Schedule 8** items — completed before the relevant Cap 123N prescribed date, requirements met, and any required **periodic** compliance maintained

**Required path:**
1. Appoint prescribed AP / RSE / RI / RGBC / RSC / RMWC to inspect
2. Submit / deliver prescribed plans, certificates, notices, documents
3. If safety requires alteration / rectification / reinforcement → carry out those **minor works** via prescribed registered contractor under simplified requirements

**This is not general legalization.** BA retains ss.24 / 24C power on any ground other than the historic s.14 / simplified-requirement breach.

**Schedule 8 prescribed types** (dims / types set in regs under s.38(1)(ke)(ic), not in Sch. 8 itself):
1. Signboard  
2. Supporting structure / metal casing for building-services installation  
3. Radio base-station supporting structure  
4. AC-unit or light-fitting supporting frame  
5. Solid fence wall  
6. Mesh fence or metal railing  
7. Pole  
8. Metal gate  
9. Canopy  
10. Retractable awning

### 17. Pre-schematic checklist (Cap 123 only)

1. Confirm Ordinance applies (not Cap 121 NTEH-exempt / not s.41 fully exempt land class).
2. Classify **use** (domestic / ND / composite / PPE / oil storage) — locks FSI certificate purpose and Cap 123F tables.
3. Overlay **Scheduled Areas 1–5** on site + basement / GI extent; if Area 1 or 5, assume landslip / groundwater / tunnel conditions in massing.
4. Appoint **AP**; trigger **RSE / RGE / RSC** from geo depth, spans, specialist works Gazette.
5. Choose track: **full s.14**, **Minor Works**, or **designated exempted / non-structural** — and verify every carve-out.
6. Design out **s.16** refusals: Cap 131 / MLP compliance, street access safety, neighbour collapse risk, FSI concept, no hand-dug caissons unless exception proven.
7. Anticipate **s.17** conditions: access-road upgrade, street works before consent, instrumentation, Area 1 sequencing, performance review.
8. Protect **EVA**, legal **street openings**, Cap **123F**-compliant **projections**, and shoring / neighbour strategy (s.18).
9. Plan OP path: street works done, water + DFS certificates, lifts/liftway, performance review if geo unconventional / scheduled.
10. For conversions: run **s.25 material-change** + s.16(1)(g) neighbourhood test; survey UBW / Sch. 8 items / s.39C path.
11. For aged stock: check MBIS (≥30 yr) / MWIS (≥10 yr) exposure; slope pipes (s.27C); EVA defects (s.29A).
12. Dig into Cap **123A / 123F / 123Q** for numbers; Cap 123 only sets the gates those regulations feed.
