# Cap 123K — Building (Oil Storage Installations) Regulations
**Architect critical summary for schematic design**  
Consolidated version: 21 October 2021 | Cap. 123 sub. leg. K

> Scope note: Cap 123K is the **licensing, operation, inspection, and A&A-control statute** for large above-ground oil tanks. It does **not** itself give spacing tables, bund volumes, or drainage geometry — those live in the **Code of Practice for Oil Storage Installations 1992** (deemed to satisfy Cap 123K via **PNAP APP-10**). This summary therefore folds Cap 123K + APP-10 + the 1992 Code into one SD brief. Use it whenever the scheme includes bulk petroleum storage, tank farms, jetty-linked receipt/issue lines, A&A to a licensed installation, or demolition of tanks.

---

## Regulatory Overview

Cap 123K applies to every **oil storage installation** under Cap 123 s.2 — any **above-ground tank ≥ 110 000 L**, or a **group of tanks in the same cluster / bunded area** where **any one tank ≥ 110 000 L**, storing **petroleum products** — plus **associated works** (bunded area, drainage, in-bund pipelines, receipt/issue pipelines to related jetties, related surface-water drainage). It imposes a performance duty to minimise **pollution, fire and explosion** (reg. 3), makes storage **licence-dependent**, and locks annual inspection / settlement / A&A hold points. Numerical layout standards are taken from the **1992 Code** (APP-10 §3: compliance with the Code = deemed to satisfy Cap 123K).

## Critical main topics and subtopics

### 1. Does Cap 123K bite? — capacity, location, product class

Lock this at concept before committing site layout, bund geometry, or consultant fees.

| Criterion | Threshold / rule | SD implication |
|---|---|---|
| Capacity | Single tank **≥ 110 000 litres**, **or** any tank in a cluster / bunded group **≥ 110 000 litres** | Below threshold → Cap 123K OSI licensing regime does **not** apply (FS / EPD / Cap 295 DG may still apply) |
| Location | Constructed **above ground level** | Buried / below-ground tanks are outside this definition |
| Contents | **Petroleum products** — crude, feedstock, semi- / wholly refined petroleum that is liquid or solid at ambient T/P (Cap 123 s.2) | Confirm product class early; “chemical” storage ≠ automatically petroleum |
| Cluster rule | Same **cluster or bunded area** (APP-10) | Do not “split” tanks across separate pads to dodge the threshold if they share one bund / cluster |
| Code definition of storage facility | Group of tanks + pipelines within a required bunded area (or series of bunded areas with shared facilities) + receipt/issue lines to related jetties | Model the whole facility, not just tank footprints |

**DG flash-point classes used by the 1992 Code** (Cap 295 Category 5 classification):

| Class | Flash point | SD planning note |
|---|---|---|
| **Class 1** | Below **23°C** | Strictest spacing, floating-roof rules, 105% bund |
| **Class 2** | **23°C–66°C** inclusive | Lighter spacing, but ≥10 m (or 6 m if total capacity <7000 m³) from outer boundary; plan as Class 1 if future Class 1 use is possible |
| **Class 3** | **≥ 66°C** | Spacing by construction / ops convenience only |
| Artificially heated above flash point | Treat as **Class 1** | Heated Class 2/3 product = Class 1 rules |

**SD takeaway:** If any above-ground tank hits **110 000 L**, budget from day one: Cap 123A plans + s.14 approval/consent, Cap 123K licence path, **RSE**, bund + drainage as building works, annual ops / inspection cost, and FSD / EPD parallel licensing.

### 2. Associated works — what must be inside the regulated envelope (reg. 2 + Code)

Cap 123K treats the tank as inseparable from its safety geometry. At schematic these are **part of the installation**, not optional M&E:

| Element | What it means for layout |
|---|---|
| **Bunded area** | Area **completely surrounded** by bund walls / embankments sized to **retain spillage** from any tank inside |
| Drainage system of the installation | Contaminated-water path: bund → controlled valve/pump → interceptor → monitored outfall — **not** free gravity to storm drain |
| Pipe-lines **within** the bunded area | All in-bund pipe runs are regulated associated works |
| Receipt & issue pipe-lines to **related jetties** | Marine interface is inside Cap 123K — coordinate jetty, berth, boom storage, coastal setbacks early |
| Related **surface water** drainage | Site SWD resolved as part of the installation package |
| Security enclosure (Code §12.1) | Fence **≥ 1.8 m** within lot; **≥ 2 exits** separated by **≥ 5 m**; gates open outwards; not self-locking |

**SD takeaway:** Site coverage ≠ building footprint. Bund walls, embankment slopes, freeboard, interceptors, jetty corridors, marine pollution stores, and security fence eat plot area and may conflict with OZP / lease setbacks — model them in early massing.

### 3. Overarching design duty & compliance path (reg. 3 + APP-10)

Every oil storage installation, its associated works, **and any A&A**, must be designed and constructed to **minimise**:

1. **Environmental pollution**
2. **Fire**
3. **Explosion**

| Path | Source | Use at SD |
|---|---|---|
| Performance duty | Cap 123K **reg. 3** | Concept risk narrative |
| Deemed-to-satisfy | **Code of Practice for Oil Storage Installations 1992** (APP-10 §3) | Primary technical brief — bund %, spacing, drainage, marine kit |
| Plan approval | Cap 123 **s.14(1)** approval + consent **before** construction (APP-10 §2) | No “build then licence” |
| Parallel regimes | FSD (fire fighting, Cap 295 DG licence conditions); EPD (effluent, chemical waste, marine dispersant licences); general structure under Cap **123Q** | Do not assume Cap 123K alone clears FSD / EPD |

Also note (Code §1.1): Hong Kong’s intensive coastal land use makes **spillage retention** a primary Code objective alongside fire/explosion.

### 4. Safety distances & tank layout (Code §2.1–2.4) — hard geometry for massing

Distances are **horizontal in plan** between nearest points of tanks, filling points, openings in buildings, boundaries. They are **minimums**. Layout must also satisfy **Director of Fire Services** for fire-fighting access (Code §2.4) — spacing ≠ circulation for appliances.

#### 4.1 Class 1 — fixed roof tanks (Table 1)

| Factor | Minimum distance |
|---|---|
| Between groups of small tanks (see grouping rule below) | **15 m** |
| Between a group of small tanks and any tank outside the group | **15 m** |
| Between tanks not in a small-tank group | Least of: **½ larger diameter**, **smaller diameter**, or **15 m** — but **never < 10 m** |
| Tank ↔ filling point / filling shed / building | **15 m** |
| Tank ↔ outer boundary of storage facility / designated non-hazardous area / fixed ignition source | **15 m** |

**Small-tank grouping (Class 1):** tanks each **≤ 10 m diameter**, none containing Class 2/3 artificially heated above flash point, may be treated as one tank. Groups may sit together; **aggregate capacity of a group ≤ 8000 m³**. Within a group, spacing is by construction / ops convenience only.

#### 4.2 Class 1 — floating roof tanks (Table 2)

| Factor | Minimum distance |
|---|---|
| Between two floating roof tanks | **10 m** if larger tank ≤ **45 m** dia; **15 m** if larger tank **> 45 m** dia |
| Floating roof ↔ fixed roof tank | Same formula as Table 1 tank-to-tank (½ larger / smaller / 15 m; min **10 m**) |
| Floating roof ↔ filling point / filling shed / building **without** ignition source | **10 m** |
| Floating roof ↔ outer boundary / non-hazardous area / fixed ignition | **15 m** |

**Height multiplier:** for tanks **> 18 m high**, multiply Table 2 distances by **height / 18**.

#### 4.3 Class 2 / Class 3

| Product | Rule |
|---|---|
| Class 2 | Spacing by construction / ops convenience; but space from Class 1 tanks per Table 1; **≥ 10 m** from outer boundary of facility |
| Class 3 | Spacing by construction / ops convenience only |
| Future Class 1 conversion | If Class 2 tanks may later store Class 1 → **plan to Class 1 distances now** |

#### 4.4 Total installation capacity **< 7000 m³** (Table 3 — relaxed Class 1)

| Factor | Minimum distance |
|---|---|
| Between tanks ≤ **10 m** dia **and** ≤ **14 m** high | Construction / ops convenience |
| Between tanks > 10 m dia **or** > 14 m high | Least of ½ larger / smaller / 15 m; min **10 m** |
| Tank ↔ filling point / shed / building | **15 m**; may reduce to **≥ 6 m** if tanks ≤ 10 m dia |
| Tank ↔ outer boundary / non-hazardous / ignition | **15 m**; may reduce to **≥ 6 m** if tanks ≤ 10 m dia |
| Class 2 from outer boundary (this capacity band) | **≥ 6 m** (not 10 m) |

#### 4.5 Floating roof mandate (Code §2.2–2.3)

| Rule | Detail |
|---|---|
| Mandatory floating roof | Class 1 product tanks **> 1000 m³** capacity |
| Acceptable floating roof | Open-top pontoon / double-deck per **API 650**; **or** fixed metal roof with top/eaves ventilation + qualifying metal floating cover (API 650 pontoon/double-deck, or liquid-tight metal flotation that still covers liquid when **half flotation lost**) |
| Counts as **fixed roof** (stricter spacing) | Internal metal pan / cover that fails §2.3(ii); or plastic-foam flotation (except seals), even if metal/fibreglass encapsulated |

**SD takeaway:** Product class + roof type + tank height/diameter drive site width before architecture starts. Lock Class 1 vs 2/3 and floating vs fixed early — switching later blows bund and boundary setbacks.

### 5. Bunded areas — containment, fire walls, construction (Code §4)

#### 5.1 Containment volume (the SD volume driver)

| Product in bund | Minimum containment | Displacement rule |
|---|---|---|
| **Class 1** | **≥ 105%** of max operating capacity of the **largest** tank | Exclude displacement of all other tanks and foundations |
| **Class 2 / Class 3** | **≥ 100%** of max operating capacity of the **largest** tank | Same exclusion |
| Class 2/3 linked bunds | Volume of another bunded area may count if connected by relief spillways / weirs sized to carry overflow from largest tank releasing **100% contents in 15 minutes** | Class 1 bunds must be **self-contained** — **no** weirs/overflows to other areas (but may contain Class 2/3 tanks inside) |

#### 5.2 Fire walls / compartmentation (Code §4.1)

| Item | Rule |
|---|---|
| Purpose | Limit fire spread; enable fire fighting |
| Fire wall height (from outside ground) | **Min 2 m – max 3.7 m** (personnel protection) |
| Location | Allow reasonably close approach to a tank fire; adequate escape over fire walls |
| Max Class 1 capacity **per fire-wall compartment** | Fixed roof: **≤ 60 000 m³**; Floating roof: **≤ 120 000 m³** |
| Below those figures | Bund walls themselves may serve as fire walls — no further compartmentation required |

#### 5.3 Wall & floor construction (schematic material decisions)

| Topic | Hard rule / practice |
|---|---|
| Design fluid density for bund strength | Take petroleum density **= water** |
| Earth / rock bund | Profile & compaction per **BS 6031**; impermeable surfacing; inner face protected (concrete blinding / precast / render); top/outer face erosion-resistant (chunam / grass) |
| Acceptable sand-cement render facing | 4:1 sand:cement; **≥ 50 mm**; light wire mesh; panels with sealed movement joints; waterproofing admixture recommended |
| RC / prestressed / reinforced masonry bund walls | Per Cap 123Q / Building (Construction) Regs; masonry that absorbs product must be rendered impermeable |
| Joints | Water seal + bonded intersections; expansion joints in bund wall **≤ 30 m** centres |
| Concrete bund floor | **≥ 125 mm**, reinforced, on solid ground; steel ≥ **0.25%** each way (plain) or **≥ 0.20%** (deformed / HY mesh); sealed bay joints |
| Bitumen sealant | **Not** for bunds containing **Class 1**; elsewhere renew when embrittled (may need annual repair) |
| Asphalt / bitumen floors | Account for solvent attack + UV; consider reflective surface layer |
| Alternative floor materials | Demonstrate in **48 hours** product penetrates **≤ 50%** of floor thickness |
| Compacted granular rock as bund floor | **Not permitted** |
| Membranes (polythene / neoprene etc.) | Required when granular soil, drinking/vital water table, nearby waterways, adjacent buildings, low-lying seepage, or seepage beyond property — must be UV/physical protected, continuous, joined to walls; abandon if predicted differential settlement would rupture membrane |
| Reclamation sites | Spec floor with extreme care — contamination of fill can leak product to sea / watercourses |

**SD takeaway:** Bund volume (105%/100% of largest tank) + fire-wall compartments + membrane triggers on coastal/reclamation sites are the biggest plot eaters after tank spacing.

### 6. Foundations, settlement & tank pads (Code §3 & §5) — licence renewal drivers

Choice of foundation is the RSE’s, but Cap 123K **reg. 7** makes settlement a **licence-renewal gate**. Design for lifelong monitoring from day one.

| Topic | Design / SD rule |
|---|---|
| References | **BS 2654:1984**, **API 650**, Cap 123Q for general structure |
| Profiles | **Cone up** (centre higher) or **Cone down** (to central sump) |
| Compacted granular pad | Sound durable granular; 150 mm layers; 6–10 t roller; top permeable drainage layer; **50 mm** bitumen-sand weatherproof / spill barrier (rigid plaster/concrete **not** recommended — cracks) |
| Pad edge | Slope away from tank **≥ 1:10** across **≥ 500 mm** strip, then down to bund floor at **1:1.5**; peripheral seal after water test |
| Concrete ring wall | Distributes shell load + horizontal surge forces (API 650 App B); **no perforations/penetrations**; flexible sealed joint if surrounded by bund paving; may sit on raft/pile cap; can resist wind overturning / uplift |
| Ground improvement (stone columns etc.) | May behave as piles — treat as piled design if so |
| Settlement | Calculate from SI; evaluate effect on tank + **connecting pipework**; differential settlement vs pipe racks may **override** minimum spacing |
| Adjoining structures | Tank foundation spread of load may force greater spacing than fire tables |
| Tank bearing pad level | High enough that severe stormwater does **not** pond to steel base |
| Corrosion / marine climate | HK tropical marine + industrial pollutants + seawater fire-fighting tests — peripheral seal must be watertight, flexible, **inspectable**; underside of bottom plate needs permanent water barrier |

### 7. Drainage, interceptors & contaminated-water logic (Code §6–7) — architectural/civils SD

This is the system that makes Cap 123K pollution control real. Do not design “normal” storm drains through bunds.

#### 7.1 Two drainage worlds (Code Fig. 6.1)

| Area type | Path |
|---|---|
| **Not** normally contaminated (roads, remote yards, offices) | Normal storm / foul |
| **May be** contaminated (bunds, filling areas, pipe racks/trenches, loading/unloading jetties, kerbed pump areas) | Manual valve/penstock or **non-automatic** pump → **interceptor** → monitoring/sampling → outfall (public SWD / watercourse / sea) |

#### 7.2 Bunded-area drainage hard rules

| Rule | Detail |
|---|---|
| No automatic drain-down | Surface water from bunds must **never** drain automatically |
| Allowed discharge methods | (i) **Non-automatic** pumped system (manual start after rain); **or** (ii) gravity via **manually** operated valves/penstocks (may be motorised but manual activation) |
| Containment integrity | When pumps off / valves closed → **no leakage** (bund still holds largest-tank spill) |
| Prefer channels in large bunds | Collect to discharge point; minimise floor falls |
| Avoid intermediate sand traps / chambers in bund | They retain product / sludge |
| Access | Walkways to all tanks and valves **before** bund is emptied of rainwater |
| Valve / penstock location | Operable from **outside** the bund; discharge into open pit/channel so flow is **visible** from operator position; rising spindle or indicating headstock showing fully closed |
| Weatherproof signs | At every valve/penstock — purpose + when to open/close |
| Design storm | Max flow for **1-in-10-year** HK rainstorm |
| Contaminated piped drains | Continuously **surcharged**; pipes horizontal; water seal **≥ 50 mm**; open channels must run dry after rain |
| Self-cleaning drains (where used) | Min velocity **0.75 m/s** at quarter depth |
| Floating roof tank roof drains | Primary + emergency per BS 2654 / API 650; size for 1-in-10-year intensity; compatible with fixed fire-water systems |
| Pumps / hose points | Concrete floor + perimeter kerbs → sump + outlet valve → interceptor (or steel drip pan for isolated pumps) |

#### 7.3 Interceptors (Code §7.1)

| Requirement | Detail |
|---|---|
| Where required | Every SW drain leaving a may-be-contaminated area |
| Sizing | Individually designed for max SW / product flow; API / IP methods OK if outfall meets **Director of Environmental Protection** standards; multiple-plate designs allowed |
| Location | **Outside** bunded areas; all bund water through interceptor |
| Bypass | Generally **no** bypass (except storm overflow for large **unbunded** high-risk areas under §6.3 / §7.1(vi)) |
| Storm overflow (unbunded only) | Treat first **(t+2)** minutes of 1-in-10-year storm, where **t** = time of concentration from furthest part of catchment |
| Access / visibility | Prefer open chambers with guardrail; if partly covered, clear view of inlet, each chamber portion, outlet |
| Product removal | Means to remove floating product |
| Outlet valve / penstock | Between outlet weir and storm drain / outfall; clear open/closed indication; operable even if interceptor full to brim |
| Sample point | At each outlet valve for effluent QC |
| WPCO licence | Effluent from installations in Water Control Zones needs Cap **358** licence; outside zones may still need DEP discharge consent (Code §9.4) |

**Ops rules that drive SD (must be reflected in ops instructions):** bund drainage valves normally **closed**; checked/closed end of each day; **closed while tank is receiving stock**; do not start stormwater discharge before rain stops unless interceptor designed for that pattern; skim / clean interceptors; chemical-waste sludge under Cap **354**.

### 8. Tanks, valves, leak detection, electrics, pipelines (Code §5, §8, §12.2)

| Topic | Rule |
|---|---|
| Vertical steel welded tanks | **BS 2654:1984** or **API 650** (design, fab, erection, test) |
| Horizontal steel tanks | **BS 2594:1975** + App to **BS 799 Part 5:1975** |
| Leak detection | **Each tank** needs a suitable leak detection system |
| Valves | BS / API; prefer full-bore; corrosion/abrasion resistant; jetty / pipeline terminal valves **steel throughout**; clear position indication if not rising spindle; heatable if product may solidify |
| Electrical / lightning / static | Institute of Petroleum **Electrical Safety Code**; static protection per European Model Code Part II §9 or BA-approved equivalent |
| Pipelines | BS / API; prefer all-welded; thermal expansion; **flexibility for expected settlement** at major tanks; isolating blanks with clear location/setting indication |

### 9. Licensing gate — cannot store without a licence (regs. 5–6 + APP-10)

#### 9.1 Absolute prohibition (reg. 5)
No person shall store (or cause / permit storage of) petroleum products in an oil storage installation **without a licence**, or otherwise than in accordance with licence terms/conditions.

#### 9.2 What BA needs before granting (reg. 6(4))

BA **shall not** grant unless **all** are true:

| Gate | Requirement | Delivery implication |
|---|---|---|
| (a) | Built under Cap 123 and **conforms to BA-approved plans** | Cap 123A plan set + s.14; as-built match |
| (b) | Constructed per **reg. 3** **and tests** prove it | Commissioning / integrity / water tests; RSE oversight |
| (d) | **Operation instructions** satisfactory for pollution / fire / explosion | Ops manual is a licensing deliverable |

Application (reg. 6(1)): specified form + ops instructions. Fee (reg. 6(3)): **$47,000** (as printed 21-10-2021) — BA may impose conditions. BA may **revoke** for breach of regs or licence terms (reg. 6(5)).

#### 9.3 APP-10 practical package for **new** installations

| Document | Requirement |
|---|---|
| Operation instructions | **6 English + 1 Chinese** (Chinese essential for site grades) |
| Plans | One set of **general arrangement** plans |
| Prerequisites before licence | **Occupation permit** + **certificate of general inspection** |
| Critical path | **OP → licence → first fill** — not OP → handover |

For endorsement of A&A where ops change: amended ops instructions + GA plans.

### 10. Operation instructions — controlled living document (regs. 6A, 6B + Code §9)

| Rule | Detail |
|---|---|
| Amend only with BA written authorization | reg. 6A(1) |
| Amended ops must still minimise pollution / fire / explosion | reg. 6A(2) |
| Operate to latest BA-accepted version | reg. 6B |
| Content focus (Code) | Only procedures pertinent to fire, explosion, environmental pollution |
| Language | English for licensing; **also Chinese**; bilingual explanation by operator staff |
| Must cover (Code §9.2) | IP Marketing Safety Code handling; bund drainage valve discipline; interceptor observation/skimming/cleaning; pump/hose kerbed-area valves; **contingency plan** (Fire Order + spill plan) |
| Contingency plan contents | Spill size/nature; nearby sensitive receptors (beaches, mariculture, ecology, seawater intakes); org roles; alert/comms; stop/contain/recover/cleanup; disposal of recovered product; equipment list + external resources; training/exercises |
| Drills | Full-scale contingency drill **≤ every 12 months**; familiarisation drill at intervening **6 months** |
| Sludge / oily waste | Chemical waste under Cap **354** — register with DEP; licensed facilities; temporary storage |
| Tank roof drains | Regular inspection / maintenance against blockage |

**SD takeaway:** Layout changes that alter filling, drainage divert routes, fire access, spill response, or jetty receipt points force **ops-instruction amendments** + BA time.

### 11. Annual licence renewal & settlement monitoring (reg. 7)

| Item | Rule |
|---|---|
| Licence life | **12 months** from grant or renewal |
| Renewal fee | **$30,300** (as printed 21-10-2021); BA may add conditions |
| Apply | Specified form, **≥ 2 months before expiry** |

Renewal may be **refused** (BA + Standing Advisory Committee advice) unless:

1. **Settlement records** for **all tanks and foundations** for the preceding 12 months; **and**
2. **RSE statement** of findings + recommendations from examining those records; **and**
3. RSE recommendations under these regulations have been **carried out**

APP-10: renewal application must also be accompanied by **external tank inspection certs** (reg. 8(1)(b), App B form) and **associated works inspection cert** (reg. 8(1)(c), App C form). App C cert is also copied to **Director of Fire Services**.

**SD takeaway:** Soft ground, reclamation, marine fills → lifelong settlement monitoring array + possible tank outage for remedial works. Flag at schematic.

### 12. Mandatory inspection cycle (reg. 8 + Code §10) — design for access & downtime

Licensee must arrange inspections by / under **RSE**:

| Inspection | Timing | Focus |
|---|---|---|
| **Internal general** | Not later than **year 10** of each tank’s life; then **every 5th year** after that first internal | Interior — drain-down / gas-free / confined space |
| **External shell + shell-to-bottom joint** | **Annually** | External shell; bottom-joint zone |
| **Associated works** | **Annually** | Bunds, drains, pipelines, related SWD |

BA may order any of these **earlier** by written notice. Certificates go to licensee then **forthwith to BA**.

**External inspection checklist the RSE must be able to see (Code §10.1)** — design so these are accessible:

- External coating condition
- Lower **300 mm** of shell + weld + bottom-plate edge (do not bury under thick cover that hides corrosion)
- Peripheral base seal (replace defective seals — do not just paint over)
- Perimeter of bottom plate / gaps from settlement
- Insulation (lift where corrosion suspected; weatherproof sheathing)
- Earthing connections
- Foundations including ring walls
- Settlement records / differential settlement warning
- Foundation pad shedding & impervious membrane condition

**Internal inspection (Code §10.2):** ultrasonic thickness (RSE personally selects **≥ 20%** of points + observes enough readings); visual with flameproof floodlighting; sludge as Cap 354 chemical waste.

**After repairs (Code §10.3–10.4):** same QC as new work; **water test** fill to max capacity, hold **48 hours**, investigate any drop, re-test before return to service.

**SD takeaway:** Permanent access platforms, clearances around shell-to-bottom joint, bund walkways, manways. Do not clad or landscape over inspection-critical zones.

### 13. Defective / restricted tanks (reg. 9)

If BA considers a tank unfit or ought not be used:

- Require tank to be **emptied**
- **Prohibit** or **restrict** use under stated conditions
- Give other storage directions for the installation
- On non-compliance: **seize, remove and detain** petroleum products
- Enter (day/night; forcible entry in emergency or after notice) to inspect or seize

**SD takeaway:** Multi-tank farms need redundancy and empty-out / transfer routes so one restricted tank does not crash the facility.

### 14. Repairs, alterations & additions (regs. 10 & 10A + APP-10)

#### 14.1 Authorization before works (reg. 10)

| Situation | Rule |
|---|---|
| Normal | **No** repairs / A&A to installation or associated works without **BA written authorization** |
| Application | In writing; BA may require plans |
| APP-10 submission | **2 copies** of proposals **signed by RSE** |
| Also need Cap 123 s.14 | If repairs are **extensive**, or A&A are **non-exempted building works** |
| Emergency | May proceed without prior authorization **only** to reduce pollution / fire / explosion risk — give BA **written notice ASAP after commencement** |

#### 14.2 Licence endorsement after A&A (reg. 10A)

During licence period, after any alterations / additions:

1. Apply in writing for **endorsement**; **and**
2. **Return the licence** with the application (APP-10: same for new tanks in existing installation)

BA endorses only if works comply with Cap 123 / approved plans; satisfy **reg. 3** + passing **tests**; updated **ops instructions** remain satisfactory.

**Typhoon risk (APP-10 §9):** partially constructed tanks have been damaged in typhoons — temporary bracing / weather protocol during construction and repair.

**SD takeaway:** Adding a tank, moving a bund wall, changing jetty lines, or major drainage diversion = authorization + (often) s.14 + licence endorsement + possibly revised ops instructions. Not “maintenance.”

### 16. Marine pollution equipment (Code §11) — jetty-linked SD schedule

Maintain on site (minimums). Size stores / yards for this kit at schematic.

| Equipment | Minimum |
|---|---|
| **Boom** | Length **≥ 2.5 ×** max vessel length permitted to berth; internationally recognised type; anchorage; desirably connectable to government ‘unicorn’ connectors; ancillaries stocked |
| **Dispersant** (Type III concentrate baseline) | ≤150 000 DWT → **8000 L**; ≤100 000 DWT → **5500 L**; ≤50 000 DWT → **2700 L**; coastal / local barges <10 000 DWT → **1100 L**. Type II: multiply by dilution factor. Must be HK Gov-approved; supplier + user **DEP licences** |
| **Oil skimmer** | One unit; open-sea capable; **≥ 10 m³/h** medium fuel oil; ancillaries for **≥ 18 m³** recovered oil |
| **Sorbent** | **≥ 1000** pads or **≥ 500 m** material; hydrocarbon-only; pad **≥ 450 mm** wide × **≥ 4 mm** thick |
| Maintenance | Per manufacturer; else inspect/maintain **≤ every 6 months** |

### 17. Demolition of OSI (APP-10 §10 + Appendix D) — if clearance / redevelopment is in the brief

Demolition of whole or part must minimise pollution / fire / explosion. Submit **method statement** for BA agreement **before or with** consent / written authorization. Also follow PNAP on demolition public safety (APP-10 cites PNAP 71).

| Topic | Hard practice |
|---|---|
| Environmental | Follow EPD contaminated-land guidance; PNAP on construction nuisance; Air Pollution Control Ordinance asbestos investigation/management if asbestos present |
| Hoarding | Enclose work area against unauthorised access |
| Staged / partial demo | Erect **fire screen** between demolition zone and remaining live facility |
| Gas freeing | Institute of Petroleum Marketing Safety Code precautions; formal **permit-to-work**; all pipes/tanks certified gas-free by competent person (defined training / certification programme; SAC may review adequacy) |
| Sludge / chemical waste | Identify early; dispose under Waste Disposal (Chemical Waste) (General) Regulations |
| Waste water | Avoid ground contamination; keep drainage + oil interceptors **intact and serviceable** until all tanks, pipes, sludge removed |
| Seaboard valves on interceptors | Remain **closed and locked** during demo (except licensed effluent discharge) until tanks/pipes/sludge gone; WPCO licence conditions apply |
| Bund floors / walls | Prefer demolish **after** tanks and pipe spills removed; if not possible, precautions against product escape |
| Live facility during demo | Programme for operating environment; comply with agreed ops instructions at all times |
| Storm drains | Prevent contaminated wash water entering SWD; keep storm-drain plan on site |
| Extreme weather | Precautions during tank demolition |
| Contamination assessment | **Required before work starts** — past activities; inventory of chemicals/hazmat with locations; manufacturing processes; soil sampling plan/methods |
| Post-demo soils | After above-ground structures removed, test soil for **hydrocarbons and lead** per assessment report |

### 18. Standing Advisory Committee (reg. 4)

BA may refer OSI matters to SAC, including:

- Prevention of pollution / fire / explosion in operation
- Remedial works / risk reduction at a given installation
- Regular review of how Cap 123K (and the Code) requirements are achieved

Renewal refusals under reg. 7(4) may follow SAC advice. Expect specialist review on novel tank forms, unusual foundations, or bad settlement history. SAC also oversees Code reviews (Code Foreword).

---

### 19. Offences snapshot (reg. 12) — why these hold points are non-negotiable

Conviction risks include contravening regs 5, 6A(1), 6B, 10(1)/(3), 10A(1); failing required inspections / delivery of certificates; failing emergency leak remedial measures; ignoring BA notices; wilful oil spillage causing / likely causing external pollution or increased fire/explosion risk; knowing misrepresentation in plans / forms / ops instructions. Penalty: fine at **level 6** + imprisonment up to **2 years**.

### 20. Schematic-design checklist (Cap 123K + Code + APP-10)

- [ ] Above-ground tank capacity vs **110 000 L** threshold (single or any in cluster / bund)
- [ ] Contents = petroleum products; lock **Class 1 / 2 / 3** (flash point) and any heating above flash point
- [ ] Total facility capacity vs **7000 m³** (which spacing table applies)
- [ ] Class 1 tanks **> 1000 m³** → floating roof specified to API 650 / Code §2.3
- [ ] Tank-to-tank / boundary / filling / building distances locked (Tables 1–3 + height multiplier if >18 m)
- [ ] FSD layout / appliance access agreed in principle (Code §2.4)
- [ ] Bund containment **105%** (Class 1) or **100%** (Class 2/3) of largest tank; fire-wall compartments if Class 1 capacity exceeds 60k / 120k m³
- [ ] Bund wall/floor materials + membrane triggers (reclamation / water table / coastal) resolved
- [ ] Contaminated vs clean drainage split; **manual** bund discharge; interceptors outside bund; sample points; WPCO path
- [ ] Walkways, external valve operation, weatherproof valve signs
- [ ] Security fence ≥1.8 m, ≥2 exits ≥5 m apart
- [ ] Jetty-linked: boom / dispersant / skimmer / sorbent stores sized; DEP licences flagged
- [ ] Cap 123A / s.14 plan path; RSE (tanks, foundations, associated works); settlement monitoring for annual renewal
- [ ] Access for annual external + year-10 / 5-year internal RSE inspections (esp. shell-to-bottom joint + peripheral seal)
- [ ] Leak detection per tank; static / electrical hazardous-area strategy
- [ ] Programme: OP → licence docs (ops EN+ZH, GA plans, inspection certs) → first fill
- [ ] Contingency plan + annual/6-month drill regime reflected in ops brief
- [ ] A&A = BA authorization + licence endorsement (not site-only works)
- [ ] If demolition in brief: contamination assessment, gas-free competent person, fire screen for staged works, interceptor lock-down sequence

### 21. What Cap 123K does *not* give you (go elsewhere)

| Need | Go to |
|---|---|
| Numerical design standards (bund %, spacing, drainage, marine kit) | **Code of Practice for Oil Storage Installations 1992** + **PNAP APP-10** (summarised above) |
| Ordinary building plan form / consent / OP admin | Cap **123A** |
| General structural / construction performance | Cap **123Q** + APP-53 accepted codes |
| Fire service installations detail | **Director of Fire Services** / FSD codes (outside 1992 Code scope) |
| Dangerous goods licence conditions | Cap **295** — FSD |
