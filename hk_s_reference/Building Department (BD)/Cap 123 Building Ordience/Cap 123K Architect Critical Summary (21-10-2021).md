# Cap 123K — Building (Oil Storage Installations) Regulations
**Architect critical summary for schematic design**  
Consolidated version: 21 October 2021 | Cap. 123 sub. leg. K

> Scope note: Cap 123K is the **licensing, bunded-works, inspection, and A&A-control rulebook for large above-ground oil tanks** — not ordinary underground tanks, drum stores, or petrol filling station canopies below the Cap 123 capacity threshold. Design geometry and construction standards themselves live mainly in the **Code of Practice for Oil Storage Installations 1992** (deemed to satisfy via **PNAP APP-10**). Use this before schematic design whenever the brief includes bulk petroleum storage, tank farms, jetty-linked receipt/issue lines, or A&A to an existing licensed installation.

---

## Regulatory Overview

Cap 123K applies to every **oil storage installation** as defined in Cap 123 s.2 — any **above-ground tank ≥ 110 000 L**, or a **group of tanks in the same cluster / bunded area** where **any one tank ≥ 110 000 L**, storing **petroleum products** — plus its **associated works** (bunded area, drainage, in-bund pipelines, receipt/issue pipelines to related jetties, related surface-water drainage). It imposes a performance duty to minimise **pollution, fire and explosion**, and makes storage **licence-dependent** (cannot operate without a BA licence; A&A need BA authorization and licence endorsement).

## Critical main topics and subtopics

### 1. Does Cap 123K bite? — capacity & location gate (Cap 123 s.2 + PNAP APP-10)

Lock this at concept before committing site layout, bund geometry, or consultant fees.

| Criterion | Threshold / rule | SD implication |
|---|---|---|
| Capacity | Single tank **≥ 110 000 litres**, **or** any tank in a cluster / bunded group **≥ 110 000 litres** | Below threshold → Cap 123K **does not** apply as OSI licensing regime (other FS / EPD / Dangerous Goods rules may still apply) |
| Location | Constructed **above ground level** | Buried / below-ground tanks are outside this definition |
| Contents | **Petroleum products** (crude, feedstock, semi- / wholly refined petroleum that is liquid or solid at ambient T/P — Cap 123 s.2) | Confirm product class early; “chemical” storage ≠ automatically petroleum |
| Cluster rule | Same **cluster or bunded area** (APP-10) | Do not “split” tanks across separate pads to dodge the threshold if they will share one bund / cluster — BA will treat as one installation |

**SD takeaway:** If any above-ground tank will hit **110 000 L**, plan from day one for: Cap 123A building plans + s.14 approval/consent, Cap 123K licence path, **RSE**, bund + drainage as building works, and annual ops compliance cost.

---

### 2. What you must design into the envelope — “associated works” (reg. 2)

Cap 123K treats the tank as inseparable from its safety geometry. At schematic, the following are **part of the installation**, not optional M&E add-ons:

| Element | What it means for layout |
|---|---|
| **Bunded area** | Area **completely surrounded** by bund walls / embankments sized to **retain spillage** from any tank inside |
| Drainage system of the installation | Spill / process drainage coordinated with pollution control — not a normal storm-only design |
| Pipe-lines **within** the bunded area | All in-bund pipe runs are regulated associated works |
| Receipt & issue pipe-lines to **related jetties** | Marine interface is inside the Cap 123K envelope — coordinate jetty works, berth geometry, and coastal setbacks early |
| Related **surface water** drainage | Site SWD must be resolved as part of the installation package |

**SD takeaway:** Site coverage ≠ building footprint alone. Bund walls, embankment slopes, freeboard, interceptor / drain routes, and jetty corridors eat plot area, access, fire brigade routes, and may conflict with OZP / lease setbacks — model them in early massing.

---

### 3. Overarching design duty (reg. 3) — the only construction standard in Cap 123K itself

Every oil storage installation, its associated works, **and any A&A**, must be designed and constructed so as to **minimise the risk of**:

1. **Environmental pollution**
2. **Fire**
3. **Explosion**

arising from or associated with storage of petroleum products.

| Compliance path | Source | Use at SD |
|---|---|---|
| Performance duty | Cap 123K **reg. 3** | Written into concept risk narrative |
| Deemed-to-satisfy | **Code of Practice for Oil Storage Installations 1992** (PNAP APP-10 §3) | Primary technical brief for bund capacity, tank spacing, materials, testing, etc. |
| Plan approval | Cap 123 **s.14(1)** approval + consent before construction starts (APP-10 §2) | No “build then licence” shortcut |

> Cap 123K does **not** prescribe numerical bund volumes, tank spacing, or fire wall ratings in the Regulations text. Pull those from the **1992 Code** (and any later BA-accepted updates) and lock them before schematic freeze.

---

### 4. Licensing gate — cannot store without a licence (regs. 5–6)

#### 4.1 Absolute prohibition (reg. 5)
No person shall store (or cause / permit storage of) petroleum products in an oil storage installation **without a licence**, or otherwise than in accordance with the **terms / conditions** of that licence.

#### 4.2 What BA needs before granting a licence (reg. 6(4)) — design programme checkpoints

BA **shall not** grant a licence unless **all** of the following are true:

| Gate | Requirement | SD / delivery implication |
|---|---|---|
| (a) | Built **in accordance with Cap 123** and **conforms to BA-approved plans** | Normal Cap 123A plan set + s.14 approval; as-built must match |
| (b) | Constructed per **reg. 3** **and tests** show it was so constructed | Budget commissioning / integrity tests; RSE oversight |
| (d) | **Operation instructions** are satisfactory for minimising pollution / fire / explosion | Ops manual is a **design + licensing deliverable**, not an operator afterthought |

Application package (reg. 6(1)): specified form + **one copy of operation instructions**. Fee (reg. 6(3)): **$47,000** (as at this consolidation) — BA may impose terms/conditions.

Per APP-10 practical package for **new** installations:

- Ops instructions: **6 English + 1 Chinese** (Chinese needed for site grades even though licensing copy is English-led)
- General arrangement plans
- **Occupation permit** + certificate of general inspection **before** licence grant

**SD takeaway:** Treat **OP → licence → first fill** as a critical path, not OP → handover. Client cannot legally put product in the tanks on OP day alone.

---

### 5. Operation instructions — living controlled document (regs. 6A, 6B)

| Rule | Detail |
|---|---|
| Amend only with BA written authorization | reg. 6A(1) |
| Amended ops instructions must still minimise pollution / fire / explosion | reg. 6A(2) |
| Install must be **operated** to the **latest BA-accepted** version | reg. 6B |

**SD takeaway:** Layout changes that alter filling, drainage divert routes, fire fighting access, spill response, or jetty receipt points force **ops-instruction amendments** — price BA authorization time into A&A programmes.

---

### 6. Annual licence renewal & settlement monitoring (reg. 7) — foundation / geotech drivers

| Item | Rule |
|---|---|
| Licence life | **12 months** from grant or renewal |
| Renewal fee | **$30,300** (as at this consolidation); BA may add terms/conditions |
| Apply | Specified form, **≥ 2 months before expiry**, with documents below |

Renewal may be **refused** (BA + Standing Advisory Committee advice) unless:

1. **Settlement records** for **all tanks and foundations** for the preceding 12 months; **and**
2. **RSE statement** of findings + recommendations from examining those records; **and**
3. RSE recommendations under these regulations have been **carried out**

**SD takeaway:** Soft ground, reclamation, marine fills, and piled vs raft foundations are not “structural only” — they drive **lifelong settlement monitoring**, licensability, and possible tank outage for remedial works. Flag geotech risk and monitoring array at schematic.

---

### 7. Mandatory inspection cycle (reg. 8) — design for access & downtime

Licensee must have inspections by / under **RSE** supervision:

| Inspection | Timing | What is looked at |
|---|---|---|
| **Internal general** | Not later than **year 10** of each tank’s life; then **every 5th year** after that first internal | Tank interior — expect drain-down / gas-free / confined-space access |
| **External shell + shell-to-bottom joint** | **Annually** | External shell; critical bottom-joint zone — keep clear for inspection |
| **Associated works** | **Annually** | Bunds, drains, pipelines, related SWD |

Outputs: RSE issues BA-format **certificate of inspection** (results + repair recommendations / fitness for continued use); licensee must **deliver it forthwith to BA**. BA may also order an inspection **at any earlier time** by written notice.

**SD takeaway:** Provide permanently usable access for RSE (platforms, manways, clearances around shell-to-bottom joint, bund walkways). Do not bury critical joints or block external shell with permanent cladding / landscaping that cannot be removed yearly.

---

### 8. Defective / restricted tanks (reg. 9) — contingency in layout

If BA considers a tank unfit or ought not be used for petroleum storage, BA may by notice:

- Require the tank to be **emptied**
- **Prohibit** continued use, or **restrict** use under stated conditions
- Give other directions for storage in the installation
- On failure to comply: **seize, remove and detain** petroleum products
- Enter (including forcible entry in emergency / after notice) to inspect or seize product

**SD takeaway:** Multi-tank farms need operational redundancy and empty-out / transfer routes so one BA-restricted tank does not crash the whole facility.

---

### 9. Repairs, alterations & additions (regs. 10 & 10A) — A&A hold points

#### 9.1 Authorization before works (reg. 10)
| Situation | Rule |
|---|---|
| Normal | **No** repairs / alterations / additions to the installation or associated works without **BA written authorization** |
| Application | In writing; BA may require **plans** of proposed works |
| Emergency | Licensee may proceed **without** prior authorization **only** if works are necessary to reduce pollution / fire / explosion risk — and must give BA **written notice as soon as possible after commencement** |

APP-10 practice: submit **2 copies of proposals signed by RSE** for written authorization. If repairs are extensive, or A&A are **non-exempted building works**, also obtain Cap 123 **s.14(1)** approval — normal BO procedures apply.

#### 9.2 Licence endorsement after A&A (reg. 10A)
During a licence period, after any alterations / additions, licensee must:

1. Apply in writing for **endorsement** on the licence; **and**
2. **Return the licence** with the application

BA endorses only if:

- Works comply with Cap 123 and (where approved) with approved plans
- Works satisfy **reg. 3** and passing **tests**
- Updated **operation instructions** remain satisfactory for pollution / fire / explosion control

**SD takeaway:** Adding a tank, moving a bund wall, changing jetty lines, or major drainage diversion = **authorization + (often) s.14 plans + licence endorsement + possibly revised ops instructions**. Do not treat as “maintenance.”

---

### 10. Leak response (reg. 11) — operational design interface

If the licensee has reason to believe a tank is leaking:

| Case | Duty |
|---|---|
| Non-emergency | Report circumstances to BA **in writing** |
| Emergency | Take **suitable remedial measures forthwith** to prevent oil spillage and reduce pollution / fire / explosion risk, **and immediately** report in writing to BA |

**SD takeaway:** Schematic must include spill detection / containment / emergency isolation logic consistent with the ops instructions (bund integrity, shut-off valves, drains that can be isolated from public sewers / sea).

---

### 11. Standing Advisory Committee (reg. 4) — when BA escalates technical questions

BA may refer OSI matters to a Standing Advisory Committee, including:

- Prevention of pollution / fire / explosion in operation
- Remedial works / measures to reduce risk at a given installation
- Regular review of how Cap 123K requirements are achieved

Renewal refusals under reg. 7(4) may follow SAC advice. Expect specialist review on novel tank forms, unusual foundations, or unsatisfactory settlement history.

---

### 12. Construction / demolition interfaces (outside Cap 123K text but SD-critical — APP-10)

| Topic | Practical rule |
|---|---|
| Start of construction | Cap 123 **s.14(1)** approval + consent first |
| Typhoon during build / repair | Partial tanks are vulnerable — design temporary bracing / weather protocol (APP-10 §9) |
| Demolition of whole / part | Method must minimise pollution / fire / explosion; demolition method statement to BA **before or with** consent / authorization; follow APP-10 Appendix D + PNAP on demolition public safety |
| Partial demolition / staged demo | Fire screen between working part and remaining live tanks |
| Agency coordination | EMSD (gas / related) and **EPD** (pollution) routinely in the loop (see ADM notes) |

### 13. Schematic-design checklist (Cap 123K)

Use as a freeze checklist before Concept / SD submission:

- [ ] Confirm above-ground tank capacity vs **110 000 L** threshold (single or any in cluster / bund)
- [ ] Confirm contents = **petroleum products** under Cap 123 s.2
- [ ] Layout includes **full bunded area** (complete surround) + spill drainage + in-bund pipes
- [ ] Jetty / receipt–issue corridors shown if marine-linked
- [ ] Design narrative addresses **pollution + fire + explosion** (reg. 3); target **1992 Code** compliance path
- [ ] Cap 123A / s.14 plan path budgeted; RSE appointed for tanks, foundations, associated works
- [ ] Geotech / settlement monitoring strategy suitable for **annual licence renewal**
- [ ] Access for **annual external** and **year-10 / 5-year internal** RSE inspections
- [ ] Programme: OP → licence docs (ops instructions EN+ZH, GA plans, inspection certs) → first fill
- [ ] A&A strategy assumes BA authorization + licence endorsement (not site-only works)
- [ ] Multi-tank redundancy if BA can empty / seize one defective tank

---

### 14. What Cap 123K does *not* give you (go elsewhere)

| Need | Go to |
|---|---|
| Numerical design standards (bund volume, spacing, materials, tests) | **Code of Practice for Oil Storage Installations 1992** + **PNAP APP-10** |
| Ordinary building plan form / consent / OP admin | Cap **123A** |
| Structural / construction performance (general) | Cap **123Q** + APP-53 codes |
| Dangerous goods / petrol filling station specifics (often under threshold) | Cap **295** / FSD codes — confirm separately |
| Planning / land-use permission for tank farm | OZP + lease + PlanD / LandsD |
