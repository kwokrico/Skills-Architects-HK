# Cap 123N — Building (Minor Works) Regulation
**Architect critical summary for schematic design**  
Consolidated version: 31 March 2022 | Cap. 123 sub. leg. N

> Scope note: Cap 123N is the **Minor Works Control System (MWCS) rulebook** — the alternate path when small building works need **not** go through full Cap 123 s.14 plan approval + consent. It defines **what is minor works** (Sch. 1), what is **designated exempted** (Sch. 2 — no notice/approval), who must be appointed, simplified submission packs, and Cap 123 **s.39C** validation of certain unauthorized works (Sch. 3). It is **not** the Building (Administration) plan package (Cap 123A), not volume/GFA control (Cap 123F), and not construction performance (Cap 123Q).

---

## Regulatory Overview

Cap 123N designates Class I / II / III minor works (and Types A–H by trade) for the purposes of Cap 123, and prescribes **simplified requirements** (Part 6) plus **designated exempted works** under Cap 123 s.41(3B). It applies whenever schematic design proposes A&A, signage, drainage, amenity structures, finishes, demolition of small unauthorized features, ventilation ducts, or similar works that may fall inside Schedule 1 dimensional ladders instead of a full BD submission.

## Critical main topics and subtopics

### 1. First SD decision — pick the legal path by size / risk

At concept, sort every proposed intervention into **one** of four buckets. Crossing a Schedule number moves you up a class or out of MWCS entirely.

| Bucket | Legal effect | Who / notice | When to use |
|---|---|---|---|
| **Designated exempted** (Sch. 2) | Cap 123 s.14(1) **does not apply** (s.41(3B)) | No AP / MW notice required for the exemption itself | Tiny low-risk works under Sch. 2 hard numbers |
| **Class III MW** | Simplified requirements; **post-completion** notice only | Registered contractor (item / Type) | Small works within Class III ceilings |
| **Class II MW** | **≥7 days** pre-start notice + plans by contractor | Registered contractor | Mid-band dimensional works |
| **Class I MW** | **≥7 days** pre-start notice + AP (or RI for prescribed repair) + RSE/RGE if triggered + plans | AP/RI + RSE/RGE as needed + registered contractor | Higher-risk / larger MW items |
| **Outside Sch. 1 / Sch. 2** | Full Cap 123 + Cap 123A approval / consent | AP / RSE / RGC as normal | Exceeds MW ceilings, loads cantilevered slabs illegally, alters protected structural systems, MOE stairs, etc. |

**Universal kill-switches (almost every MW item):**
- Do **not** put **additional load on any cantilevered slab** (unless the item expressly allows a limited case)
- Do **not** alter structural elements beyond the narrow “simply supported, non-prestressed beam not supporting column / flat slab / ribbed beam” carve-out
- MOE / firefighting stairs and protected lobby openings are **not** ordinary staircase items — openings to those enclosures are Class I (item **1.42**)

> SD takeaway: freeze projection depths, display areas, wall heights, excavation depths, AC plant weights, and “on cantilever vs not” before choosing MW vs full BP.

### 2. Who must be appointed (ss. 27–28) — lock the team

#### 2.1 Class I — prescribed building professionals (s. 27)
| Role | Required when |
|---|---|
| **Authorized Person** | Always for Class I (or **Registered Inspector** acting as AP for prescribed repair / associated demolition) |
| **Registered Structural Engineer** | Works involve **structural elements** |
| **Registered Geotechnical Engineer** | Works involve **geotechnical elements** |

**AP may sign Class I foundation / structural docs alone only if ALL of s.37(4) apply** (mirrors Cap 123A small-works logic):
- Construction height ≤ **10 m**
- Any structural span ≤ **6 m**
- Materials: timber / masonry / steel / plain or RC only
- Spread footings ≤ **300 kPa**, founded ≤ **2 m** below existing ground
- No alteration of existing RSE-designed structural elements

#### 2.2 All classes — prescribed registered contractor (s. 28)
One of: **RGBC**; **RMWC** registered for the Type/item; or specialist contractor when the item demands it:

| Specialist category | Trigger items (examples) |
|---|---|
| **Demolition** | 1.2, 1.5, 1.9, 1.10, 1.24, 1.30, 1.32–1.34, 1.37–1.40 + listed Class II/III demo / removal counterparts |
| **Site formation** | 1.11, 1.12, 2.10, 2.11, 3.53 |
| **Foundation** | 1.11, 1.12, 2.10, 2.11 |
| **Ground investigation field works** | 1.12, 2.11 |
| **Ventilation works** | 1.46, 1.51, 2.42, 2.46 |

**Class III natural-person RMWC** (registered under s.11) must **personally** carry out the works (s.44).

### 3. Simplified submission clocks (Part 6) — programme hold points

| Class | Before start | On completion (≤ **14 days**) |
|---|---|---|
| **I** | ≥ **7 days**: Form + appointments + load/stress capacity cert (A&A) + photos + **prescribed plans** + supervision plan if TM requires | Revised plans if as-built differ; photos; structural (and geo) safety certs. Demo: remaining structure / affected land or street adequate MoS |
| **II** | ≥ **7 days**: Form + contractor cert of capacity (A&A) + photos + **prescribed plans** | Same pattern; contractor certifies structural safety / MoS for demo |
| **III** | **No** pre-start BD notice | Form + as-completed plans/description + before/after photos |

**Signboard extra:** particulars of the person for whom the signboard is erected — mandatory in Class I/II/III notices involving erection.

**Plan hygiene (ss. 37–39):**
- Scales ≥ **1:100** (block plan ≥ **1:500**)
- Colour to differentiate **existing vs new** and parts of new work
- Clear / durable hard copy if paper

### 4. Types A–H (Sch. 1 Part 2) — match contractor registration to the work

| Type | Focus | Typical SD topics |
|---|---|---|
| **A** | Alteration & Addition | Walls, openings, stairs, A&A fabric |
| **B** | Repair | Structural / wall / slope repairs |
| **C** | Signboards | Projecting / wall / roof / on-grade signs |
| **D** | Drainage | Above- & underground drains |
| **E** | Amenities | Canopies, AC frames, racks, ducts, water tanks, poles, fences |
| **F** | Finishes | Cladding, rendering, roof finishes, dowelled panels |
| **G** | Demolition / removal | Removal counterparts of the above |
| **H** | Ventilation inside building | Large metal ducts / suspended plant frames (Class I/II) |

One physical package can span Types — register / appoint against **every** Type/item touched.

### 5. Universal definitions that change numbers (Sch. 1 Part 1)

| Term | Design meaning |
|---|---|
| **Projecting signboard** | Fixed to external wall **and** projects **> 600 mm** |
| **Wall signboard** | Fixed to external wall; **no part** projects **> 600 mm** |
| **Display area** | Largest planar face of virtual rectangular prism containing the board (**exclude** slim single post Ø ≤ 100 mm, or lateral-restraint-only members) |
| **Thickness (signboard)** | Smallest prism dimension of board + support (same exclusions) |
| **Flat** | Separate unit as delineated on approved plan (domestic or non-domestic) |
| **Aggregate length of additional wall** | Increase in total non-loadbearing wall length (exclude door widths) vs approved plan — flat or roof |
| **Building services installation** | Includes solar WH, PV, antenna, transceiver, AC unit, cooling tower, light fitting, pump set + ducts; **excludes** water tank, lift, stairlift, lifting platform, ventilation duct, radio base station |
| **Inaccessible roof** | Not for occupation; access only for M&R |
| Make-good / replace / strengthen / UBW removal | Built into item descriptions (Sch. 1 §§2–6) — removal items include **unauthorized** equivalents; alteration includes Cap 123 s.39C(4) rectification |

### 6. Master dimensional ladders (design hard numbers)

Use these as schematic filters. Values are ceilings for remaining **inside** that Class/item — exceed → next Class or full BP.

#### 6.1 Slab openings & reinstatement
| Work | Class / item | Hard numbers |
|---|---|---|
| Form opening in slab | **II 2.1** | Area ≤ **1 m²** |
| Form opening in slab | **I 1.2** | Area **> 1 m² and ≤ 4.5 m²** |
| Reinstatement | **I 1.35** / **II 2.35** | Same area bands / small openings ≤ **150 mm** farthest points for tiny cases |
| Outside | Full BP | > **4.5 m²**, or prestressed / transfer / column-support beam altered |

#### 6.2 Internal stairs / lifts (not MOE)
| Work | Class / item | Hard numbers |
|---|---|---|
| Erect / alter internal stair (not MOE / firefighting) | **I 1.1** | Structural carve-out only |
| Remove whole lowest-storey internal stair | **III 3.1** | Height ≤ **1.5 m** |
| Remove internal stair | **I 1.32** | Larger / general removal |

| Work | Class II | Class I | Exceeds MW |
|---|---|---|---|

### 8. Cap 123 s.39C validation pathway (Part 9 + Sch. 3)

Use when the site already has **unauthorized** features that match prescribed types — validation by inspection / certification rather than pretend they are legal MW from day one.

| Route | Covered structures | Inspector | Cut-off dates (s.62A) |
|---|---|---|---|
| **s.39C(1)** / Sch.3 Part 2 | Unauthorized AC support ≤ **100 kg**; AC wall frame ≤ **600 mm** / ≤ **100 kg**; drying rack ≤ **750 mm**; non-concrete canopy ≤ **500 mm** (with street/common rules) | AP / RSE / RI / RGBC / RMWC (Types A or E under I–III, or listed Class III amenity items) | — |
| **s.39C(1A)** / Sch.3 Part 3 | Batch 1 unauthorized signboards matching Class I–III sign items | Class I “specified” → AP; other Class I → AP+RSE; Class II/III → AP/RSE/RI/RGBC/RMWC for item/Type | Structures existing before **2 Sep 2013** |
| **s.39C(1A)** / Sch.3 Part 4 | Batch 2 (2020 expansion — further prescribed UBW types) | Same pattern | Before **1 Sep 2020** |

Submit inspection notice + photos/description within **14 days** of inspection (ss.62 / 62A). Strengthening / alteration / removal of matching unauthorized items is folded into related Sch.1 MW items (Sch.1 §§5–6; Class III **3.34–3.38**).

### 9. Supervision & site duties under simplified requirements (Part 7)

- Applies **only** to works commenced under simplified requirements (s.40)
- AP/RI / RSE / RGE: periodical supervision — no material divergence from submitted plans (s.42)
- Contractor (except personal Class III RMWC): **continuous** supervision; keep records **≥ 12 months** after completion (s.43)
- Class I: TCP appointments per Technical Memorandum; notify BA ≥ **7 days** before start; works **stop** if TCP appointment fails (ss.45–47)
- Keep prescribed plans / supervision plans on site (s.57)

### 10. Schematic design checklist (use before freezing the package)

1. **List every intervention** (wall, opening, projection, plant, drain, sign, dig, fence).
2. **Tag location**: cantilevered slab? MOE stair? inaccessible roof? street / common part projection?
3. **Measure against Sch. 2** → if fully inside, book as designated exempted.
4. Else measure against **Class III → II → I** ladders in §6; stop at first matching item.
5. If any kill-switch fails (extra cantilever load, protected structure, size overrun) → **full Cap 123A** path.
6. Appoint **AP/RSE/RGE/specialist/RMWC** per §§2–3; align contractor **Types A–H**.
7. Put **≥7-day** Class I/II notices and plan production on the programme; Class III = completion pack only.
8. For existing UBW amenity / signs → test **Sch. 3 / s.39C** before assuming illegal demolition or full BP.
9. Still verify **lease, OZP Notes, FS Code, PNAP, Cap 123F GFA**, and adjoining-owner interfaces — Cap 123N never waives those (s.60–61).
