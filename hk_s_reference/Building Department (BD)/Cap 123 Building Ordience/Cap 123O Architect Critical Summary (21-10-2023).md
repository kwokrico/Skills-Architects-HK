# Cap 123O — Building (Minor Works) (Fees) Regulation
**Architect critical summary for schematic design**  
Consolidated version: 21 October 2023 | Cap. 123 sub. leg. O

> Scope note: Cap 123O is the **RMWC registration fee schedule** — fixed statutory amounts payable under Cap 123N for application / grant / renewal / restoration / add items or classes-types / add authorized signatory (AS) / review / provisional registration. It is **not** the MWCS design rulebook (Cap 123N + Sch. 1–3), not Cap 123A plan-approval fees, and not contractor tender rates. Use this when the scheme intends a **minor-works path** and you need to (a) specify the exact registration scope the appointed contractor must hold, and (b) understand how class / AS roster economics affect who can bid.

---

## Regulatory Overview

Cap 123O prescribes the fees payable under Cap 123N for registration (and related applications) of **registered minor works contractors**. It splits into three fee regimes: **Part 2** — natural-person registration for **one or more Class III items**; **Part 3** — registration for **one or more types under one or more classes** (typical firm RMWC); **Part 4** — **provisional** registration. Fees scale by the **highest class** (I > II > III under s. 2(3)) and by the number / class-band of **proposed authorized signatories**. There are no occupancy-type or building-geometry controls in this Cap — every dollar here is paperwork for the contractor register under Cap 123 s. 8A(1)(c).

## Critical main topics and subtopics

### 1. When Cap 123O bites at schematic design

| Situation | Cap 123O relevant? | SD / procurement implication |
|---|---|---|
| Scheme uses Cap 123N MWCS (Class I / II / III) | **Yes** — appointed contractor must be correctly registered | Lock MW class + Type / item early; fee + AS stack shape the appointable pool |
| Full Cap 123 s.14 BP + Cap 123A consent | **No** | Cap 123O does **not** price BD plan approval or consent |
| Client / specialist wants to **become** an RMWC for the works | **Yes** | Budget: application fee + (Part 3 / 4) **grant-of-registration fee** + any extra-AS add-ons — and programme for BA / MWCRC processing |
| Spec only says “engage RMWC” in prelims | Indirect but risky | Still specify **class + type(s) / item number(s)**; vague wording invites wrong registration track |
| Named RMWC’s registration is about to expire / was removed | **Yes** (renewal / restoration fees) | Check expiry **before** award; Class I restoration is the expensive path |
| Works need a second AS or a new Type mid-project | **Yes** (ss. 14–16) | Mid-project add-AS / add-type fees + processing time can stall Class I notices |
| Contractor is on the **provisional** register only | **Yes** (Part 4) | Different (lower) fee table; confirm Cap 123N provisional status is still valid for the works |
| Designated exempted works only (Cap 123N Sch. 2) | **No** | No RMWC registration trigger for the exemption itself |

**Hierarchy rule (s. 2(3)) — drives every Part 3 / Part 4 “highest class” fee band:**
- Class I is higher than Class II and Class III
- Class II is higher than Class III  
→ Fee is set by the **highest class the application covers**, not by how many Type A–H items you pile under a lower class, and not by the number of Class III items alone.

> SD takeaway: Crossing Class III → II → I is a **registration-cost cliff** for the contractor, on top of Cap 123N’s notice / AP / RSE appointment cliff.

### 2. Definitions that control which fee you pay (s. 2) — do not skip

Cap 123O fees only make sense if you classify people and works the way the Cap defines them.

#### 2.1 Registers and persons

| Term | Meaning (s. 2) | Why it matters |
|---|---|---|
| **register** | Register of minor works contractors under Cap 123 **s. 8A(1)(c)** | Ordinary RMWC — Parts 2 & 3 |
| **provisional register** | Provisional register under Cap 123 **s. 8A(1)(c)** | Part 4 fees only |
| **registered minor works contractor** | Name currently on the **register** | Must be current — expired / removed ≠ registered |
| **registered minor works contractor (provisional)** | Name currently on the **provisional register** | Do not use Part 3 rates for these persons |
| **item** | Same meaning as Cap 123N | Part 2 is **item**-based (Class III Sch. 1 Div. 3 item numbers) |
| **type** | Same meaning as Cap 123N (Types A–H) | Part 3 / 4 is **type**-under-**class** based |
| **authorized signatory** | Cap 123N s. 2(1) meaning | The individual who can sign for the firm on MW submissions |
| **Minor Works Regulation** | Cap 123N | Every Cap 123O fee is pegged to a Cap 123N payment duty |

#### 2.2 “Proposed authorized signatory” — five Cap 123N gateways (s. 2(1))

Who counts as a **proposed AS** (and therefore can trigger per-AS fees) depends on which Cap 123N application is running:

| Cap 123N application | Proposed AS = |
|---|---|
| Registration under **s. 10(1)(b)** or provisional **s. 64(1)** | Individual nominated under Cap 123N **s. 10(3)** / **s. 64(3)** |
| Renewal under **s. 14(1)** | Individual named as AS on the renewal form |
| Restoration under **s. 18(1)** | Individual named as AS on the restoration form |
| Add class(es) / type(s) under **s. 21(2)** | Individual nominated under Cap 123N **s. 21(4)** |
| Approve additional AS under **s. 24(1)** | That additional individual |

#### 2.3 Class-band of a proposed AS (s. 2(1)) — fee classification trap

| Label | Who qualifies | Fee consequence |
|---|---|---|
| **Proposed class I AS** | Nominated in respect of **Class I** (even if also nominated for II / III) | Always charged at **Class I** AS rates |
| **Proposed class II AS** | Nominated for **Class II**, or **Class II + III** (but **not** Class I) | Class II AS rates |
| **Proposed class III AS** | Nominated for **Class III only** | Class III AS rates |

> Critical: once an individual is tagged for Class I on the form, they are a **class I AS** for fee purposes even if most of their day-to-day work is Class III. Do not “under-nominate” Class I coverage to save fees if the works actually need Class I signing capacity — Cap 123N competence / appointment rules still control.

#### 2.4 What “class I / II / III minor works” means here (s. 2(2))

| Cap 123O term | Cap 123N source |
|---|---|
| Class I minor works | Any item in **Sch. 1 Part 3 Division 1** |
| Class II minor works | Any item in **Sch. 1 Part 3 Division 2** |
| Class III minor works | Any item in **Sch. 1 Part 3 Division 3** |

### 3. Three registration tracks — pick the fee table before you pick the contractor

| Track | Who | Cap 123N gateway | Cap 123O Part | Has separate “fee on grant”? |
|---|---|---|---|---|
| **Item-based Class III** | Usually natural person / craftsman | Register **s. 10(1)(a)** → Cap 123N **s. 11**; renew / restore / add **items** | **Part 2** (ss. 3–8) | **No** — application fees only in Cap 123O |
| **Type(s) under class(es)** | Typical company RMWC | Register **s. 10(1)(b)** → Cap 123N **s. 12**; renew / restore / add **classes or types** / add AS | **Part 3** (ss. 9–17) | **Yes** — s. 11 flat **$595** on Cap 123N s. 12(7) registration |
| **Provisional** | Provisional RMWC | Cap 123N **ss. 64–65** | **Part 4** (ss. 18–21) | **Yes** — s. 20 flat **$315** on Cap 123N s. 65(5) |

**Do not mix tables.**  
- Part 2 = fees for **items of Class III** (experience surcharge possible).  
- Part 3 = fees for **types under classes**, highest-class banded, AS-stackable.  
- Part 4 = provisional only.

**Restoration path split (easy to get wrong):**
- Removed after registration under Cap 123N **s. 11** → Cap 123O **s. 6** (Part 2, $105)
- Removed after registration under Cap 123N **s. 12** → Cap 123O **s. 13** (Part 3, class-banded $435–$1,300)

Same Cap 123N restoration section (s. 18) — different Cap 123O fee because the **original registration route** differed.

### 4. Full Cap 123N → Cap 123O fee bridge

Use this when a contractor (or client) asks “what do we pay BD for?”

#### 4.1 Part 2 bridge — Class III **items**

| Cap 123N trigger | Cap 123O fee section | What is paid |
|---|---|---|
| Application under **s. 10(1)(a)** — pay under **s. 10(2)(b)** | **s. 4** | $290 or $155 (see §5) |
| Renewal under **s. 14(1)** — pay under **s. 14(2)(b)** (person registered under s. 11) | **s. 5** | $105 |
| Restoration under **s. 18(1)** — pay under **s. 18(2)(b)** (was s. 11; removed under s. 17(1)) | **s. 6** | $105 |
| Add items under **s. 21(1)** — pay under **s. 21(3)(b)** | **s. 7** | $290 or $155 |
| Review under **s. 26(1)** — pay under **s. 26(2)(b)** (of the above applications) | **s. 8** | $110; refund if BA substitutes under Cap 123N **s. 26(8)(a)(ii)** |

#### 4.2 Part 3 bridge — **types under classes**

| Cap 123N trigger | Cap 123O fee section | What is paid |
|---|---|---|
| Application under **s. 10(1)(b)** — pay under **s. 10(2)(b)** | **s. 10** | Highest-class base + extra-AS stack |
| Registration granted under **s. 12(7)** | **s. 11** | Flat $595 (all classes) |
| Renewal under **s. 14(1)** — person registered under **s. 12** | **s. 12** | Highest-class renewal |
| Restoration under **s. 18(1)** — was **s. 12**; removed under **s. 17(1)** | **s. 13** | Highest-class restoration |
| Add class(es)/type(s) under **s. 21(2)** — pay under **s. 21(3)(b)** | **s. 14** (or **s. 16** if combined) | Aggregate per proposed AS |
| Add AS under **s. 24(1)** — pay under **s. 24(2)(b)** | **s. 15** (or **s. 16** if combined) | Per additional individual by class band |
| Review under **s. 26(1)** of Part 3 applications (not the s. 12(7) grant itself — see Cap 123O s. 9(6)) | **s. 17** | Aggregate per involved proposed AS |

**Note on s. 9(6):** Part 3 review (s. 17) covers Cap 123O s. 9(1), (3), (4), (5) applications — i.e. initial application, renewal, restoration, add class/type, add AS. It does **not** list the bare “grant under s. 12(7)” event (that is a fee on registration, not a reviewable application fee under s. 17’s bridge).

#### 4.3 Part 4 bridge — provisional

| Cap 123N trigger | Cap 123O fee section | What is paid |
|---|---|---|
| Application under **s. 64(1)** — pay under **s. 64(2)(b)** | **s. 19** | Highest-class base + extra-AS stack |
| Registration granted under **s. 65(5)** | **s. 20** | Flat $315 |
| Review under **s. 26(1)** of the provisional application | **s. 21** | Aggregate per involved proposed AS |

### 5. Part 2 — Class III **item** fees (HK$) — full rules

Applies only when Cap 123N registration is for **one or more Class III items** under s. 10(1)(a) / related item applications — **not** firm multi-type Class I/II.

| Action | Cap 123O | Fee | Hard rule |
|---|---|---|---|
| Application for registration | s. 4 | **$290** or **$155** | **$290** if for **at least one** item the applicant relies **solely on the applicant’s own experience** to meet Cap 123N **s. 11(2)(a)**; otherwise **$155** |
| Renewal | s. 5 | **$105** | Flat — no experience split |
| Restoration of name | s. 6 | **$105** | Flat — only for persons who were registered under Cap 123N **s. 11** |
| Add additional Class III items | s. 7 | **$290** or **$155** | Same experience test, but against Cap 123N **s. 22(2)(a)** |
| Review (decision **or** MWCRC recommendation) | s. 8 | **$110** | Refund **mandatory** if BA substitutes the decision under Cap 123N **s. 26(8)(a)(ii)** |

**Experience surcharge detail:**
- Trigger is **sole reliance on own experience** for compliance — not “has experience and also has a qualification.”
- Trigger fires if **any one** item in the batch uses the experience-only route → whole application / add-items fee jumps to $290.
- Renewal / restoration do **not** carry this surcharge.

**Part 2 vs Part 3 structural gap for cost plans:**  
Part 2 Cap 123O has **no** separate “fee for registration on grant” equivalent to Part 3 s. 11. Budget Part 2 as application (± review) only, unless BD / Cap 123N practice notes impose other charges outside this Cap.

**SD takeaway:** For cheap Class III scopes, confirm the appointed person is already registered for the **exact Cap 123N Sch. 1 Div. 3 item numbers** — mid-project “register extra items” costs $155/$290 **plus** BA processing time, and Class III natural-person RMWCs must personally carry out the works (Cap 123N s. 44).

### 6. Part 3 — multi-type / multi-class fees (HK$) — main SD cost map

#### 6.1 Application to register (s. 10) — base by highest class + extra AS

| Highest class in application | Base application fee (s. 10(1)) |
|---|---|
| Class I | **$2,920** |
| Class II | **$1,100** |
| Class III types only | **$730** |

**Extra-AS stacking (s. 10(2)–(3)):**  
If **2 or more** proposed AS are nominated on the form → pay base **plus** the amount below for **each** proposed AS **other than the first one nominated for the highest class** of the application:

| That extra person’s AS class band | Add-on |
|---|---|
| Proposed class I AS | **$2,440** |
| Proposed class II AS | **$810** |
| Proposed class III AS | **$560** |

**Worked examples (application fee only):**

| Nomination pattern | Calculation | Total |
|---|---|---|
| Class I works; **1** Class I AS | Base only | **$2,920** |
| Class I works; **2** Class I AS | $2,920 + $2,440 | **$5,360** |
| Class I works; 1 Class I AS + 1 Class III AS | $2,920 + $560 | **$3,480** |
| Class II works; **1** Class II AS | Base only | **$1,100** |
| Class II works; 2 Class II AS | $1,100 + $810 | **$1,910** |
| Class III types only; 1 Class III AS | Base only | **$730** |
| Class III types only; 3 Class III AS | $730 + $560 + $560 | **$1,850** |

> “First one nominated for the highest class” is the free AS slot inside the base fee. A second Class I AS is never free. A Class III AS sitting beside a Class I AS is an **extra** and always attracts the Class III add-on.

#### 6.2 Fee on grant of registration (s. 11) — second cheque

When Cap 123N **s. 12(7)** registers the person:

| Highest class for which registered | Fee |
|---|---|
| Class I | **$595** |
| Class II | **$595** |
| Class III only | **$595** |

Flat across classes — class scaling applies to **application / renewal / restoration / AS**, not to this grant fee.

> Budget **application + $595 grant** as a pair for any new Part 3 registration. Application fee alone understates true BD statutory exposure.

#### 6.3 Renewal / restoration (ss. 12–13) — by highest class

| Highest class | Renewal (s. 12) | Restoration (s. 13) | Δ (restore − renew) |
|---|---|---|---|
| Class I | **$1,130** | **$1,300** | +$170 |
| Class II | **$835** | **$915** | +$80 |
| Class III only | **$425** | **$435** | +$10 |

**SD takeaway:** Letting a Class I registration lapse is not catastrophic in fee terms versus timely renewal (+$170), but Cap 123N removal / restoration process time and eligibility are the real programme risk — check expiry **before** awarding MW packages that depend on a named firm.

#### 6.4 Add classes / types (s. 14) — aggregate per proposed AS

Subject to the combined-fee rule in s. 16, fee for Cap 123N **s. 21(2)** add-class/type application =

**sum**, for **each** proposed AS nominated on that form, of:

| Proposed AS class band | Amount each (s. 14(2)) |
|---|---|
| Class I | **$2,700** |
| Class II | **$965** |
| Class III | **$615** |

Unlike s. 10, there is **no free first AS** here — every nominated AS on an add-class/type form is charged.

| Example | Fee |
|---|---|
| Add Type; 1 Class I AS | **$2,700** |
| Add Type; 2 Class I AS | **$5,400** |
| Add Type; 1 Class I AS + 1 Class II AS | **$2,700 + $965 = $3,665** |

#### 6.5 Approve nomination of additional AS (s. 15)

Subject to s. 16, fee for Cap 123N **s. 24(1)** =

| Individual’s AS class band | Fee |
|---|---|
| Proposed class I AS | **$2,700** |
| Proposed class II AS | **$965** |
| Proposed class III AS | **$615** |

Same dollar amounts as s. 14(2) — but charged **per additional individual** under s. 24, not as an aggregate over a multi-AS add-type form (unless you use s. 16 combined calculation).

#### 6.6 Combined fee when add-class/type **and** add-AS are filed together (s. 16) — conditions are strict

Cap 123O calculates **one combined fee** for Cap 123N s. 21(2) + s. 24(1) **only if both**:

1. The s. 21(2) application form **indicates** that the applicant is also making a s. 24(1) application; **and**
2. The **two application forms are submitted together**.

Miss either condition → separate s. 14 + s. 15 fees (risk of double-counting the same person).

**Combined calculation (s. 16(2)–(3)) — charge once per individual:**

| That individual’s role across the two forms | Amount |
|---|---|
| AS for **add-class/type only** (s. 21(2)) | s. 14(2) amount for their class band |
| AS for **add-AS only** (s. 24(1)) | s. 15 amount for their class band |
| AS for **both** applications | **whichever of s. 14(2) or s. 15 is higher** |

At current prescribed rates, s. 14(2) and s. 15 are **identical** by class band → dual-role nominee pays **once** at that band (no double charge).

**Procurement takeaway:** When a firm needs both a new Type and a new AS for your Class I package, instruct them to **link and co-submit** under s. 16 — cleaner process and fee math already equalizes dual-role nominees.

#### 6.7 Review fees — multi-type track (s. 17)

Review fee under Cap 123N s. 26(2)(b) =

**sum**, for **each proposed AS involved** in the application **or part of the application** under review, of:

| Highest class of minor works that individual is involved in | Fee each |
|---|---|
| Class I | **$1,240** |
| Class II | **$835** |
| Class III only | **$410** |

| Example | Fee |
|---|---|
| Review whole Class I application with 1 Class I AS | **$1,240** |
| Review with 2 Class I AS involved | **$2,480** |
| Review only the Class III portion involving 1 Class III AS | **$410** |

Refund if BA substitutes the decision under Cap 123N **s. 26(8)(a)(ii)** (s. 17(3)).

> “Or part of the application” means a partial challenge can cost less than reviewing the whole multi-AS filing — but only if the request is framed that way under Cap 123N s. 26.

### 7. Part 4 — provisional RMWC fees (HK$)

Use **only** when the contractor is seeking / holding Cap 123N provisional registration (ss. 64–65). Do not apply these numbers to ordinary register RMWCs.

**Editorial note:** Part 4 pages in the consolidated PDF show last-updated **2.4.2020**; Parts 1–3 show **21.10.2023**. Amounts below are as printed in the 21.10.2023 whole-chapter consolidation — confirm against current e-Legislation if quoting years later.

#### 7.1 Application (s. 19) — base + extra AS

| Highest class | Base | Extra AS (each beyond first for highest class) |
|---|---|---|
| Class I | **$865** | Class I **$710** / Class II **$480** / Class III **$480** |
| Class II | **$590** | same extras |
| Class III only | **$590** | same extras |

**Oddity vs Part 3:** Class II and Class III provisional **base** fees are the **same** ($590). Class II and Class III **extra-AS** add-ons are also the **same** ($480). Only Class I provisional is priced materially higher.

#### 7.2 Fee on grant (s. 20)

Cap 123N **s. 65(5)** registration as RMWC (provisional): **$315** flat.

#### 7.3 Review (s. 21)

Aggregate per involved proposed AS:

| Highest class that individual is involved in | Fee each |
|---|---|
| Class I | **$1,100** |
| Class II | **$810** |
| Class III only | **$395** |

Refund if BA substitutes under Cap 123N **s. 26(8)(a)(ii)** (s. 21(3)).

**Compared with ordinary Part 3 review (s. 17):** provisional review is slightly cheaper at each band ($1,100 / $810 / $395 vs $1,240 / $835 / $410).

### 8. Review refund rule — same trigger in all three Parts

| Cap 123O | Track | Review fee | Refund when |
|---|---|---|---|
| s. 8 | Part 2 Class III items | $110 flat | BA substitutes decision under Cap 123N **s. 26(8)(a)(ii)** |
| s. 17 | Part 3 types/classes | Per involved AS | Same |
| s. 21 | Part 4 provisional | Per involved AS | Same |

**Not refundable merely because:**
- The applicant withdraws
- The review is dismissed
- BA upholds the original decision / recommendation

> Treat review fees as a real cost of challenging MWCRC / BA registration decisions — not free insurance.

Part 2 / Part 3 review also covers a **recommendation** of the Minor Works Contractors Registration Committee (where Cap 123N s. 26 applies to that recommendation). Part 4 s. 21 text is framed as review of a **decision** on the provisional application (see Cap 123O s. 18(3)).

### 10. Architect / AP / procurement checklist before locking MWCS at concept

1. **Class / Type / item first (Cap 123N Sch. 1)** — Cap 123O fee band is a lagging consequence. Freeze dimensional ceilings so the required registration class does not silently jump.
2. **Specify registration scope in appointment docs** — write Class + Type(s) **or** Class III item number(s), not “registered minor works contractor.”
3. **Confirm which track the appointee is on** — Cap 123N s. 11 item person vs s. 12 type/class firm vs provisional. Wrong track → wrong fee table and possibly wrong legal capacity.
4. **AS roster** — for Class I packages needing multiple AS, prefer a firm that **already** holds them; mid-project s. 14 / s. 15 / s. 16 filings cost $615–$2,700+ per person and burn programme.
5. **Expiry / removal check** — diary renewal; know s. 12 vs s. 13 (and Part 2 s. 6) before naming a sole-source RMWC.
6. **Experience-only Class III individuals** — expect $290 not $155 on first registration / add-items if Cap 123N s. 11(2)(a) / s. 22(2)(a) experience route is used for any item.
7. **New firm registration cost plan** — always include Part 3 **s. 11 $595** (or Part 4 **s. 20 $315**) on top of application.
8. **Combined s. 16 filing** — if both add-type and add-AS are needed, require linked co-submission language on the forms.
9. **No geometry in Cap 123O** — projection depth, AC weight, excavation, cantilevers, notice periods, plan scales stay in **Cap 123N / Cap 123 / FS Code**. This Cap only prices register paperwork.
10. **Review refund** — only on BA substitution under Cap 123N s. 26(8)(a)(ii).
11. **Fee concession history** — older BD circulars waived/reduced fees for limited windows (including periods ending around the 21.10.2023 consolidation). For current cost plans, use the **amounts in this Cap** unless a later amendment or BD payment notice says otherwise.
12. **Cross-read Cap 123N** for: who must be appointed (ss. 27–28), notice clocks (Part 6), personal performance duty for Class III natural persons (s. 44), and Types A–H matching.
