# Cap 123O — Building (Minor Works) (Fees) Regulation
**Architect critical summary for schematic design**  
Consolidated version: 21 October 2023 | Cap. 123 sub. leg. O

> Scope note: Cap 123O is the **RMWC registration fee schedule** — fixed statutory amounts tied to Cap 123N (Minor Works) application / renewal / restoration / additional works / authorized signatory / review / provisional registration. It is **not** the MWCS design rulebook (that is Cap 123N + Sch. 1–3), not Cap 123A plan fees, and not contractor tender rates. Use this when the scheme intends a **minor-works path** and you need to budget / specify **which class / type / item registration** the appointed contractor must hold.

---

## Regulatory Overview

Cap 123O prescribes the fees payable under Cap 123N for registration (and related applications) of **registered minor works contractors** — both **individual Class III item** registration (Part 2) and **firm / multi-type / multi-class** registration (Part 3), plus **provisional** registration (Part 4). Fees scale by the **highest class** (I > II > III) and by the number / class of **proposed authorized signatories (AS)**; there are no occupancy-type or building-geometry controls in this Cap.

---

## Critical main topics and subtopics

### 1. When Cap 123O matters at schematic design

| Situation | Cap 123O relevant? | SD implication |
|---|---|---|
| Scheme uses Cap 123N MWCS path (Class I / II / III) | **Yes** — contractor must be correctly registered | Lock MW class early; fee + AS stack drive which RMWC you can appoint |
| Full Cap 123 s.14 BP + Cap 123A consent | **No** (different fee regime) | Cap 123O does not price BD plan approval |
| Client / specialist wants to **become** an RMWC for the works | **Yes** | Budget registration + (Part 3) **registration fee on grant** + AS add-ons before awarding |
| Only specifying “engage RMWC” in prelims | Indirect | Still specify **class + type/item** so the appointable pool and fee exposure are clear |

**Hierarchy rule (s. 2(3)) — drives every Part 3 fee band:**
- Class I is higher than Class II and Class III
- Class II is higher than Class III  
→ Fee is set by the **highest class** the application covers, not by how many Type A–H items you pile under a lower class.

> SD takeaway: Crossing from Class III → II → I is a **registration-cost cliff** for the contractor, not only a notice / professional-appointment cliff under Cap 123N.

### 2. Two registration tracks — which fee table applies

| Track | Cap 123N gateway | Cap 123O Part | Who typically |
|---|---|---|---|
| **Item-based Class III only** | s. 10(1)(a) / renew / restore / add items | **Part 2** (ss. 3–8) | Individual craftsman / small Class III specialist |
| **Type(s) under class(es)** | s. 10(1)(b) / firm-style registration | **Part 3** (ss. 9–17) | Company RMWC covering Types A–H at Class I / II / III |
| **Provisional** | Cap 123N Part 10 (ss. 64–65) | **Part 4** (ss. 18–21) | Transitional / provisional RMWC only |

**Do not mix tables:** Part 2 fees are flat / experience-surcharged for **items of Class III**; Part 3 fees are **highest-class + per-extra-AS** for **types under classes**.

### 3. Part 2 — Class III **item** registration fees (HK$)

Applies when Cap 123N registration is for **one or more Class III items** only (not firm multi-type Class I/II).

| Action | Cap 123O | Fee |
|---|---|---|
| Application for registration | s. 4 | **$290** if ≥1 item relies **solely on applicant’s own experience** (Cap 123N s. 11(2)(a)); otherwise **$155** |
| Renewal | s. 5 | **$105** |
| Restoration of name | s. 6 | **$105** |
| Add additional Class III items | s. 7 | Same split: **$290** (experience-only route) / **$155** (other) |
| Review of BA decision / MWCRC recommendation | s. 8 | **$110** — **refunded** if BA substitutes the decision (s. 8(2)) |

**SD takeaway:** Experience-only Class III individuals pay almost **double** on first registration / add-items. For cheap Class III scopes, confirm the appointed person is already registered for the **exact item numbers** in Cap 123N Sch. 1 Div. 3 — avoid mid-project “register extra items” lag + $155/$290 fee.

### 4. Part 3 — multi-type / multi-class fees (HK$) — main SD cost map

#### 4.1 Application to register (s. 10) — by highest class + extra AS

| Highest class in application | Base application fee | Each **extra** AS (beyond first AS for that highest class) |
|---|---|---|
| Class I | **$2,920** | Class I AS **$2,440** / Class II AS **$810** / Class III AS **$560** |
| Class II | **$1,100** | Same extra-AS scale as above |
| Class III types only | **$730** | Same extra-AS scale as above |

**Stacking rule (s. 10(2)–(3)):** Nominate 2+ proposed AS → pay base **plus** the add-on for **every AS other than the first nominated for the highest class**.

#### 4.2 Fee on grant of registration (s. 11)
Once Cap 123N s. 12(7) registration is granted: **$595** flat — same for Class I, II, or Class III-only (not class-scaled).

> Budget application fee **and** grant fee as a pair for any new firm registration.

#### 4.3 Renewal / restoration (ss. 12–13) — by highest class

| Highest class | Renewal (s. 12) | Restoration (s. 13) |
|---|---|---|
| Class I | **$1,130** | **$1,300** |
| Class II | **$835** | **$915** |
| Class III only | **$425** | **$435** |

**SD takeaway:** A lapsed RMWC is expensive to restore at Class I — check registration expiry **before** awarding MW packages that depend on a named firm.

#### 4.4 Add classes/types or add an AS (ss. 14–15) — per nominated person

| Purpose | Class I AS | Class II AS | Class III AS |
|---|---|---|---|
| Add class(es) / type(s) to registration (s. 14) | **$2,700** | **$965** | **$615** |
| Approve nomination of **additional** AS (s. 15) | **$2,700** | **$965** | **$615** |

Fee for adding classes/types = **sum** of the table amount for **each** proposed AS named on that application (s. 14(1)–(2)).

#### 4.5 Combined application — add works **and** add AS together (s. 16)

If Cap 123N ss. 21(2) + 24(1) applications are **linked on the form and submitted together**, fees are calculated **once per individual** as:

| Individual’s role | Amount charged |
|---|---|
| For add-classes/types only | s. 14 amount |
| For add-AS only | s. 15 amount |
| Named for **both** applications | **Whichever of s. 14 / s. 15 is higher** (same rate today → no double-pay for dual role) |

**Procurement takeaway:** Bundle AS nomination with class/type expansion in one filing when both are needed — avoids two separate process tracks; fees already equalize dual-role nominees.

#### 4.6 Review fees — multi-type track (s. 17)

Review fee = **sum per proposed AS involved** in the challenged application (or part):

| Highest class of that individual’s involvement | Review fee each |
|---|---|
| Class I | **$1,240** |
| Class II | **$835** |
| Class III only | **$410** |

Refund if BA substitutes the decision (s. 17(3)).

### 5. Part 4 — provisional RMWC fees (HK$)

Only when Cap 123N provisional registration (ss. 64–65) is in play — do not use these numbers for ordinary RMWC appointment unless the contractor is actually on the **provisional register**.

| Action | Fee |
|---|---|
| Application (s. 19) — Class I highest | **$865** + extra AS: I **$710** / II **$480** / III **$480** |
| Application — Class II highest | **$590** + same extras |
| Application — Class III only | **$590** + same extras |
| Fee on grant of provisional registration (s. 20) | **$315** flat |
| Review (s. 21) per involved AS | I **$1,100** / II **$810** / III **$395** — refund if BA substitutes |

### 6. Architect / AP checklist before locking MWCS at concept

1. **Class decision first** (Cap 123N Sch. 1) → Cap 123O fee band is a lagging consequence, not a design input — but Class I registration economics may push some contractors off small Class I packs.
2. **Specify registration scope in appointment docs:** Class + Type(s) / item number(s), not vague “registered minor works contractor.”
3. **AS coverage:** Complex Class I packages needing multiple AS → budget s. 10 / 14 / 15 stacks for the contractor (or prefer a firm already holding the needed AS roster).
4. **No geometry in Cap 123O:** Projection depth, AC weight, excavation depth, cantilever rules, notice periods remain **Cap 123N / Cap 123** — this Cap only prices the registration paperwork.
5. **Review refund exists** (ss. 8, 17, 21) but only if BA **substitutes** the decision — do not treat review fee as free insurance.
6. **Consolidated date 21.10.2023:** Fee concession eras referenced in older BD circulars are time-limited; use the amounts in this Cap unless a later amendment / BD fee notice supersedes.
