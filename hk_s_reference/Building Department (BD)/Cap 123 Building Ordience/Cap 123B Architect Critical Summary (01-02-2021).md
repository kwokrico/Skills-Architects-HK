# Cap 123B — Building (Construction) Regulations
**Architect critical summary for schematic design**  
Consolidated version: 1 February 2021 | Cap. 123 sub. leg. B

> ⛔ **Status: REPEALED.** The attached consolidated PDF is a **repeal stub only** (title + “已廢除 / Repealed”). There are **no operative design constraints** left in Cap 123B.  
> **Use instead:** Cap **123Q** Building (Construction) Regulation (in force 1.2.2021) + BD Codes of Practice accepted under **PNAP APP-53**.  
> Cap 123B was repealed by L.N. 8 of 2020 concurrently with Cap 123Q (L.N. 5 of 2020).

---

## Regulatory Overview

Cap 123B formerly prescribed the design and construction requirements for **buildings, streets, building works and street works** under the Buildings Ordinance. With effect from **1 February 2021** it is wholly repealed and superseded by Cap **123Q**, a largely **performance-based** regulation covering materials, loads, structural design methodology, geotechnics / foundations / site formation, enclosure & weathertightness, fire-resisting construction, protective barriers, lifts/escalators, and miscellaneous works (wells, chimneys, ducts, vermin control).

Non-compliance is not criminalised in Cap 123B/123Q themselves — BA enforces via Cap 123 ss.14 / 16 (refuse plan approval / refuse consent).

### 2. What Cap 123B used to own → where it lives now (Cap 123Q map)

Use this as the schematic-design topic checklist. Former Cap 123B substance was reorganised into Cap 123Q’s **12 Parts + Schedule**:

| Cap 123Q Part | Design topics (ex-Cap 123B territory) | Schematics implication |
|---|---|---|
| **Part 2** | Materials for building / street works | Specify materials that meet BA-accepted criteria (APP-53); early agreement if exotic / non-listed |
| **Part 3 + Schedule** | Dead / imposed / wind loads — prevent overloading | Lock **use class & occupancy loads** at concept (Loading Code 2011/2021 Ed.); old Cap 123B load tables → Schedule to Cap 123Q |
| **Part 4** | Design methodology, strength & serviceability, stability, construction methods / procedures | Structural system + buildability narrative; coordinate AP/RSE methods |
| **Parts 5–7** | Site investigation; foundations; site formation; retaining walls; **bulk excavation** | Basement envelope, ELS, slopes / retaining → RGE + Cap 123A geo packs; bulk dig is a control trigger |
| **Parts 8–9** | External walls, cladding, curtain walls; moisture / water protection | Curtain wall / cladding strategy; waterproofing at envelope & wet rooms; **external M&R access** (see §3) |
| **Parts 10–11** | Fire-resisting construction; protective barriers; **lifts & escalators** | FS Code + barrier geometry; lift/escalator planning now Cap 123Q ss.40–41 (replaced Cap 123B reg. 9A) |
| **Part 12** | Ground treatment; wells; chimneys / flues / fireplaces; vermin control; ducts | Wells rare but controlled; chimneys/flues; accessible ducts (person-entry ducts) |

### 3. New / sharpened constraints that arrived with Cap 123Q (not optional at SD)

These replaced or tightened former Cap 123B practice — bake into massing, elevation depth, and plant strategy:

#### 3.1 External maintenance & repair (M&R) access — **hard SD hold point**
Cap 123Q requires **adequate means of access** to the exterior / outer surface for maintenance or repair, notably at:
- External walls / cladding / curtain walls / roofs and projections — e.g. Cap 123Q **ss.27(2), 28(5), 31(3)** (and related roof / projection provisions such as **s.34(3)**)

**Design takeaways:**
- AC platforms / catwalks / BMU / gondola / reachable recesses are **statutory spatial claims**, not “nice-to-have”
- Follow **Code of Practice on Access for External Maintenance 2021 (2024 Ed.)** — BA accepts CoP compliance as satisfying performance requirements
- Coordinate with FS Code Part B/C (escape & fire service access) so M&R access does not clash with MOE / FSI
- Projection depth, façade module, AC outdoor unit zoning, and podium/roof plant paths must be settled **before** freezing elevations

#### 3.2 Protective barriers (Cap 123Q ss.36–38)
- Guard heights, strength, and openings controlled under Cap 123Q (not repealed Cap 123B reg. 8 wording)
- Apply to balconies, roofs, voids, accessible edges — cross-check Cap 123F reg. 3A, PNAP APP-110, Loading Code
- Fall ≥ ~2 m triggers occupational safety guard-rails / toe-boards unless Cap 123Q barrier provided (see External Maintenance CoP)

#### 3.3 Lifts & escalators (Cap 123Q ss.40–41)
- Replaces former Cap 123B **reg. 9A**
- Shaft, pit, overrun, structural support, and machine-room / machine-room-less planning must satisfy Cap 123Q performance + Lift Ordinance / EMSD practice — brief M&E / lift consultant at concept

### 4. Codes of Practice that actually set your schematic numbers (via APP-53)

Cap 123Q states *performance*; APP-53 names the documents BA treats as compliant. At SD, design to these (current editions as listed in APP-53):

| Domain | Primary accepted document |
|---|---|
| Concrete structure | CoP Structural Use of Concrete 2013 (2020 Ed.) |
| Steel | CoP Structural Use of Steel 2011 (2023 Ed.) |
| Wind | CoP on Wind Effects in Hong Kong 2019 |
| Dead / imposed loads | CoP for Dead and Imposed Loads 2011 (2021 Ed.) |
| Foundations | CoP for Foundations 2017 |
| Precast | CoP for Precast Concrete Construction 2016 |
| Glass (structural) | CoP for Structural Use of Glass 2018 |
| Fire resisting construction / MOE interface | CoP for Fire Safety in Buildings 2011 (June 2023 Ed.) |

### 6. Schematic-design go / no-go gate (architect checklist)

Before freezing massing, façades, and consultant briefs:

1. **Confirm statute** — Cap 123B is dead; all construction-performance citations = Cap **123Q**.
2. **Assign use / occupancy for loading** — Loading Code categories drive slab depths, transfer structures, and column grids.
3. **Wind + height** — early massing height / aspect ratio ↔ Wind Effects CoP; set cladding / CW pressure narrative.
4. **Basement / slope / fill** — if any excavation, retention, or site formation → RGE + Cap 123Q Parts 5–7 path; do not assume Cap 123B wording.
5. **Façade & AC strategy** — mandatory **external M&R access** (Cap 123Q ss.27/28/31 family + External Maintenance CoP); allocate platform / BMU / reachable zones in the first massing.
6. **Protective barriers** — balconies, roofs, voids, accessible plant decks sized to Cap 123Q ss.36–38 + APP-110.
7. **Lift / escalator cores** — Cap 123Q ss.40–41 spatial envelopes (pit / overrun / machine space) locked with lift consultant.
8. **Fire-resisting construction** — FS Code 2011 via Cap 123Q Part 10; do not treat fire rating as only an FSD item.
9. **Materials** — stick to APP-53 list or secure BD AIP for alternatives before DD.
10. **Admin package** — Cap 123A plan / consent process still applies; geo / foundation / ELS demonstrations must show Cap **123Q** compliance (not Cap 123B).
