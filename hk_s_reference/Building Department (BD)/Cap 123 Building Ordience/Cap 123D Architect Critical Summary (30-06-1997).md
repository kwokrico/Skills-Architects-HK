## Regulatory Overview

Cap 123D formerly prescribed **building works requirements for escalator installations** under the Buildings Ordinance (originally G.N.A. 79/80 of 1959; enactment history through Ordinance 43 of 1993). The consolidated copy dated **30 June 1997** records the Cap as wholly **repealed**.

Today, BA enforces the **building envelope** around escalators (and lifts) via Cap **123Q** ss.39–41 + the **2011 Lift Code**; DEMS enforces the **escalator/lift apparatus** via Cap **618**. Cap 123D itself has no remaining design rules.

### 3. Cap 123Q Division 2 — when it applies (s.39) — full exclusions

Applies to a building with a **lift or escalator** installed / to be installed, **other than**:

| Exclusion | Typical example |
|---|---|
| (a) Amusement device / ride (Cap 449) | Theme-park ride |
| (b) Belt / bucket / scoop / roller conveyor | Industrial conveyor |
| (c) Furnace / similar charging hoist | Process plant |
| (d) Machine feed-only hoist | Production line feeder |
| (e) Goods lift ≤**3.5 m** travel, **does not pass a floor**, goods/stacking/loading only | Short warehouse stacker |
| (f) Same as (e) but **vehicle-raising only** | Short car hoist |
| (g) Construction-site lift for workers / materials | Temporary builders’ hoist |
| (h) Ramp connected to wharf / pier | Ferry pier ramp |
| (i) Stage / orchestra lift | Theatre stage machinery |
| (j) Stairlift along a flight of stairs (with/without wheelchair) | Domestic stairlift |
| (k) Disability lifting platform, level difference ≤**2 m** | Short BFA platform lift |

**SD takeaway:** Mall / office / podium / transit **passenger escalators and lifts are in scope**. Confirm carve-outs with lift consultant — exclusion from Cap 123Q Div. 2 does **not** automatically exclude Cap 618.

**Definitions (s.39(2)):**
- **Restricted space (lift)** = lift shaft + space containing associated equipment/machinery  
- **Restricted space (escalator)** = space containing associated escalator equipment/machinery  
- “Associated equipment or machinery” → Cap 618 s.2(1)

### 4. Cap 123Q performance duties for the *building* (ss.40–41)

#### 4.1 Design & construction (s.40) — hard SD brief

Building (and any later A&A addition of a lift/escalator) must:

| Duty | Spatial / structural claim |
|---|---|
| Adequate **structural strength** | Well / wellway / supports sized for operation, maintenance, buffer / safety-gear forces; RSE owns fixings (APP-29 §3) |
| Adequate **space** | Table 1 / §5 envelopes; landings; MR clearances; unrestricted passenger areas |
| Adequate **protection** | Enclosure, partitions, obstruction guards, fall protection |
| Adequate **access** | Permanent common-area routes to plant / restricted space |
| Adequate **ventilation** | Liftwell top vents (§3.3); MR environmental design |
| Restricted space **inaccessible** except inspection / maintenance / repair / rescue | Lockable doors/traps; no public through-routes |

#### 4.2 Notices (s.41)

| Notice | Where |
|---|---|
| Caution: danger of entering restricted space + interfering with operation | Conspicuous on every door/access to restricted space |
| “Do not use lift in fire” | Every **lift** entrance (escalators are **not** MOE substitutes) |

### 5. Escalators — Lift Code §5 (primary Cap 123D successor) — SD space killers

#### 5.1 Separate machine / driving / return stations (§5.1)

| Item | Rule |
|---|---|
| When | Separate machinery space and/or separate driving & return stations provided |
| Enclosure | Solid walls, ceilings, door and/or traps |
| Use | Escalator equipment **only** |
| Permitted extras | Lift / service-lift machines; room A/C or ventilation; FSI as required by DFS for the room |
| Sprinklers | **Not normally** fitted |
| Clear height | ≥ **2.1 m** |
| Size | Enough for safe access to all components, especially electrical |
| Access | Prefer **stairs**; ladder only if stairs impractical |
| Ladder rules | Permanently fixed; if height **> 2 m** → safety hoops or fall-arrest; top platform + railings + handholds |
| Notices | Machine-room danger notice ≥**25 mm**, durable, incised/embossed, bilingual |

#### 5.2 Unrestricted passenger landings (§5.2) — lock before atrium width freezes

| Item | Rule |
|---|---|
| Width of unrestricted area | ≥ distance between escalator **handrail centrelines** |
| Depth (from end of balustrade) | ≥ **2.5 m** — **or** ≥ **2.0 m** if unrestricted width ≥ **2×** handrail-centreline distance |
| Clear height | ≥ **2.3 m** above **all steps** **and** above the unrestricted landing area |

**Worked check:** handrail CL spacing 1000 mm → landing box ≥1000 mm wide × **2500** mm deep; **or** ≥**2000** mm wide × **2000** mm deep. Clear height **2300** under soffits, beams, signs, and false ceilings over the **entire** run and both landing boxes.

**SD takeaway:** Retail seating, queue rails, planter boxes, and “temporary” kiosks in the unrestricted zone fail put-into-service inspections — keep the box clear on plans and fit-out guidelines.

#### 5.3 Obstructions & handrail clearances (§5.3 + APP-29 App. D)

| Item | Rule |
|---|---|
| Building parts that may hit riders | Obstruction guards at floor intersections, building obstacles, **criss-cross** banks — detailing per Cap 618 Design & Construction CoP |
| Handrail outer edge → wall / obstacle | Horizontal clearance **never < 80 mm** |
| Balustrade exterior panel ↔ adjacent wall/rail | If clearance **> 100 mm** → proper protective guards (APP-29 App. D) |
| Surfaces facing handrail | Wellway / building obstacle / adjacent criss-cross escalator wall → **smooth continuous vertical** face |
| Underside at floor intersections / upper escalator deck | **Smooth continuous flat** underside (false ceiling / bottom deck) |

#### 5.4 Escalator associated works — DEMS fail list (APP-29 Appendix D)

Complete **before** Cap 618 put-into-service application:

1. Permanent machine-room doors — self-closing, lockable, permanent warning notices  
2. Clear and safe access to machine rooms  
3. Unnecessary holes in machine rooms filled  
4. Adequate electricity by **permanent** cables  
5. MCB + isolation switches with permanent labels for lighting/sockets **per escalator**  
6. Protective guards where balustrade exterior ↔ adjacent rail/wall clearance **> 100 mm**  
7. Permanent obstruction guards at floor intersections / obstacles / criss-cross  
8. Adequate handrail ↔ wall / criss-cross / obstacle clearance  
9. Clear height ≥ **2.3 m** above steps and unrestricted areas  
10. Required unrestricted passenger area at **both** landings  
11. Permanent adequate lighting around escalator including both landings (later décor must not kill illumination)  
12. Smooth continuous vertical face toward handrail (wellway / obstacles / adjacent escalator)  
13. Smooth continuous flat underside at floor intersections / adjacent criss-cross bottom deck

### 6. Companion Cap 123E territory — lifts (Lift Code §§2–3 + Table 1)

Cap 123D schemes almost always also carry lifts. Treat Cap **123E** as equally repealed; design to the same Cap 123Q + Lift Code path.

#### 6.1 Lock type first (SD fork)

| Decision | Which rules | SD takeaway |
|---|---|---|
| Passenger / goods lift (not service) | §§2–3 + **Table 1** | Size well, pit, headroom, entrance, MR by **load + speed** |
| Service lift | **§4 only** | ≤250 kg; floor ≤1 m²; height ≤1.2 m; pit often not required |
| **MRL** | §2.3 / §3.11.1 | Table 1 **MR area** and **overall Uh** do **not** apply; EMSD-approved model on GBP; control panels from **common areas** only |
| Firemen’s lift | This CoP + **B(P)R 41B + FS Code Part D** | Parallel track — lobby, size, power, FR |
| Deviation from Table 1 | §2.2 | Written confirmation from registered lift engineer / authorised person of registered lift contractor (fit + safe maintainability + Cap 618 compliance) |

**SD takeaway:** Fix rated load (kg/persons), speed (m/s), conventional vs MRL, and firemen’s-lift need **before** locking floor-to-floor, basement excavation, roof plant, and core width.

#### 6.2 Table 1 — dimension key

| Code | Meaning |
|---|---|
| **Cw / Cd** | Car internal width / depth (mm); Cw×Cd must **not exceed Ca** |
| **Ca** | Max. car internal area (m²) |
| Car height | Typically **2300** mm internal |
| **Ww / Wd** | Well min. width / depth |
| **Ew / Eh** | Clear entrance; Eh = **2100** throughout Table 1 |
| **Ph** | Pit depth |
| **Sh** | Headroom above top landing |
| **Ra / Rw / Rd / Rh** | Machine-room min. area / width / depth / height |
| **Uh** | Overall headroom (pit + travel + MR stack — see CoP Figure 1) |

#### 6.3 Quick Table 1 extract — common loads (verify full CoP for exact speed row)

| Rated load | Persons | Car Cw×Cd | Well Ww×Wd (typ.) | Ew | Pit Ph (by speed) | Headroom Sh (by speed) | MR Ra (typ.) | Overall Uh (by speed) |
|---|---|---|---|---|---|---|---|---|
| **630 kg** | 8 | 1100×1400 | 1800×2100 | 800 | 1700–1800 | 4450–4850 | 15 m² | 7250–7650 |
| **800 kg\*** | 10 | 1400×1350 | 1900×2300 | 800 | 1700–2800 | 4450–5650 | 15 | 7250–8450 |
| **900 kg** | 12 | 1600×1350 | 2100×2100 | 900 | 1700–2800 | 4450–5650 | 15 | 7250–8450 |
| **1000 kg** | 13 | 1600×1400 | 2400×2300 | 1100 | 1800–3650 | 4300–6300 | 20 | 7200–10400 |
| **1250 kg** | 16 | 1950×1400 | 2600×2300 | 1100 | 1900–3650 | 4400–6300 | 22 | 7250–10400 |
| **1350 kg** | 18 | 2000×1500 | 2600×2300 | 1100 | 1900–3650 | 4800–6300 | 22 | 7800–10400 |
| **1600 kg** | 21 | 2000×1750 | 2600×2600 | 1100 | 1900–3650 | 4800–6300 | 25 | 7800–10600 |

\*750 kg: Ca max = **1.9 m²** (Note 4).

**Speed jumps (massing killers):**
- Crossing ~**2.0 m/s** → pit often jumps to **~2800** mm; Sh to **~5650–6150** mm; Uh +**1–2+ m**
- At **3.5 m/s** → Ph ≈ **3650** mm; Uh often **10.4–10.6 m**

#### 6.4 Table 1 Notes — do not skip

| Note | Rule |
|---|---|
| Intermediate load/speed | **Linear interpolation** |
| Cw × Cd | Product ≤ **Ca** |
| MR Rw × Rd | Each ≥ tabulated **and** product ≥ **Ra** |
| Load **>1600 kg** | Ca +**0.16 m²** / extra 100 kg; Ra +**0.6 m²** / extra 100 kg; other dims ≥ 1600 kg same-speed |
| Alt. to MR footprint | Machine-room **clearances** approach (GBP note) — see §6.9 |
| Speeds outside listed | Table dims still apply (extrapolate per Note 2) |

#### 6.5 Liftwell enclosure, exclusive use, entrance-band finishes

| Item | Rule |
|---|---|
| Enclosure | Totally enclosed imperforate walls, floor, ceiling (FRC → **FS Code Part C**) |
| Materials | Non-combustible, durable, non-dust-creating; adequate strength |
| Inner wall face | Continuous smooth hard vertical surface (unless inaccessible from car top) |
| Exclusive use | Well **for lift only** — no foreign cables/devices |
| Sprinklers | **Must not** be fitted in the liftwell |
| Structure must support | Machine loads; guide loads at safety-gear; buffer from off-centre car; anti-rebound; hydraulic: jacks / clamping / pawl devices |
| **350 mm band** below each landing sill | Continuous smooth hard vertical face; extend ≥**25 mm** beyond full entrance width each side |
| Forbidden in band | **Plaster-faced** and **glass** walls |
| Projections >5 mm | Connect to lintel below **or** splay underside ≥**75°** from horizontal for ≥**20 mm** |

#### 6.6 Inspection / emergency doors & traps

| Opening | Size | Key rules |
|---|---|---|
| Inspection door | ≥ **1.4 m** H × **600 mm** W | Imperforate; **must not open into well** |
| Emergency / rescue door | ≥ **1.8 m** H × **500 mm** W | Readily accessible to rescuers |
| Inspection trap | ≤ **500 × 500 mm** | Key lock; reclose/relock without key |
| Landing sill spacing **>11 m** | Intermediate emergency doors so sill spacing ≤**11 m** | Double-height / skipped-stop shafts trigger this |

**Locks (default):** Dual key-operated (two designated keys simultaneously from outside); openable from inside without key; warning figure ≥**100 mm** at keyholes — **except** single designated key OK if: landing doors serve as emergency doors, **or** door sill ≥**1 m** above adjoining floor.

#### 6.7 Liftwell ventilation (§3.3) — roof / plant coordination

| Item | Rule |
|---|---|
| Principle | Well suitably ventilated; **must not** ventilate other rooms (except rooms serving the lifts) |
| Opening | **Top of well** → open air directly, via ducting, or via machine/pulley room |
| Area | ≥ **1%** of well horizontal cross-section; **never < 0.3 m²** net free area **per well** |
| Shared well 4 / 5 / 6 lifts | Min vent **0.4 / 0.5 / 0.6 m²** |
| Fall protection | Ferrules ≥**50 mm** above slab/FFL |
| Solid-wall multi-lift separation | **Each** well needs its own vents |
| “Open air” | B(P)R definition: vertically uncovered & unobstructed; any horizontal dim ≥**1.5 m**; if enclosed 4 sides → horizontal area ≥**1 m² per 6 m** mean enclosing-wall height |

#### 6.8 Space below lift / multi-lift / pit

| Topic | Rule |
|---|---|
| Prefer | Do **not** place liftwell above space accessible to persons |
| If accessible space below car or CW | Pit base ≥**5 kPa** imposed **and** either solid pier under CW buffer to solid ground **or** CW with safety gear |
| Multi-lift well partition | Full height from pit floor through well; full depth; adequate strength (solid wall / wire mesh) |
| Pit bottom | Smooth, approx. level; **impervious to water** after equipment built-in |
| Access door if Ph **>2 m** | **Mandatory** — to stair meeting §3.12.4; imperforate; not into well; ≥1.4 m × 600 mm |
| Ph **>1.6 m and ≤2 m** | Access door also if layout permits |
| Fall-arrest anchorage | ≥1 point at **lowest** landing, **1.5–1.8 m** above FFL — **waived** if compliant pit access door provided |
| Handholds (no other access) | ~**1.3 m** above sill and ≤**0.9 m** from landing opening; must not project into clear running space |

#### 6.9 Machine / pulley rooms (conventional) & MRL

| Item | Rule |
|---|---|
| Preferred location | **Above** the well |
| Use | Lift equipment only (+ service-lift/escalator machines; room A/C; DFS-required FSI) |
| Sprinklers | **Not normally** in MR/pulley rooms |
| Access from | **Common areas only** — never through private premises |
| Access clear height | ≥ **2 m** (door sills/edges ≤400 mm OK) |
| Level change **>0.6 m** | **Stairs** per §3.12.4; ≤0.6 m → fixed ladder + top platform/railings/handholds |
| MR clear height | ≥ **2.1 m** to underside of **structural roof beams** |
| Pulley room height | ≥ **1.5 m** under roof |
| Access door MR | ≥ **600 mm** W × **1.8 m** H; **must not open inward** |
| Person trap | ≥ **800 × 800 mm**; counter-balanced; support 2 persons (**2000 N**); not open downward |
| Hoisting | Metal supports/hooks + SWL notice in MR **and** at liftwell ceiling soffit |
| MRL | No MR if EMSD-approved; control cabinets from common areas; Table 1 Uh/MR dims N/A |

**Stair standard (§3.12.4):** width ≥**900 mm**; tread ≥**225 mm** anti-slip; riser ≤**175 mm**; ≤**16** steps per flight; handrails both sides; outer barrier ≥**1.1 m**.

**Alternative MR clearances (GBP note) if not using Table 1 Ra:**
- In front of panels: depth ≥**0.7 m**; width ≥**0.5 m** or full panel width  
- Moving parts / manual emergency ops: ≥ **0.5 × 0.6 m**  
- Access ways: ≥ **0.5 m** (or **0.4 m** if no moving parts / hot surfaces)

#### 6.10 Landing interface outside the well

| Item | Rule |
|---|---|
| Landing sill | Strength for loads entering car |
| Water control | **Slight counter-slope** in front of each sill — no washing/drainage into well |
| Fire notice | Bilingual, durable, ≥**15 mm**: **IN CASE OF FIRE DO NOT USE THE LIFT / 如遇火警切勿使用升降機** — each landing (or one in every two if paired) |

#### 6.11 Service lifts (§4) — compact carve-out

| Topic | Rule |
|---|---|
| Trigger | ≤**250 kg**; floor ≤**1 m²**; height ≤**1.2 m** |
| Pit | **Not required** unless lowest level at/near floor; if provided, drain; if depth >**1 m** → permanent access + handholds |
| Landing notice ≥**25 mm** | **SERVICE LIFT — PERSONS FORBIDDEN… / 貨物專用升降機 禁止內進** |
| Machinery ≥150 kg | Floor ≥**1.5 × 1.5 m**; clear H ≥**1.2 m**; door ≥**1.0 × 1.2 m** |
| Machinery <150 kg | Depth ≤**600 mm**; clear H ≥**800 mm**; door ≥800 mm H × lesser of 900 mm or full machine width; **or** reach-all-from-outside + mesh to stop fall into well |

#### 6.12 Lift associated works — DEMS fail list (APP-29 Appendix C) — highlights

Permanent outward-opening MR/pulley doors (self-closing, lockable, notices); clear access; railings/steps at level changes; wells & rooms fully enclosed / holes sealed; permanent lighting (wells, pits, MR, lobbies); vent fans with guards / wind guards; permanent power + labelled isolation per lift; partitions in common wells; waterproof pits; cat ladders/handholds; reinforced mesh to top vents; earth bonding; bilingual car/landing instructions; SWL on lifting beams/hooks. Lobby décor after permission must not kill permanent illumination.

### 7. Structural fixings (APP-29 §3) — AP + RSE hold point

Buildings must provide adequate structural strength for safe operation, **maintenance and inspection**. Attend early to fixing layout, materials, anchors, and load distribution between supporting structure and:

| Component | Typical interface |
|---|---|
| (a) Guide rail mounting bracket | Shaft wall / structure |
| (b) Lift machine | MR floor / beams |
| (c) Deflector sheave (pulley) bracket | MR / well structure |

Escalator truss supports and machine-room loads follow the same AP/RSE discipline under Cap 123Q s.40.

### 11. Schematic-design go / no-go gate (architect checklist)

**Statute & team**
1. Cap 123D / 123E are **dead** — cite Cap **123Q** ss.39–41 + 2011 Lift Code, not Cap 123D or Cap 123B reg. 9A.  
2. Confirm Cap 123Q Div. 2 applies (check s.39 exclusions).  
3. Appoint lift/escalator consultant + RSE for shafts, wellways, fixings, pits, MR.

**Escalators (ex-Cap 123D)**
4. Unrestricted landings: width ≥ handrail CL; depth ≥**2.5 m** (or ≥**2.0 m** if width ≥2× CL).  
5. Clear height ≥**2.3 m** over all steps **and** landing boxes.  
6. Handrail clearance ≥**80 mm**; balustrade–wall >100 mm → guards; smooth vertical/flat faces at intersections & criss-cross.  
7. Machine / driving stations: ≥**2.1 m** clear; lockable access from safe route; no sprinklers normally.  
8. Programme APP-29 App. D associated works before DEMS put-into-service.

**Lifts (ex-Cap 123E)**
9. Lock load, speed, conventional vs **MRL**, firemen’s-lift need with supplier.  
10. Size Ww×Wd, Ph, Sh, Uh, MR from **Table 1** — watch **≥2.0 m/s** and **3.5 m/s** jumps.  
11. Well: exclusive use; no sprinklers; smooth entrance band (no plaster/glass); vent ≥1% (min 0.3 m²).  
12. Multi-lift partition full height/depth; accessible space under shaft → 5 kPa + pier or CW gear.  
13. Pit access strategy if Ph >1.6–2.0 m; anchorage if no pit door; landing counter-slope + fire notices.  
14. MR access from common area; stairs if rise >0.6 m (§3.12.4); hoisting hooks in MR + well soffit.  
15. Landing sill spacing ≤**11 m** or intermediate emergency doors.

**Always**
16. Escalators/lifts ≠ MOE — coordinate FS Code Part B escape separately.  
17. Cross-check DMBA / firemen’s lift / Cap 618 in parallel — do not size from Cap 123D nostalgia.
