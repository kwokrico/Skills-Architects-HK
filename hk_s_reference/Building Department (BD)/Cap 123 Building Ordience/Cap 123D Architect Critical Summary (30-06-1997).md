# Cap 123D — Building (Escalators) Regulations
**Architect critical summary for schematic design**  
Consolidated version: 30 June 1997 | Cap. 123 sub. leg. D

> ⛔ **Status: REPEALED.** The attached consolidated PDF is a **repeal stub only** (title + “已廢除 / Repealed”). There are **no operative design constraints** left in Cap 123D.  
> **Use instead (BD building works):** Cap **123Q** Part 11 Division 2 (**ss.39–41**) + *Code of Practice for the Design and Construction of Buildings and Building Works for the Installation and Safe Use of Lifts and Escalators 2011* (**2011 Lift Code**) via **PNAP APP-29**.  
> **Use instead (equipment / put-into-service):** Cap **618** Lifts and Escalators Ordinance + EMSD CoPs (design/construction of the escalator itself; lift/escalator works).  
> Twin repealed instrument: Cap **123E** Building (Lifts) Regulations — same redirect path.

## Regulatory Overview

Cap 123D formerly prescribed **building works requirements for escalator installations** under the Buildings Ordinance (originally G.N.A. 79/80 of 1959; last listed enactment history through Ordinance 43 of 1993). The consolidated copy dated **30 June 1997** records the Cap as wholly **repealed** — no section text, tables, or geometric rules remain.

Non-compliance is not actionable under Cap 123D. BA enforces current lift/escalator **building-envelope** duties via Cap **123Q** / Cap 123 plan approval; DEMS enforces the **escalator plant** via Cap **618** (permission to put into service).

### 3. Cap 123Q escalator building-works rules that *do* bind schematic design

#### 3.1 When Division 2 applies (s.39)
Applies to a building with a lift **or escalator** installed / to be installed, **except** listed plant (amusement rides Cap 449; conveyors; furnace/machine-feed hoists; short travel ≤3.5 m goods/vehicle lifts that do not pass a floor; construction-site lifts; pier/wharf ramps; stage/orchestra lifts; stairlifts along stairs; disability lifting platforms with ≤2 m level difference).

**SD takeaway:** Standard passenger / public escalators in malls, podiums, transit hubs, and office lobbies = **in scope**. Confirm with lift consultant if a “platform / stairlift / industrial hoist” might fall outside Cap 123Q Div. 2 (it may still fall under Cap 618 or other regimes).

#### 3.2 Performance requirements for the *building* (s.40) — hard SD brief
The building (and any later A&A addition of an escalator) must be designed and constructed to:

| Duty | Spatial / structural claim at SD |
|---|---|
| Adequate **structural strength** | Wellway / supporting structure sized for escalator loads + maintenance forces; RSE owns fixings (APP-29 §3) |
| Adequate **space** | Machine/plant space, truss envelope, landings, clear headroom, unrestricted passenger areas |
| Adequate **protection** | Guarding of holes/voids; obstruction guards at floor intersections & criss-cross banks |
| Adequate **access** | Clear, permanent route to machine rooms / restricted spaces for inspection & maintenance |
| Adequate **ventilation** | Plant / machine space environmental design — brief M&E early |
| Restricted space **inaccessible** except for inspection / maintenance / repair / rescue | Lockable machine doors; no public shortcut through plant voids |

#### 3.3 Notices (s.41) — minimal spatial claim
Restricted-space access doors need permanent conspicuous cautions against entry and against interfering with operation. Design door/approach so notices and locks are feasible (APP-29 App. D).

### 4. Escalator geometry & interface “gotchas” that kill SD (from 2011 Lift Code practice + APP-29 App. D)

Bake these into massing, section, and atrium / mall section cuts **before** freezing floor-to-floor and atrium width:

| Topic | Typical SD hold point |
|---|---|
| **Landing stacking areas** | Required **unrestricted area** for passengers at **both** landings — not finish / retail / seating spillover |
| **Clear height** | ≥ **2.3 m** clear above steps **and** over the unrestricted landing areas |
| **Handrail ↔ wall / obstacle clearance** | Adequate clearance between outer handrail edges and adjacent walls, building obstacles, or criss-cross escalators |
| **Balustrade exterior ↔ adjacent wall/rail** | If clearance exceeds **100 mm**, provide proper protective guards |
| **Floor intersections / criss-cross banks** | Permanent obstruction guards; wellway / facing surfaces form a **smooth continuous vertical** face toward the handrail |
| **False ceilings / underside of upper escalator** | Smooth continuous flat underside where it faces the escalator bank — decoration must not create finger traps or rough interfaces |
| **Lighting** | Permanent adequate lighting around escalator including both landings; later décor / false ceiling must not kill required illumination |
| **Machine room / plant access** | Permanent doors (self-closing, lockable), warning notices, safe access path, holes filled, permanent power / isolation labelling |
| **Structural fixings** | AP + RSE attend guide/mount layout, anchors, load paths for machine & associated equipment (APP-29 §3) |

> Cap 123Q states *performance*; the **2011 Lift Code** (with APP-29 App. A Aug 2015 / App. B Sep 2019 amendments) supplies the numeric / detailing path BA expects. App. B applies to GBP / major revision / A&A plans submitted on or after **1 Dec 2019**.

### 5. Dual authority map — do not conflate BD and EMSD

| Gate | Authority | What freezes at SD |
|---|---|---|
| GBP / building works for fabric around escalator | **BD** (Cap 123 + Cap 123Q + 2011 Lift Code + APP-29) | Wellway, landings, structure, access, guarding, notices locations |
| Escalator plant design / works / put-into-service | **EMSD / DEMS** (Cap 618 + EMSD CoPs) | Supplier selection, drive type, safety devices; programme hold for incomplete “associated works” |
| Occupation / OP programme | Both | EMSD permission to put escalators into service often fails on **building** incomplete works (App. D list) — treat as AP completion checklist |

### 6. Schematic-design go / no-go gate (architect checklist)

Before freezing atria, mall banks, podium connectors, or lobby escalators:

1. **Confirm statute** — Cap 123D is dead; cite Cap **123Q** ss.39–41, not Cap 123D / not Cap 123B reg. 9A.
2. **Confirm Cap 123Q applies** — passenger escalator? (check s.39 exclusions for industrial / special plant).
3. **Appoint lift/escalator consultant + RSE pathway** — structure, openings, machine rooms owned jointly with AP.
4. **Lock floor-to-floor + landing depths** — ≥2.3 m clear height and unrestricted landing areas both ends (2011 Lift Code / APP-29).
5. **Model criss-cross / multi-level banks early** — obstruction guards, smooth facing walls/undersides, handrail clearances.
6. **Allocate machine / restricted space + permanent access** — Cap 123Q s.40; lockable access; no public route through plant.
7. **Brief façade / fit-out / lighting** — landings and illumination must survive later décor.
8. **Cross-check MOE** — escalators are **not** a substitute for protected escape stairs / FS Code Part B.
9. **Programme Cap 618** — associated building works complete before DEMS put-into-service application (APP-29 App. D).
10. **Companion Cap 123E** — if the same scheme has lifts, apply the same Cap 123Q + 2011 Lift Code gate to shafts/pits/overrun (do not invent separate Cap 123D/E rules).
